บทที่ 9

ตัดต่อขั้นสูงใน CapCut

Speed Ramp, Green Screen, Multi-track, Color Grade, Template

🎯 วัตถุประสงค์การเรียนรู้
  • ใช้เทคนิค Speed Ramp สร้างจังหวะน่าสนใจ
  • ใช้ Green Screen (Chroma Key) เปลี่ยนพื้นหลัง
  • จัดการ Multi-track (หลาย Layer) ได้
  • ปรับ Color Grade ให้วิดีโอมี "โทน" เดียวกัน
  • ใช้ Template สำเร็จรูปเร่งงานได้

⚡ Speed Ramp — ปรับความเร็วแบบมีจังหวะ

Speed Ramp คือเทคนิคการเปลี่ยนความเร็ววิดีโอแบบค่อยๆ เร็ว → ช้า → เร็ว สร้างจังหวะที่น่าตื่นเต้น ใช้ได้กับวิดีโอ PR และ Event

📋 วิธีทำ Speed Ramp
  1. เลือกคลิปที่ต้องการ
  2. ไปที่ Speed > Curve
  3. เลือก Preset:
    • Hero Moment: ช้าตรงกลาง เร็วหัวท้าย
    • Bullet Time: ช้ามากตรงกลาง (Slow-mo)
    • Flash In: เร็วตอนต้น ช้าตอนท้าย
  4. หรือปรับ Custom: ลาก Curve ตามต้องการ
  5. Preview ดูจังหวะ
🎯 ใช้ Speed Ramp เมื่อไหร่?
  • เน้นจุดสำคัญ: ช้าตรงจุดที่อยากให้ผู้ชมจับตาดู
  • Transition สวยๆ: ใช้เร็วตรงจุดเปลี่ยน Scene
  • Timelapse: เร่งเวลาช่วงที่ไม่สำคัญ
  • ไม่เหมาะ: วิดีโอที่มีเสียงพูดตลอด (จังหวะจะเพี้ยน)

🟩 Green Screen (Chroma Key)

Green Screen คือเทคนิคลบพื้นหลังสีเขียว (หรือสีอื่น) แล้วใส่พื้นหลังใหม่ ใช้กับ Avatar จาก HeyGen (ถ้าเลือก Green Screen) หรือคลิปที่ถ่ายหน้าจอเขียว

📋 วิธีใช้ Green Screen ใน CapCut
  1. นำ Background (ภาพหรือวิดีโอ) วางใน Main Track
  2. นำคลิป Green Screen วางใน Track ด้านบน (Overlay)
  3. เลือกคลิป Green Screen > ไปที่ Remove BG > Chroma Key
  4. ใช้ Eyedropper คลิกสีเขียว (หรือสีที่ต้องการลบ)
  5. ปรับ Intensity จนพื้นหลังหายไปสมบูรณ์
  6. ปรับ Shadow และ Edge ให้ขอบเนียน
💡 ทางเลือก: Auto Remove Background

CapCut มี AI ลบพื้นหลังอัตโนมัติ (ไม่ต้องใช้ Green Screen) — เลือกคลิป > Remove BG > Auto Removal เหมาะกับคลิปที่ถ่ายหน้าพื้นหลังทั่วไป

📚 Multi-track — จัดการหลาย Layer

วิดีโอที่ซับซ้อนต้องใช้หลาย Track (Layer) ซ้อนกัน CapCut รองรับหลาย Track ทั้ง Video และ Audio:

📐 แนะนำการจัดเรียง Track
Trackเนื้อหาหมายเหตุ
Video Track 1วิดีโอหลัก (คลิปเรียงต่อกัน)เป็น Base
Video Track 2Overlay (Avatar, Logo, ภาพซ้อน)อยู่ด้านบน
Video Track 3Text, Sticker, Lower Thirdด้านบนสุด
Audio Track 1เสียงพากย์ (Voice Over)เสียงหลัก
Audio Track 2เพลงประกอบ (BGM)เบา 20%
Audio Track 3Sound Effectตามจุดที่ต้องการ

🎨 Color Grade — ปรับโทนสี

Color Grade คือการปรับโทนสีให้วิดีโอทั้งเรื่องมี "Look" เดียวกัน ทำให้ดู Professional ขึ้น

📋 วิธีปรับ Color ใน CapCut
  1. เลือกคลิป > ไปที่ Adjust
  2. ปรับค่าพื้นฐาน:
    • Brightness: ความสว่าง (+5 ถึง +10 เพื่อให้สดใส)
    • Contrast: ความคมชัด (+10 ถึง +20)
    • Saturation: ความอิ่มสี (+5 ถึง +15)
    • Temperature: อุณหภูมิสี (บวก = อบอุ่น, ลบ = เย็น)
  3. หรือใช้ Filter สำเร็จรูป (แนะนำ: Natural, Warm, Film)
  4. สำคัญ: ใช้ค่าเดียวกันกับทุกคลิป เพื่อให้สีสม่ำเสมอ
🎭 โทนสีแนะนำตามประเภทวิดีโอ
ประเภทโทนสีการปรับ
PR องค์กรสดใส ProfessionalBrightness +5, Contrast +15, Saturation +10
อบอุ่น ให้กำลังใจโทนส้ม/ทองTemperature +15, Saturation +10
สมัยใหม่ Cleanโทนเย็น สะอาดTemperature -5, Contrast +20
สารคดี Cinematicโทนมืด มีน้ำหนักContrast +25, Brightness -5

📋 Template — ใช้แม่แบบสำเร็จรูป

CapCut มี Template สำเร็จรูปมากมายที่ช่วยให้ทำวิดีโอสวยๆ ได้เร็ว เพียงใส่ภาพ/คลิปของเรา:

📐 วิธีใช้ Template
  1. เปิด CapCut > Templates (มุมบน)
  2. ค้นหา Template ที่ต้องการ (พิมพ์: corporate, intro, promo)
  3. Preview Template > กด Use Template
  4. เปลี่ยน ภาพ/คลิป ในแต่ละช่องเป็นของเรา
  5. แก้ข้อความ (ชื่อองค์กร, เบอร์โทร ฯลฯ)
  6. Export

💡 แนะนำ Template: ค้นหาคำว่า "corporate intro", "promo", "slideshow" จะเจอ Template ที่เหมาะกับงานองค์กร

🔬 Lab 9.1: Speed Ramp + Color Grade

เวลา: 25 นาที

คำสั่ง: ปรับแต่งวิดีโอจาก Lab 8 ด้วยเทคนิคขั้นสูง

  1. เปิด Project จาก Lab 8
  2. ทำ Speed Ramp:
    • เลือกคลิป Scene ที่มี Action (เช่น คนเดินเข้ามา)
    • Speed > Curve > เลือก "Hero Moment"
    • Preview ดูว่าจังหวะสวยไหม
  3. ปรับ Color Grade:
    • เลือกคลิปแรก > Adjust > Brightness +5, Contrast +15, Temperature +10
    • จดค่าไว้ > ใช้ค่าเดียวกันกับทุกคลิป
    • หรือเลือก Filter "Warm" ใช้กับทุกคลิป
  4. Preview ทั้งวิดีโอ — ตรวจสอบว่าสีสม่ำเสมอ
🔬 Lab 9.2: Green Screen + Multi-track

เวลา: 20 นาที

คำสั่ง: ทดลอง Green Screen และ Overlay

  1. นำวิดีโอ Avatar (ถ้าใช้ Green Screen จากบท 5):
    • วาง Background ใน Track 1
    • วาง Avatar Green Screen ใน Track 2
    • ลบ BG > Chroma Key > เลือกสีเขียว
  2. เพิ่ม Logo องค์กร:
    • Import ภาพ Logo (PNG พื้นใส)
    • วางใน Track 3 > ปรับขนาด > วางมุมขวาล่าง
    • ปรับ Opacity 70-80%
  3. เพิ่ม Lower Third (ป้ายชื่อ):
    • Text > เลือก Template Lower Third
    • พิมพ์ชื่อองค์กร + ตำแหน่ง
    • วางในจังหวะที่ Avatar เริ่มพูด
🔬 Lab 9.3: ใช้ Template สร้าง Intro

เวลา: 15 นาที

คำสั่ง: สร้าง Intro 5 วินาทีจาก Template

  1. ไปที่ Templates > ค้นหา "corporate intro"
  2. เลือก Template ที่ชอบ > Use Template
  3. เปลี่ยนข้อความเป็นชื่อองค์กรของท่าน
  4. เปลี่ยนภาพ/สี ให้เข้ากับองค์กร
  5. Export แล้วนำไปวางหน้าวิดีโอหลัก
💡 Tips & Tricks
  • Copy ค่า Color: ปรับคลิปแรกเสร็จ > Copy > Paste Adjustment ไปคลิปอื่น
  • Speed Ramp กับเสียง: ถ้าคลิปมีเสียงพูด อย่าใช้ Speed Ramp (เสียงจะเพี้ยน)
  • Keyframe: ใช้ Keyframe ทำ Animation เช่น Logo ค่อยๆ ปรากฏ
  • Save Preset: Save ค่า Color ที่ชอบเป็น Preset ใช้ซ้ำได้
  • Template + Custom: ใช้ Template เป็น Intro/Outro แล้ว Custom ส่วนกลาง
  • Performance: ถ้าเครื่องช้า ลด Preview Quality ลง แล้วค่อย Export Full
📋 สรุปบทที่ 9
  • Speed Ramp สร้างจังหวะน่าสนใจ ใช้กับคลิปที่ไม่มีเสียงพูด
  • Green Screen/Chroma Key ลบพื้นหลัง ใส่ BG ใหม่ได้
  • Multi-track ช่วยจัดเรียงหลาย Layer: วิดีโอ + Overlay + Text + เสียง + เพลง
  • Color Grade ทำให้วิดีโอทั้งเรื่องมีโทนสีเดียวกัน ดู Professional
  • Template ช่วยทำ Intro/Outro สวยๆ ได้เร็ว
