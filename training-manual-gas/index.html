📘 คู่มือฝึกอบรม Google Apps Script

สำหรับผู้เริ่มต้น — เรียนรู้จากศูนย์ สู่การสร้างระบบอัตโนมัติ

โครงการ "การประยุกต์ใช้ปัญญาประดิษฐ์ในทุกมิติ เพื่อคนพิการ"

📅 วันที่ 23–27 มิถุนายน 2569

📍 โรงแรมฮิลล์ไซด์ เขาใหญ่ จ.ปราจีนบุรี

🎯 เกี่ยวกับโครงการ

โครงการ "การประยุกต์ใช้ปัญญาประดิษฐ์ในทุกมิติ เพื่อคนพิการ" มุ่งเน้นพัฒนาศักยภาพเจ้าหน้าที่องค์กรเพื่อคนพิการ ให้สามารถประยุกต์ใช้เทคโนโลยีดิจิทัลและปัญญาประดิษฐ์ในงานประจำวันได้อย่างมีประสิทธิภาพ

👨‍🏫 วิทยากร:

  • นายปัญญา ผาธรรม (สนพ.สุรินทร์)
  • นายพรพจน์ แสวงดี (สนพ.ยโสธร)

👥 กลุ่มเป้าหมาย:

  • เจ้าหน้าที่องค์กรเพื่อคนพิการ
  • ไม่จำเป็นต้องมีพื้นฐาน IT
  • เริ่มเรียนรู้จากศูนย์

📑 สารบัญ

  1. บทที่ 1: Google Apps Script คืออะไร — แนะนำ GAS, เปิด Script Editor, Hello World
  2. บทที่ 2: Google Sheets เป็นฐานข้อมูล — SpreadsheetApp, อ่าน/เขียนข้อมูล
  3. บทที่ 3: ฟังก์ชัน CRUD — สร้าง/อ่าน/แก้ไข/ลบ ข้อมูลใน Sheet
  4. บทที่ 4: Docs AI + Form — สร้างเอกสารอัตโนมัติ + Google Forms Trigger
  5. บทที่ 5: Trigger — ตั้งเวลาทำงานอัตโนมัติ (Time-based, Event-based)
  6. บทที่ 6: doGet / doPost — สร้าง Web App รับส่งข้อมูลจากภายนอก
  7. บทที่ 7: HTML Service + Bootstrap — สร้างหน้าเว็บด้วย GAS
  8. บทที่ 8: MailApp — ส่งอีเมลอัตโนมัติ + แจ้งเตือน
  9. บทที่ 9: Deploy Web App — ขั้นตอนเผยแพร่ Web App
  10. บทที่ 10: Workshop — สร้างระบบลงทะเบียนกิจกรรมครบวงจร

💡 แนวทางการเรียนรู้

  • เนื้อหาเน้น ปฏิบัติ 70%+ — ลงมือเขียน Code จริงทุกบท
  • เริ่มจากง่ายไปยาก — ไม่ต้องมีพื้นฐานก็เรียนได้
  • ทุกบทมี Lab ปฏิบัติ ให้ทำตามทีละขั้นตอน
  • มี Tips และเทคนิคจากประสบการณ์จริง
  • บทสุดท้ายเป็น Workshop สร้างระบบจริงครบวงจร

🛠️ สิ่งที่ต้องเตรียมก่อนเริ่มเรียน

จำเป็นต้องมี:
  • บัญชี Google (Gmail)
  • คอมพิวเตอร์ที่เชื่อมต่ออินเทอร์เน็ต
  • เว็บเบราว์เซอร์ (แนะนำ Chrome)
ไม่จำเป็นต้องมี:
  • ❌ ไม่ต้องติดตั้งโปรแกรมใดๆ
  • ❌ ไม่ต้องมีพื้นฐานเขียนโปรแกรม
  • ❌ ไม่ต้องใช้เงินเปิดใช้งาน
