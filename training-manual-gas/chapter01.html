📚 Google Apps Script (GAS) คืออะไร?

Google Apps Script (GAS) คือ ภาษาโปรแกรมที่ Google สร้างขึ้น เพื่อให้เราสั่งงาน Google Workspace (Sheets, Docs, Forms, Gmail ฯลฯ) แบบอัตโนมัติ โดยใช้ภาษา JavaScript เป็นพื้นฐาน

🔑 จุดเด่นของ GAS

  • ฟรี 100% — ไม่มีค่าใช้จ่ายใดๆ
  • ไม่ต้องติดตั้ง — เขียนในเบราว์เซอร์ได้เลย
  • ไม่ต้องเช่า Server — Google ดูแลให้
  • เชื่อมต่อ Google ทั้งหมด — Sheets, Docs, Gmail, Drive
  • ทำ Web App ได้ — สร้างหน้าเว็บฟรี
  • ส่ง Line/Email อัตโนมัติ — แจ้งเตือนทันที

🖥️ วิธีเปิด Script Editor

มี 2 วิธีในการเปิด Script Editor:

วิธีที่ 1: เปิดจาก Google Sheets

1

เปิด Google Sheets ที่ sheets.google.com

2

สร้างหรือเปิด Spreadsheet ที่ต้องการ

3

ไปที่เมนู ส่วนขยาย (Extensions) → Apps Script

4

จะเปิดหน้าต่าง Script Editor ขึ้นมาในแท็บใหม่

วิธีที่ 2: เปิดจาก script.google.com โดยตรง

1

เปิดเบราว์เซอร์ไปที่ script.google.com

2

คลิก "โปรเจกต์ใหม่" (New Project)

✍️ เขียนโปรแกรมแรก: Hello World

มาเขียนโปรแกรมแรกกัน! พิมพ์โค้ดนี้ลงใน Script Editor:

function myFirstFunction() {
  // แสดงข้อความ Hello World
  Logger.log("สวัสดี Google Apps Script!");
  
  // แสดงกล่องข้อความ (ถ้าเปิดจาก Sheets)
  SpreadsheetApp.getUi().alert("สวัสดีครับ! นี่คือโปรแกรมแรกของผม");
}

วิธีรันโปรแกรม:

1

เลือกฟังก์ชัน myFirstFunction จาก dropdown ด้านบน

2

กดปุ่ม ▶ Run (หรือกด Ctrl+R)

3

ครั้งแรกจะขอ Permission — กด "Review Permissions" → เลือกบัญชี → "Allow"

4

ดูผลลัพธ์ที่ Execution Log ด้านล่าง

📋 Logger.log() — เครื่องมือตรวจสอบผลลัพธ์

Logger.log() คือคำสั่งที่ใช้ "พิมพ์" ข้อความออกมาดูใน Log เหมือนการจดบันทึกว่าโปรแกรมทำอะไรอยู่

function testLogger() {
  Logger.log("ข้อความธรรมดา");
  Logger.log(123);                    // ตัวเลข
  Logger.log(10 + 20);               // คำนวณ = 30
  Logger.log("ชื่อ: " + "ปัญญา");    // ต่อข้อความ
  
  var today = new Date();
  Logger.log("วันนี้: " + today);     // แสดงวันที่
}

💡 Tips

  • Logger.log() ใช้ตรวจสอบค่าขณะพัฒนา — เหมือน "ส่องดู" ว่าตัวแปรเก็บค่าอะไร
  • ดู Log ได้ที่ View → Logs หรือดูที่ Execution Log ด้านล่างหน้าจอ
  • ใช้ console.log() ก็ได้เหมือนกัน (GAS รองรับทั้งคู่)

📐 โครงสร้างพื้นฐานของ GAS

// 1. ฟังก์ชัน — กล่องคำสั่งที่เราสร้างขึ้น
function ชื่อฟังก์ชัน() {
  // คำสั่งที่ต้องการให้ทำ
}

// 2. ตัวแปร — กล่องเก็บข้อมูล
var name = "ปัญญา";          // ข้อความ (String)
var age = 30;                // ตัวเลข (Number)
var isActive = true;         // จริง/เท็จ (Boolean)

// 3. เงื่อนไข — ถ้า...แล้ว...
if (age >= 18) {
  Logger.log("เป็นผู้ใหญ่");
} else {
  Logger.log("เป็นเด็ก");
}

// 4. วนซ้ำ — ทำซ้ำหลายรอบ
for (var i = 0; i < 5; i++) {
  Logger.log("รอบที่ " + (i + 1));
}

🔬 Lab 1: ทดลองเขียนโปรแกรมแรก

เวลา: 15 นาที

ขั้นตอน:

1

เปิด Google Sheets → Extensions → Apps Script

2

ลบโค้ดเดิมทั้งหมด แล้วพิมพ์โค้ดต่อไปนี้:

function lab1_helloWorld() {
  // แสดงชื่อของคุณ
  var myName = "ใส่ชื่อคุณตรงนี้";
  Logger.log("สวัสดีครับ ผมชื่อ " + myName);
  
  // คำนวณอายุ
  var birthYear = 2537;  // เปลี่ยนเป็น พ.ศ. เกิดของคุณ
  var currentYear = 2569;
  var age = currentYear - birthYear;
  Logger.log("อายุ " + age + " ปี");
  
  // แสดงวันเวลาปัจจุบัน
  var now = new Date();
  Logger.log("ตอนนี้เวลา: " + now);
}
3

กด ▶ Run แล้วดูผลลัพธ์ที่ Execution Log

4

ลองเปลี่ยนชื่อและปีเกิด แล้วรันอีกครั้ง

🎯 ผลที่คาดหวัง: เห็นชื่อ อายุ และเวลาปัจจุบันใน Log

🔬 Lab 2: แสดง Alert ใน Google Sheets

เวลา: 10 นาที

function lab1_showAlert() {
  var ui = SpreadsheetApp.getUi();
  
  // ถามชื่อ
  var response = ui.prompt("กรุณาใส่ชื่อของคุณ:");
  var name = response.getResponseText();
  
  // แสดงข้อความต้อนรับ
  ui.alert("ยินดีต้อนรับ " + name + " สู่ Google Apps Script!");
  
  Logger.log("ผู้ใช้กรอกชื่อ: " + name);
}

หมายเหตุ: ฟังก์ชันนี้ต้องรันจาก Sheets เท่านั้น (ไม่ใช่ script.google.com) เพราะใช้ UI ของ Sheets
