5.1 Projects คืออะไร — Knowledge Base ส่วนตัวให้ AI

💡 สรุปง่ายๆ: Projects คือการสร้าง "ห้องทำงาน" ที่มีเอกสารอ้างอิงให้ Claude ใช้ตอบคำถามได้ตรงประเด็นกว่าการถามทั่วไป
Projects ทำงานอย่างไร?

ปกติเวลาเราถาม Claude คำถาม AI จะตอบจากความรู้ทั่วไปที่ถูกฝึกมา แต่ถ้าเราต้องการให้ Claude ตอบจาก เอกสารเฉพาะของหน่วยงาน เช่น ระเบียบ กฎหมาย คู่มือปฏิบัติงาน — Claude จะไม่รู้ข้อมูลเหล่านี้

Projects แก้ปัญหานี้โดยให้เราอัปโหลดเอกสารเข้าไปเป็น "ฐานข้อมูลความรู้" (Knowledge Base) ให้ Claude อ้างอิงตอบคำถาม

❌ ถาม Claude แบบปกติ

ถาม: "ระเบียบการจ่ายค่าวิทยากรของ สนพ. จ่ายได้ชั่วโมงละเท่าไร?"

Claude ตอบ: "ขออภัย ฉันไม่มีข้อมูลระเบียบเฉพาะของสำนักงานพัฒนาฝีมือแรงงาน แนะนำให้ตรวจสอบจากเว็บไซต์หน่วยงาน..."

→ ตอบไม่ได้ เพราะไม่มีเอกสารอ้างอิง

✅ ถาม Claude ใน Project (มีเอกสารอ้างอิง)

ถาม: "ระเบียบการจ่ายค่าวิทยากรของ สนพ. จ่ายได้ชั่วโมงละเท่าไร?"

Claude ตอบ: "ตามระเบียบกรมพัฒนาฝีมือแรงงาน ว่าด้วยค่าใช้จ่ายในการฝึกอบรม พ.ศ. 2563 ข้อ 12 กำหนดให้จ่ายค่าตอบแทนวิทยากร ชั่วโมงละ 600 บาท (ภายใน) และ 1,200 บาท (ภายนอก)..."

→ ตอบถูกต้อง อ้างอิงจากเอกสารที่ใส่ไว้ใน Project

5.2 สร้าง Project + ใส่เอกสารอ้างอิง

ขั้นตอนการสร้าง Project
1 เข้า claude.ai แล้วคลิก "Projects"

จากหน้าแรก ให้มองที่แถบด้านซ้าย (Sidebar) แล้วคลิก Projects

2 คลิก "Create Project" หรือ "New Project"

คลิกปุ่มสร้างโปรเจกต์ใหม่ที่มุมขวาบน

3 ตั้งชื่อ Project

ตั้งชื่อให้สื่อถึงงานที่จะทำ เช่น:

  • ผู้ช่วยงานฝึกอบรม
  • ที่ปรึกษาระเบียบราชการ
  • ผู้ช่วยวิเคราะห์งบประมาณ
  • FAQ สำนักงาน
4 เพิ่มเอกสารอ้างอิง (Knowledge Base)

คลิกปุ่ม Add content แล้วเลือก:

  • Upload files — อัปโหลดไฟล์ PDF, TXT, Code
  • Add text content — พิมพ์หรือวาง text ลงไป
  • Add web content — วาง URL ให้ Claude ดึงเนื้อหา
5 เริ่มใช้งาน — คลิก "Start chat"

เมื่อใส่เอกสารเสร็จ ก็เริ่มสนทนากับ Claude ได้เลย Claude จะอ้างอิงจากเอกสารที่ใส่ไว้

ประเภทเอกสารที่ใส่ได้
ประเภทไฟล์ ตัวอย่าง เหมาะสำหรับ
PDF ระเบียบ, คู่มือ, รายงาน, หนังสือเวียน เอกสารราชการ, กฎหมาย, แบบฟอร์ม
Text / TXT บันทึกการประชุม, รายงาน, FAQ ข้อมูลทั่วไป, เนื้อหาคัดลอกมา
Code .js, .py, .gs, .html, .css ให้ Claude วิเคราะห์โค้ด, หาข้อผิดพลาด
CSV / Data ข้อมูลสถิติ, ตาราง Excel ที่ export วิเคราะห์ข้อมูล, สร้างรายงาน
Markdown .md, README, Documentation เอกสารเทคนิค, คู่มือระบบ
💡 เคล็ดลับ: ใส่เอกสารที่เกี่ยวข้องกับงานที่จะทำให้ครบ แต่ไม่ต้องใส่ทุกอย่าง เลือกเฉพาะเอกสารที่สำคัญ เพราะยิ่งเอกสารตรงประเด็น Claude จะตอบได้ดีขึ้น
ตัวอย่างเอกสารที่ควรใส่สำหรับแต่ละงาน
📋 งานฝึกอบรม
  • หลักสูตรฝึกอบรมทั้งหมด
  • ระเบียบค่าใช้จ่าย
  • แบบฟอร์มที่เกี่ยวข้อง
  • ขั้นตอนการจัดฝึกอบรม
  • FAQ ที่ถามบ่อย
📊 วิเคราะห์งบประมาณ
  • แผนงบประมาณปีนี้
  • ข้อมูลผลเบิกจ่ายที่ผ่านมา
  • ระเบียบงบประมาณ
  • หลักเกณฑ์จัดสรร
  • ตัวชี้วัด KPI

5.3 Custom Instructions — กำหนดบทบาท AI ประจำโปรเจกต์

Custom Instructions คือการ "กำหนดตัวตน" ให้ Claude ก่อนเริ่มสนทนา เหมือนบอก Claude ว่า "คุณเป็นใคร ทำอะไร ตอบแบบไหน" ซึ่งจะทำให้ทุกครั้งที่เปิดแชทใน Project นี้ Claude จะจำบทบาทที่กำหนดไว้

📌 จุดแตกต่างจาก System Prompt: Custom Instructions จะ "ติดอยู่กับ Project" ตลอด ไม่ต้องพิมพ์ใหม่ทุกครั้ง ทุก conversation ใน Project จะใช้ Instructions เดียวกัน
วิธีตั้ง Custom Instructions
  1. เปิด Project ที่สร้างไว้
  2. คลิก Project settings หรือไอคอนรูปเฟือง
  3. หา Custom Instructions แล้วพิมพ์คำสั่งลงไป
  4. กด Save
ตัวอย่าง Custom Instructions สำหรับงานราชการ
ตัวอย่างที่ 1: ผู้ช่วยงานฝึกอบรม 
คุณเป็นผู้เชี่ยวชาญด้านการฝึกอบรมของสำนักงานพัฒนาฝีมือแรงงานสุรินทร์ 
กรมพัฒนาฝีมือแรงงาน กระทรวงแรงงาน

หน้าที่ของคุณ:
- ตอบคำถามเกี่ยวกับหลักสูตรฝึกอบรม
- ช่วยร่างเอกสาร เช่น โครงการฝึกอบรม แผนการสอน
- แนะนำขั้นตอนการจัดฝึกอบรมตามระเบียบ
- อ้างอิงข้อมูลจากเอกสารใน Project เท่านั้น

รูปแบบการตอบ:
- ตอบเป็นภาษาไทย ใช้คำราชการที่เหมาะสม
- ถ้ามีข้อมูลในเอกสารอ้างอิง ให้ระบุแหล่งที่มา
- ถ้าไม่มีข้อมูล ให้บอกตรงๆ ว่าไม่พบในเอกสาร
- ตอบกระชับ ตรงประเด็น ใช้หัวข้อย่อยเมื่อจำเป็น
ตัวอย่างที่ 2: ที่ปรึกษากฎหมายแรงงาน 
คุณเป็นที่ปรึกษาด้านกฎหมายแรงงานและระเบียบราชการ

บทบาท:
- ตอบคำถามเกี่ยวกับ พ.ร.บ. ส่งเสริมการพัฒนาฝีมือแรงงาน
- อธิบายระเบียบ คำสั่ง หนังสือเวียนที่เกี่ยวข้อง
- ช่วยตีความข้อกฎหมายให้เข้าใจง่าย
- แนะนำแนวทางปฏิบัติที่ถูกต้องตามกฎหมาย

ข้อจำกัด:
- ตอบเฉพาะข้อมูลที่มีในเอกสารอ้างอิงเท่านั้น
- ถ้าเป็นประเด็นซับซ้อน แนะนำให้ปรึกษานิติกรโดยตรง
- ไม่ให้คำปรึกษาทางกฎหมายที่อาจก่อให้เกิดความเสียหาย
- ใช้ภาษาที่เข้าใจง่าย หลีกเลี่ยงศัพท์กฎหมายที่ซับซ้อน
ตัวอย่างที่ 3: ผู้ช่วยเขียนหนังสือราชการ 
คุณเป็นผู้เชี่ยวชาญด้านการเขียนหนังสือราชการ 
สังกัดสำนักงานพัฒนาฝีมือแรงงานสุรินทร์

ความสามารถ:
- ร่างหนังสือราชการทุกประเภท (บันทึกข้อความ, หนังสือภายนอก, คำสั่ง, ประกาศ)
- ตรวจแก้ไขภาษาราชการให้ถูกต้อง
- จัดรูปแบบตามระเบียบสำนักนายกรัฐมนตรี ว่าด้วยงานสารบรรณ พ.ศ. 2526

กฎการเขียน:
- ใช้ภาษาราชการ สุภาพ ชัดเจน กระชับ
- อ้างอิงระเบียบหรือกฎหมายที่เกี่ยวข้องเสมอ
- ระบุ ที่, วันที่, เรื่อง, เรียน, อ้างถึง, สิ่งที่ส่งมาด้วย ให้ครบ
- เสนอตัวเลือกคำลงท้ายที่เหมาะสม
ตัวอย่างที่ 4: ผู้ช่วยวิเคราะห์ข้อมูลและงบประมาณ 
คุณเป็นนักวิเคราะห์ข้อมูลและงบประมาณ ประจำสำนักงานพัฒนาฝีมือแรงงานสุรินทร์

ความสามารถ:
- วิเคราะห์ข้อมูลผลการดำเนินงาน เทียบกับเป้าหมาย
- คำนวณอัตราเบิกจ่ายงบประมาณ
- สรุปรายงานเป็นตาราง กราฟ หรือ bullet points
- เสนอแนะแนวทางปรับปรุงผลงาน

รูปแบบผลลัพธ์:
- ใช้ตารางเมื่อเปรียบเทียบข้อมูล
- คำนวณเป็นเปอร์เซ็นต์และจำนวน
- สรุปจุดเด่น จุดที่ต้องปรับปรุง
- เสนอแนะ 3-5 ข้อ เรียงลำดับความสำคัญ
💡 เคล็ดลับการเขียน Instructions ที่ดี:
  1. ระบุบทบาทชัดเจน — Claude เป็นใคร ทำอะไร
  2. กำหนดขอบเขต — ตอบเรื่องอะไรได้/ไม่ได้
  3. กำหนดรูปแบบ — ตอบยาว/สั้น ใช้ภาษาแบบไหน
  4. ระบุข้อจำกัด — สิ่งที่ห้ามทำ เช่น ห้ามแต่งเรื่องเอง

5.4 ใช้ Project สำหรับงานประจำ

Projects มีประโยชน์มากสำหรับงานที่ต้องทำซ้ำๆ หรือต้องใช้ข้อมูลเฉพาะทาง ต่อไปนี้คือตัวอย่าง Use Cases ที่นำไปใช้ได้จริงในงานราชการ:

🤖 Use Case 1: FAQ Bot หน่วยงาน

ปัญหา: ประชาชนถามคำถามซ้ำๆ เช่น "สมัครฝึกอบรมอย่างไร" "ค่าใช้จ่ายเท่าไร"

วิธีทำ:

  • สร้าง Project "FAQ สนพ.สุรินทร์"
  • ใส่เอกสาร: รายการหลักสูตร, ขั้นตอนสมัคร, ค่าใช้จ่าย, สถานที่ติดต่อ
  • ตั้ง Instructions: "ตอบคำถามประชาชนที่สอบถามเกี่ยวกับบริการฝึกอบรม ใช้ภาษาสุภาพ เข้าใจง่าย"

ตัวอย่างการใช้:

ถาม: อยากเรียนช่างไฟฟ้า สมัครยังไง?

Claude ตอบ: สมัครได้ดังนี้ค่ะ
1. เตรียมสำเนาบัตรประชาชน
2. มาสมัครที่ สนพ.สุรินทร์ ถ.เลี่ยงเมือง
   หรือสมัครออนไลน์ที่ www.dsd.go.th
3. หลักสูตรช่างไฟฟ้า 120 ชม. 
   เปิดรับรุ่นถัดไป กค. 2569
4. ไม่เสียค่าใช้จ่าย
โทร 044-511xxx ในเวลาราชการ
📝 Use Case 2: ผู้ช่วยร่างเอกสาร

ปัญหา: ต้องร่างหนังสือราชการบ่อย ใช้เวลาเยอะกับรูปแบบและภาษา

วิธีทำ:

  • สร้าง Project "ร่างหนังสือราชการ"
  • ใส่เอกสาร: ตัวอย่างหนังสือราชการเดิม, ระเบียบสารบรรณ, คำขึ้นต้น/ลงท้าย
  • ตั้ง Instructions: "ร่างหนังสือราชการตามแบบฟอร์มที่กำหนด ใช้ภาษาราชการที่ถูกต้อง"

ตัวอย่างการใช้:

ถาม: ร่างบันทึกข้อความ ขออนุมัติจัดฝึกอบรม
หลักสูตร "ช่างแอร์เบื้องต้น" 30 คน
วันที่ 1-5 กค. 2569 งบ 150,000 บาท

Claude ตอบ: [ร่างบันทึกข้อความครบถ้วน
ตามรูปแบบที่ถูกต้อง พร้อมอ้างอิงระเบียบ]
⚖️ Use Case 3: ที่ปรึกษากฎหมาย

ปัญหา: ต้องตอบคำถามเกี่ยวกับกฎหมายแรงงาน/ฝึกอบรม บ่อยครั้ง

วิธีทำ:

  • สร้าง Project "กฎหมายและระเบียบ"
  • ใส่เอกสาร: พ.ร.บ. ส่งเสริมการพัฒนาฝีมือแรงงาน, ระเบียบค่าใช้จ่าย, ประกาศกระทรวง
  • ตั้ง Instructions: "ตอบคำถามกฎหมาย อ้างอิงมาตราที่เกี่ยวข้อง อธิบายเข้าใจง่าย"

ตัวอย่างการใช้:

ถาม: สถานประกอบการที่มีพนักงาน 100 คน 
ต้องส่งเงินสมทบกองทุนเท่าไร?

Claude ตอบ: ตาม พ.ร.บ. ส่งเสริมฯ มาตรา 29
สถานประกอบการที่มีลูกจ้าง 100 คนขึ้นไป
ต้องจัดฝึกอบรมไม่น้อยกว่า 50% ของลูกจ้าง
หากไม่จัด ต้องส่งเงินสมทบกองทุน...
📊 Use Case 4: วิเคราะห์งบประมาณ

ปัญหา: ต้องวิเคราะห์ข้อมูลงบประมาณ เปรียบเทียบผลงาน ทำรายงาน

วิธีทำ:

  • สร้าง Project "วิเคราะห์งบประมาณ 2569"
  • ใส่เอกสาร: แผนงบประมาณ, ข้อมูลผลเบิกจ่าย, ตัวชี้วัด KPI, เป้าหมาย

5.5 Workshop: สร้าง Project "ผู้ช่วยงานฝึกอบรม"

⏱️ เวลา: 45 นาที | รูปแบบ: ฝึกปฏิบัติจริงบน claude.ai
เป้าหมาย Workshop

ผู้เข้าอบรมจะได้สร้าง Project ที่ใช้งานได้จริง สำหรับช่วยตอบคำถามเกี่ยวกับการฝึกอบรมของหน่วยงาน

ขั้นตอนปฏิบัติ
1 สร้าง Project ใหม่
  1. เข้า claude.ai → คลิก Projects (แถบซ้าย)
  2. คลิก Create a project
  3. ตั้งชื่อ: ผู้ช่วยงานฝึกอบรม สนพ.สุรินทร์
  4. คลิก Create project
2 ใส่เอกสารอ้างอิง

คลิก Add content แล้วอัปโหลดหรือวางเอกสารต่อไปนี้:

# เอกสาร วิธีใส่
1 รายการหลักสูตรฝึกอบรมปี 2569 Upload PDF หรือ Add text
2 คู่มือขั้นตอนจัดฝึกอบรม Upload PDF
3 ระเบียบค่าใช้จ่ายในการฝึกอบรม Upload PDF
4 FAQ คำถามที่ถามบ่อย (เตรียมให้) Add text content
5 ข้อมูลติดต่อหน่วยงาน Add text content
3 ตั้ง Custom Instructions

คลิกไอคอนเฟือง (Project settings) แล้ววาง Instructions ต่อไปนี้:

คุณเป็นผู้ช่วยงานฝึกอบรมของสำนักงานพัฒนาฝีมือแรงงานสุรินทร์

หน้าที่:
1. ตอบคำถามเกี่ยวกับหลักสูตรฝึกอบรม (ประเภท, ระยะเวลา, คุณสมบัติผู้สมัคร)
2. แนะนำขั้นตอนการจัดฝึกอบรม
3. อธิบายระเบียบค่าใช้จ่ายที่เกี่ยวข้อง
4. ช่วยร่างเอกสารที่เกี่ยวกับการฝึกอบรม
5. สรุปข้อมูลสถิติผลการฝึกอบรม

กฎ:
- ตอบภาษาไทย สุภาพ เข้าใจง่าย
- อ้างอิงจากเอกสารใน Project เท่านั้น
- ถ้าไม่มีข้อมูล ให้บอกว่า "ไม่พบข้อมูลในเอกสาร กรุณาติดต่อเจ้าหน้าที่"
- ใช้ bullet points และตารางเมื่อเหมาะสม
- ระบุแหล่งอ้างอิง (ชื่อเอกสาร, หน้า) เมื่อทำได้
4 ทดสอบถามคำถาม

เปิด Chat ใน Project แล้วลองถามคำถามต่อไปนี้:

คำถามทดสอบ 1:

"ปีนี้มีหลักสูตรอะไรบ้าง ที่เปิดรับสมัครอยู่?"

คำถามทดสอบ 2:

"อยากจัดฝึกอบรมหลักสูตร 30 ชม. ต้องเตรียมเอกสารอะไรบ้าง?"

คำถามทดสอบ 3:

"ค่าตอบแทนวิทยากรภายนอก จ่ายได้ชั่วโมงละเท่าไร?"

คำถามทดสอบ 4:

"สรุปผลการฝึกอบรมไตรมาส 1 เป็นตาราง"

5 ปรับปรุง Instructions (ถ้าจำเป็น)

ถ้าผลลัพธ์ยังไม่ถูกใจ ลองปรับ Instructions เช่น:

  • เพิ่ม "ตอบสั้นๆ ไม่เกิน 200 คำ" ถ้าตอบยาวเกินไป
  • เพิ่ม "ใส่อิโมจิหัวข้อ" ถ้าอยากให้อ่านง่ายขึ้น
  • เพิ่ม "ถ้าเป็นคำถามที่ไม่เกี่ยวกับฝึกอบรม ให้ปฏิเสธสุภาพ"
📝 แบบฝึกหัดเพิ่มเติม

แบบฝึกหัด 5.1: สร้าง Project อีก 1 อัน สำหรับงานที่ท่านรับผิดชอบ (เลือก 1 ข้อ)

  1. Project "ที่ปรึกษางานพัสดุ" — ใส่ระเบียบพัสดุ + ตัวอย่างเอกสาร
  2. Project "ผู้ช่วยงานบุคคล" — ใส่ระเบียบบุคคล + แบบฟอร์มลา
  3. Project "สรุปรายงานประชุม" — ใส่ตัวอย่างรายงานเดิม + รูปแบบมาตรฐาน

แบบฝึกหัด 5.2: ทดสอบ Project ที่สร้าง โดยถามคำถาม 5 ข้อ แล้วให้คะแนนความถูกต้องของ Claude (1-5 ดาว)

แบบฝึกหัด 5.3: แลกเปลี่ยนกับเพื่อนร่วมกลุ่ม ว่าแต่ละคนสร้าง Project อะไร ตั้ง Instructions อย่างไร
