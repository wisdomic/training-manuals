📖 สารบัญ — 10 บทเรียน

🌐 ภาค 1: Claude AI (Web) — วันที่ 1-2

🟢 วันที่ 1 — ภาคเช้า (3 ชั่วโมง)

บทที่ 1: รู้จัก Claude AI
  • 1.1 AI กับงานราชการ — ทำไมต้องใช้ AI
  • 1.2 Claude คืออะไร — Anthropic, Constitutional AI, เปรียบเทียบ AI
  • 1.3 สมัครและเริ่มใช้งาน Claude.ai
  • 1.4 Interface — ส่วนประกอบหน้าจอ
  • 1.5 แผน Free vs Pro vs Team
🟡 วันที่ 1 — ภาคบ่าย (3 ชั่วโมง)

บทที่ 2: เทคนิค Prompt — สั่งงาน AI อย่างมืออาชีพ
  • 2.1 Prompt Engineering คืออะไร
  • 2.2 โครงสร้าง Prompt ที่ดี (สูตร RCCFO)
  • 2.3 เทคนิค Zero-shot, Few-shot, Chain-of-Thought
  • 2.4 ใช้ Claude วิเคราะห์ข้อมูล/ตาราง
  • 2.5 ใช้ Claude เขียนเอกสารราชการ
  • 2.6 ใช้ Claude แปลภาษา/สรุปเอกสาร/ตอบคำถาม
🔵 วันที่ 2 — ภาคเช้า (3 ชั่วโมง)

บทที่ 3: Claude กับงานเอกสาร + ข้อมูล
  • 3.1 อัปโหลดไฟล์ (PDF, Word, Excel, รูปภาพ)
  • 3.2 วิเคราะห์เอกสาร PDF — สรุป, ดึงข้อมูล, เปรียบเทียบ
  • 3.3 วิเคราะห์ตาราง Excel/CSV — สถิติ, กราฟ, แนวโน้ม
  • 3.4 อ่านรูปภาพ (Vision) — OCR, อธิบายรูป, อ่านกราฟ
  • 3.5 Artifacts — สร้างเอกสาร, Code, แผนภูมิ แบบ Interactive
🟣 วันที่ 2 — ภาคบ่าย (3 ชั่วโมง)

บทที่ 4: Claude สำหรับงานสร้างสรรค์
  • 4.1 เขียนเนื้อหา — บทความ, โพสต์, ข่าวประชาสัมพันธ์
  • 4.2 สร้าง Presentation Outline
  • 4.3 สร้างแบบสอบถาม/แบบประเมิน
  • 4.4 วางแผนโครงการ/กิจกรรม
  • 4.5 สร้าง Infographic Content

🤝 ภาค 2: Co-Work (Kiro) — วันที่ 3

🟢 วันที่ 3 — ภาคเช้า (3 ชั่วโมง)

บทที่ 5: Projects + Custom Instructions + ฟีเจอร์ขั้นสูง
  • 5.1 Projects — จัดกลุ่มงานเป็นโปรเจกต์
  • 5.2 Custom Instructions — ตั้งค่า AI ให้เหมาะกับงาน
  • 5.3 Extended Thinking — ให้ AI คิดลึกขึ้น
  • 5.4 เทคนิค Advanced Prompt Patterns
  • 5.5 Workshop: ตั้งค่า Claude ให้เหมาะกับงานราชการ
🟡 วันที่ 3 — ภาคบ่าย (3 ชั่วโมง)

บทที่ 6: Kiro Co-Work — ทำงานร่วมกับ AI อย่างเป็นระบบ
  • 6.1 Kiro คืออะไร — IDE ที่ทำงานร่วมกับ AI
  • 6.2 ติดตั้งและเริ่มต้นใช้งาน Kiro
  • 6.3 Steering Files — ควบคุม AI ให้ทำตามกฎ
  • 6.4 Spec-driven Development — Requirements → Design → Tasks
  • 6.5 Workshop: สร้างระบบด้วย Kiro

⌨️ ภาค 3: Claude Code (CLI) — วันที่ 4-5

🔵 วันที่ 4 — ภาคเช้า (3 ชั่วโมง)

บทที่ 7: เริ่มต้น Claude Code
  • 7.1 Claude Code คืออะไร — CLI tool สำหรับเขียนโค้ด
  • 7.2 ติดตั้ง Claude Code (Node.js + npm)
  • 7.3 ตั้งค่า API Key + เริ่มใช้งาน
  • 7.4 คำสั่งพื้นฐาน — สร้างไฟล์, แก้ไข, อ่านโค้ด
  • 7.5 เปรียบเทียบ Claude Code vs Kiro vs Cursor vs Copilot
🟣 วันที่ 4 — ภาคบ่าย (3 ชั่วโมง)

บทที่ 8: ใช้ Claude Code สร้างระบบ
  • 8.1 ออกแบบระบบด้วย Claude Code (อธิบาย requirement)
  • 8.2 Workshop: สร้าง Web App (HTML + CSS + JS)
  • 8.3 Workshop: สร้างระบบ GAS + Sheets
  • 8.4 Git Integration — จัดการ version
  • 8.5 Debug + แก้ Bug ด้วย Claude Code
🟢 วันที่ 5 — ภาคเช้า (3 ชั่วโมง)

บทที่ 9: Claude Code ขั้นสูง
  • 9.1 Custom Commands — สร้างคำสั่งเอง
  • 9.2 CLAUDE.md — กำหนดกฎโปรเจกต์
  • 9.3 Multi-file editing — แก้หลายไฟล์พร้อมกัน
  • 9.4 MCP Connections — เชื่อมต่อ API ภายนอก
  • 9.5 ใช้ Claude Code กับ Docker/Cloud
🟡 วันที่ 5 — ภาคบ่าย (3 ชั่วโมง)

บทที่ 10: สรุป + Workshop + นำเสนอ
  • 10.1 เปรียบเทียบ Claude AI vs Co-Work vs Code
  • 10.2 เลือกเครื่องมือให้เหมาะกับงาน (Decision Tree)
  • 10.3 Workshop: ผู้เข้าอบรมเลือกสร้างงาน/ระบบ (4 ระดับ)
  • 10.4 นำเสนอผลงาน (5 นาที/ทีม + เกณฑ์ประเมิน)
  • 10.5 Tips + แหล่งเรียนรู้ + ปิดอบรม
