📖 AI เสียงและดนตรี บทที่ 8: เทคนิค Prompt

✍️ บทที่ 8: เทคนิค Prompt สำหรับ AI เสียง

เขียน Prompt อย่างมืออาชีพ — ได้เพลงและเสียงตรงใจทุกครั้ง

🎯 วัตถุประสงค์การเรียนรู้
  • เข้าใจหลักการเขียน Prompt สำหรับ AI เสียงและเพลง
  • รู้จัก Keywords สำคัญ: Genre, Mood, Tempo, Instrument
  • สามารถเขียน Prompt สร้างเพลงได้ตรงใจ
  • สามารถเขียน Prompt สำหรับเสียงพากย์ที่มีอารมณ์ต่างๆ

📘 8.1 หลักการเขียน Prompt ที่ดี

Prompt คือ "คำสั่ง" ที่เราบอก AI ว่าต้องการอะไร ยิ่ง Prompt ชัดเจน ผลลัพธ์ยิ่งตรงใจ หลักการสำคัญ:

📐 สูตร Prompt เพลง

Genre + Mood + Tempo + Instrument + Vocal + Theme + Duration

ตัวอย่าง: "Acoustic pop (Genre), inspiring and upbeat (Mood), 120 BPM (Tempo), guitar and piano (Instrument), female vocal (Vocal), about helping disabled community (Theme), 2 minutes (Duration)"
📐 สูตร Prompt เสียงพากย์

Voice Type + Language + Emotion + Speed + Context

ตัวอย่าง: เลือก "Professional female (Voice Type), Thai (Language)" แล้วปรับ Stability ต่ำสำหรับ emotion สูง, Speed 0.9x สำหรับเนื้อหาจริงจัง

📘 8.4 Tempo Keywords — ความเร็วจังหวะ

BPM Keywords ลักษณะ เหมาะกับ
60-80 slow, ballad, gentle ช้า นุ่มนวล เพลงบัลลาด, เพลงกล่อมนอน

📘 8.5 ตัวอย่าง Prompt เพลงสำหรับงานจริง

🎬 เพลงเปิดวิดีโอองค์กร (Intro 15 วินาที):
"Cinematic orchestral intro, 15 seconds, building energy, inspiring, corporate, no vocal, starts soft and builds to powerful climax"
📢 เพลงประชาสัมพันธ์กิจกรรม:
"Upbeat Thai pop, cheerful and friendly, acoustic guitar and ukulele, female vocal singing about community event, festival vibe, 2 minutes"
📊 Background เพลงสำหรับ Presentation:
"Soft ambient corporate music, piano and light strings, calm and professional, instrumental, 3 minutes, loop-friendly, minimal drums"
🏆 เพลงมอบรางวัล/พิธีการ:
"Grand orchestral celebration music, triumphant fanfare, brass and strings, majestic, 30 seconds, building to climax, award ceremony style"
📹 เพลงปิดวิดีโอ (Outro):
"Gentle acoustic guitar outro, warm and thankful mood, fade out slowly, 20 seconds, no vocal, calm ending"
🧘 เพลงสำหรับกิจกรรมผ่อนคลาย:
"Meditation music, ambient pads, nature sounds, flowing water, extremely calm, 5 minutes, no drums, healing frequency, 432 Hz"

📘 8.6 เทคนิค Prompt เสียงพากย์

สำหรับ ElevenLabs และ CapCut AI Voice เทคนิคการเขียนข้อความให้ AI อ่านออกเสียงได้ดี:

📝 เทคนิคเขียนสคริปต์สำหรับ AI Voice
เทคนิค วิธีทำ ตัวอย่าง
หยุดหายใจ ใส่ จุด (.) หรือ comma (,) "สวัสดีครับ, ยินดีต้อนรับ."
เน้นคำ ใส่ ... ก่อนคำสำคัญ "เราพร้อม... ดูแลทุกท่าน"
หยุดนาน ใส่ — "ศูนย์บริการ — เปิดทุกวัน"
อ่านตัวเลข เขียนเป็นคำ "เปิดแปดจุดสามสิบ น."
ชื่อภาษาอังกฤษ เขียนเป็นคำอ่าน "อีเลฟเวน แล็บส์"
📝 ตัวอย่างสคริปต์ที่ดี vs ไม่ดี:

❌ ไม่ดี: "ศูนย์บริการคนพิการจังหวัดสุรินทร์เปิดให้บริการวันจันทร์ถึงศุกร์เวลา8.30-16.30น.โทร044-511234"

✅ ดี: "ศูนย์บริการคนพิการ จังหวัดสุรินทร์, เปิดให้บริการวันจันทร์ถึงศุกร์. เวลาแปดจุดสามสิบ ถึง สิบหกจุดสามสิบ นาฬิกา. โทร ศูนย์สี่สี่, ห้าหนึ่งหนึ่ง, สองสามสี่."

📘 8.7 Prompt Template สำเร็จรูป

📋 Template สำหรับ Copy & Paste

เปลี่ยนข้อความใน [...] ตามต้องการ:

🎵 Template เพลง Intro:
"[Genre] intro music, [duration] seconds, [mood], building energy, [instrument], no vocal, starts [soft/quiet] and builds to [powerful/energetic] climax"
🎵 Template Background Music:
"Soft [genre] background music, [instrument], [mood] and professional, instrumental, [duration] minutes, loop-friendly, minimal drums"
🎵 Template เพลงมีเนื้อร้อง:
"[Genre], [mood], [tempo] BPM, [instrument], [male/female] vocal, song about [topic], [language] lyrics, [duration]"
🎵 Template เพลงปิด (Outro):
"Gentle [instrument] outro, [mood] mood, fade out slowly, [duration] seconds, no vocal, calm ending, [genre]"

🔬 Lab 8: ฝึกเขียน Prompt เพลงและเสียง

เวลา: 45 นาที

กิจกรรมที่ 1: เขียน Prompt เพลงตามโจทย์ (15 นาที)

  1. เขียน Prompt สร้างเพลง Intro 15 วินาที สำหรับวิดีโอองค์กรคนพิการ
  2. เขียน Prompt สร้าง Background Music สำหรับ Presentation 3 นาที
  3. เขียน Prompt สร้างเพลงประชาสัมพันธ์กิจกรรมวันคนพิการ
  4. นำ Prompt ไปสร้างจริงใน Suno หรือ Udio
  5. ฟังผลลัพธ์ แล้วปรับ Prompt ให้ดีขึ้น

กิจกรรมที่ 2: ปรับปรุง Prompt ให้ดีขึ้น (15 นาที)

  1. ใช้ Prompt เดิมจากกิจกรรมที่ 1
  2. เพิ่ม Keywords: เพิ่ม Mood, Tempo, Instrument ที่ชัดเจนขึ้น
  3. สร้างเพลงใหม่ เปรียบเทียบกับเวอร์ชันเดิม
  4. ปรับอีกครั้ง จนได้ผลลัพธ์ที่พอใจ
  5. จดบันทึก Prompt ที่ดีที่สุดไว้ใช้ในอนาคต

กิจกรรมที่ 3: เขียนสคริปต์เสียงพากย์ (15 นาที)

  1. เขียนสคริปต์พากย์วิดีโอ 30 วินาที แนะนำองค์กร
  2. ใส่เครื่องหมายวรรคตอนให้เหมาะสม (จุด, comma, ...)
  3. นำไปสร้างเสียงใน ElevenLabs
  4. ฟังแล้วปรับข้อความ (เว้นจังหวะ, เขียนตัวเลขเป็นคำ)
  5. สร้างใหม่จนได้ผลลัพธ์ที่เป็นธรรมชาติ
💡 Tips
  • Prompt ภาษาอังกฤษ: AI เพลงเข้าใจ Prompt อังกฤษได้ดีกว่าไทยมาก
  • เริ่มจากง่าย: เริ่มจาก 3-4 Keywords แล้วค่อยเพิ่มรายละเอียด
  • ใช้ Gemini ช่วย: ถาม Gemini "ช่วยเขียน Prompt สร้างเพลงสำหรับ..." ได้
  • เก็บ Prompt Library: สร้างไฟล์รวม Prompt ที่ใช้ได้ผลดี ไว้ใช้ซ้ำ
  • ทดลองหลายครั้ง: Prompt เดียวกัน ผลลัพธ์ต่างกันทุกครั้ง ลองจนพอใจ
  • Negative Prompt: ระบุสิ่งที่ไม่ต้องการ เช่น "no drums" หรือ "no vocal"

📋 สรุปบทที่ 8

  • Prompt ที่ดีต้องมี: Genre + Mood + Tempo + Instrument + Vocal + Theme
  • ใช้ Keywords ภาษาอังกฤษจะได้ผลดีที่สุด
  • สคริปต์เสียงพากย์ต้องใส่วรรคตอน และเขียนตัวเลขเป็นคำ
  • ทดลองหลายครั้ง ปรับ Prompt ทีละนิด จนได้ผลที่ต้องการ
  • เก็บ Prompt ที่ดีไว้เป็น Library สำหรับใช้ซ้ำ
