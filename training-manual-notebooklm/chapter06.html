6.2 Workshop สุดท้าย — เลือก 1 จาก 8 โจทย์

⏱️ เวลา: 90 นาที | รูปแบบ: กลุ่ม 4-5 คน (เลือกโจทย์ 1 ข้อ) | นำเสนอ: 5 นาที/ทีม

เลือกทำ 1 โจทย์ ตามระดับความท้าทายที่ต้องการ:

🟢 ง่าย
โจทย์ 1: สร้าง FAQ หน่วยงาน

อัปโหลดคู่มือบริการ + ข้อมูลหน่วยงาน → สร้าง FAQ 15 ข้อ สำหรับตอบประชาชน

  • Sources: คู่มือบริการ, แผ่นพับ
  • Output: FAQ 15 ข้อ พร้อม citation
  • เครื่องมือ: Notebook Guide → FAQ + Chat
🟢 ง่าย
โจทย์ 2: สรุประเบียบ 5 ฉบับ

อัปโหลดระเบียบ/หนังสือเวียน 5 ฉบับ → สร้างสรุปรวมเป็น 1 หน้า

  • Sources: ระเบียบ/กฎหมาย 5 ฉบับ (PDF)
  • Output: Study Guide + Briefing Doc
  • เครื่องมือ: Notebook Guide ทุกอัน
🟡 กลาง
โจทย์ 3: Briefing Doc + Podcast

สร้าง Briefing Doc 1 หน้า + Podcast AI สำหรับให้ ผอ. ฟัง

  • Sources: รายงานผลงาน + แผนฯ 3 ฉบับ
  • Output: Briefing Doc + Audio Overview
  • เครื่องมือ: Briefing Doc + Audio + Customize
🟡 กลาง
โจทย์ 4: Study Guide หลักสูตร

สร้างสื่อเตรียมการสอน: Study Guide + Quiz 10 ข้อ + FAQ

  • Sources: เอกสารประกอบการบรรยาย 3-5 ฉบับ
  • Output: Study Guide + Quiz + FAQ
  • เครื่องมือ: Chat + Notebook Guide
🔴 ยาก
โจทย์ 5: Q&A Bot จากเอกสาร 20+ ชิ้น

สร้าง Notebook ที่ตอบคำถามได้ครอบคลุมจากเอกสาร 20+ sources

  • Sources: เอกสารหน่วยงาน 20+ ฉบับ
  • Output: ทดสอบถาม 20 คำถาม ตรวจ citation ทุกข้อ
  • เครื่องมือ: Chat + Source management
🔴 ยาก
โจทย์ 6: สรุปนโยบายเทียบผลงาน

เปรียบเทียบนโยบาย/แผนฯ กับผลงานจริง วิเคราะห์ช่องว่าง

  • Sources: แผนยุทธศาสตร์ + รายงานผลงาน
  • Output: Gap Analysis + ข้อเสนอแนะ + Timeline
  • เครื่องมือ: Chat (วิเคราะห์) + Timeline + Briefing
🟣 ท้าทาย
โจทย์ 7: Podcast สรุปผลงานปี + สื่อ PR

สร้าง Podcast AI สรุปผลงานทั้งปี + ใช้เป็นสื่อ PR ของหน่วยงาน

  • Sources: รายงานประจำปี + สรุปผลงานเด่น + ภาพรวม
  • Output: Audio Overview + Customize 3 เวอร์ชัน + เลือกอันดีสุด
  • เครื่องมือ: Audio Overview + Customize + Briefing Doc
🟣 ท้าทาย

ออกแบบระบบ Notebook 5 อัน เป็น Knowledge Base ของหน่วยงาน

  • Sources: เอกสารหลากหลายหมวด 30+ ฉบับ
  • Output: 5 Notebooks จัดหมวด + FAQ แต่ละหมวด + Briefing รวม
  • เครื่องมือ: ทุกเครื่องมือ + Multi-notebook strategy

เกณฑ์ให้คะแนน 5 ด้าน

1. ความครบถ้วน ทำตามโจทย์ครบ, มี Sources เพียงพอ, ผลลัพธ์สมบูรณ์ 25

6.4 Tips 10 ข้อ + แหล่งเรียนรู้

💡 10 เคล็ดลับการใช้ NotebookLM ให้ได้ผลดี

1️⃣ แบ่ง Notebook ตามหัวข้อ

อย่ารวมทุกอย่างใน Notebook เดียว แบ่งตามภารกิจ/หมวดหมู่ จะได้คำตอบแม่นยำกว่า

2️⃣ ตรวจ Citation ทุกครั้ง

อย่าเชื่อคำตอบ 100% คลิกดู citation ตรวจว่าต้นทางถูกต้อง โดยเฉพาะตัวเลข/วันที่

3️⃣ ถามเจาะจง ดีกว่าถามกว้าง

"ค่าวิทยากรเบิกได้เท่าไหร่?" ดีกว่า "สรุปทุกอย่างให้หน่อย"

4️⃣ ใช้ checkbox เลือก Source

เปิดเฉพาะ source ที่เกี่ยวข้องกับคำถาม จะได้คำตอบที่โฟกัสกว่า

5️⃣ ระบุรูปแบบผลลัพธ์

บอกว่าต้องการ "ตาราง" "5 ข้อ" "1 ย่อหน้า" "bullet points" จะได้ output ที่ใช้ได้ทันที

6️⃣ ถามต่อยอดจากคำตอบ

NotebookLM จำบริบทสนทนา ถามต่อเนื่องได้ เช่น "ขยายข้อ 3 ให้หน่อย"

7️⃣ Save to Note ผลลัพธ์ดีๆ

คำตอบที่ดี บันทึกเป็น Note ไว้ ไม่ต้องถามใหม่ทุกครั้ง

8️⃣ อัปเดต Sources เมื่อมีเอกสารใหม่

เพิ่มเอกสารใหม่เข้า Notebook ได้ตลอด ไม่ต้องสร้างใหม่

9️⃣ ใช้ Customize กับ Audio Overview

กำหนดทิศทาง Podcast จะได้ผลลัพธ์ตรงประเด็นกว่าปล่อย auto

🔟 ระวังเรื่อง Privacy

อย่าอัปโหลดเอกสารลับราชการ ใช้เฉพาะเอกสารที่เปิดเผยได้
