🎯 วัตถุประสงค์การเรียนรู้

  • สมัครและเริ่มใช้งาน ChatGPT ได้ด้วยตนเอง

1.1 AI คืออะไร?

ปัญญาประดิษฐ์ — เทคโนโลยีที่เปลี่ยนโลกการทำงาน

ความหมายของ AI (Artificial Intelligence)

AI (ปัญญาประดิษฐ์) คือ ระบบคอมพิวเตอร์ที่ถูกออกแบบให้สามารถทำงานที่ปกติต้องใช้สติปัญญาของมนุษย์ เช่น การเรียนรู้ การตัดสินใจ การเข้าใจภาษา การจดจำรูปภาพ และการแก้ปัญหา

ประเภทของ AI

ประเภท คำอธิบาย ตัวอย่าง สถานะ
Narrow AI
(AI เฉพาะทาง)		 ทำงานได้ดีในเรื่องเดียว ChatGPT, Siri, Google Translate, Face ID มีแล้ว
General AI
(AI ทั่วไป)		 ฉลาดเทียบเท่ามนุษย์ ทำได้ทุกเรื่อง ยังไม่มี (กำลังพัฒนา) อนาคต
Super AI
(AI เหนือมนุษย์)		 ฉลาดกว่ามนุษย์ทุกด้าน ทฤษฎี/ภาพยนตร์ ทฤษฎี

AI ที่เราใช้อยู่ทุกวัน (โดยไม่รู้ตัว)

  • Google Search — AI จัดอันดับผลค้นหาให้ตรงใจ
  • YouTube / Netflix — AI แนะนำวิดีโอที่น่าสนใจ
  • Facebook / TikTok — AI เลือกโพสต์มาแสดงใน Feed
  • Google Maps — AI คำนวณเส้นทาง + คาดเดาเวลาถึง
  • Siri / Google Assistant — AI เข้าใจเสียงพูดและตอบคำถาม
  • ธนาคาร — AI ตรวจจับธุรกรรมผิดปกติ (Fraud Detection)

Machine Learning & Deep Learning

Machine Learning (ML) = AI ที่ "เรียนรู้" จากข้อมูล แทนที่จะเขียนกฎทุกอย่าง

Deep Learning = ML ที่ใช้ "โครงข่ายประสาท" (Neural Network) หลายชั้น ทำให้เข้าใจข้อมูลซับซ้อนได้

LLM (Large Language Model) = โมเดลภาษาขนาดใหญ่ที่ฝึกจากข้อความมหาศาล สามารถเข้าใจและสร้างภาษาได้ — ChatGPT คือ LLM

📸 ภาพ: แผนภาพแสดงความสัมพันธ์ AI → Machine Learning → Deep Learning → LLM → ChatGPT

1.2 ChatGPT คืออะไร?

ทำความรู้จักกับ AI ที่ทรงพลังที่สุดตัวหนึ่งในโลก

OpenAI — บริษัทผู้สร้าง ChatGPT

OpenAI ก่อตั้งปี 2015 โดย Sam Altman และทีมงาน (รวมถึง Elon Musk ในช่วงแรก) มีเป้าหมายพัฒนา AI ที่ปลอดภัยและเป็นประโยชน์

  • GPT = Generative Pre-trained Transformer (โมเดลที่สร้างข้อความจากการฝึกล่วงหน้า)
  • Chat = โหมดสนทนาแบบถาม-ตอบ
  • ChatGPT = Application ที่ให้คนทั่วไปใช้งาน GPT ได้ง่ายๆ ผ่านหน้าเว็บ/แอป

เวอร์ชันของ ChatGPT (2025)

โมเดล ลักษณะ ใช้กับแผน เหมาะกับ
GPT-4o รุ่นท็อป ฉลาดสุด เร็ว รับได้ทั้งข้อความ/ภาพ/เสียง Free (จำกัด) / Plus งานทุกประเภท
GPT-4o mini รุ่นเบา เร็วมาก ฉลาดพอใช้ Free (ไม่จำกัด) งานง่ายๆ ที่ต้องการเร็ว
o1 / o1-pro รุ่น "คิดลึก" ใช้เวลานาน แต่ตอบซับซ้อนได้ดี Plus / Pro คณิตศาสตร์ วิทยาศาสตร์ โค้ดซับซ้อน
DALL·E 3 สร้างรูปภาพจากข้อความ Plus ออกแบบ สื่อ โปสเตอร์

ChatGPT ทำอะไรได้บ้าง?

✅ ทำได้ดี
  • เขียน/ร่างเอกสาร ทุกประเภท
  • สรุปข้อมูลยาวๆ ให้กระชับ
  • แปลภาษา (ดีกว่า Google Translate)
  • วิเคราะห์ข้อมูล/ตาราง
  • เขียนโปรแกรม/โค้ด
  • ตอบคำถามความรู้ทั่วไป
  • ระดมความคิด/Brainstorm
  • สร้างรูปภาพ (DALL·E)
  • อ่านไฟล์ PDF/Word/Excel/รูปภาพ
❌ ข้อจำกัด
  • อาจ "แต่งเรื่อง" (Hallucination)
  • ข้อมูลอาจไม่ใช่ปัจจุบันที่สุด
  • ไม่มี "ความรู้สึก" หรือ "ความเห็น" จริง
  • คำนวณตัวเลขซับซ้อนอาจผิด
  • ไม่สามารถเข้าถึงอินเทอร์เน็ตแบบ real-time (ยกเว้นเปิด Browse)
  • ไม่เข้าใจบริบทองค์กรเฉพาะ (ถ้าไม่บอก)
  • อาจตอบแบบ "เห็นด้วย" มากเกินไป
💡 จำง่ายๆ

ChatGPT เหมือน "เลขาฯ อัจฉริยะ" ที่ร่างงานเร็วมาก แต่คุณต้อง ตรวจทาน ก่อนใช้เสมอ — เหมือนมีลูกน้องเก่งแต่ต้องตรวจงานก่อนส่ง

1.3 สมัครบัญชี ChatGPT + ตั้งค่าเริ่มต้น

ขั้นตอนสมัครบัญชีและเริ่มใช้งาน ChatGPT

1 เปิดเว็บไซต์ ChatGPT

เปิด Browser (Chrome/Edge) แล้วพิมพ์ https://chat.openai.com หรือค้นหา "ChatGPT" ใน Google

2 คลิก "Sign Up"

หน้าแรกจะมีปุ่ม "Sign up" (สมัคร) และ "Log in" (เข้าสู่ระบบ) — คลิก Sign up

3 เลือกวิธีสมัคร

มี 3 ทางเลือก:

  • Continue with Google — แนะนำ! ง่ายที่สุด ใช้ Gmail ที่มีอยู่
  • Continue with Microsoft — ใช้บัญชี Outlook/Hotmail
  • Continue with Apple — ใช้ Apple ID
  • Email — สมัครด้วย Email + ตั้งรหัสผ่านเอง
4 กรอกชื่อ + วันเกิด

กรอกชื่อ-นามสกุล (ภาษาอังกฤษ) และวันเกิด เพื่อยืนยันอายุ 13+ ปี

5 ยืนยัน Email (ถ้าสมัครด้วย Email)

เปิดกล่อง Email → คลิกลิงก์ยืนยัน → กลับมาที่หน้าเว็บ

6 เริ่มใช้งานได้ทันที!

เข้าสู่หน้า Chat → พิมพ์คำถามแรก เช่น "สวัสดีครับ ChatGPT"

📸 ภาพหน้าจอ: หน้าสมัครสมาชิก ChatGPT — ปุ่ม "Continue with Google"

ตั้งค่าเบื้องต้นหลังสมัครเสร็จ

การตั้งค่า ตำแหน่ง แนะนำ
ภาษาหน้าจอ Settings → General → Language ตั้งเป็น "ไทย" หรือ "Auto-detect"
Custom Instructions Settings → Personalization ใส่ "ตอบเป็นภาษาไทย สุภาพ กระชับ"
Theme Settings → General → Theme Light / Dark / System ตามชอบ
Data Controls Settings → Data Controls ตรวจสอบว่าเปิด/ปิด Chat History
💡 เคล็ดลับสำหรับข้าราชการ

แนะนำใช้ Email ส่วนตัว (@gmail.com) ในการสมัคร เพราะ Email ราชการบางหน่วยงานอาจถูกบล็อก หรือใช้ Google Account ที่มีอยู่แล้วเพื่อความสะดวก

1.4 ทัวร์หน้าจอ ChatGPT

ส่วนประกอบหน้าจอที่ต้องรู้จัก

📸 ภาพหน้าจอ: Interface ของ ChatGPT — แสดง Sidebar, Chat area, และปุ่มต่างๆ

1. Sidebar (แถบด้านซ้าย)

  • New Chat — เริ่มบทสนทนาใหม่
  • ประวัติสนทนา — รายการ Chat ที่เคยคุยไว้ (เรียงตามวันที่)
  • Explore GPTs — ร้านค้า GPT เฉพาะทาง (GPT Store)
  • Projects — จัดกลุ่มงานเป็นโปรเจกต์ (Plus)

2. Chat Area (พื้นที่สนทนา)

  • ช่องพิมพ์ — อยู่ด้านล่าง พิมพ์แล้วกด Enter ส่ง
  • ประวัติข้อความ — คำถามของคุณ (ขวา) + คำตอบ ChatGPT (ซ้าย)
  • ปุ่ม Copy — คัดลอกคำตอบ
  • ปุ่ม Regenerate — ให้ ChatGPT ตอบใหม่
  • ปุ่ม Like/Dislike — ให้ feedback คำตอบ

3. ปุ่มและไอคอนสำคัญ

ปุ่ม/ไอคอน ฟีเจอร์ หน้าที่
📎 Attach file แนบไฟล์ (PDF, รูป, Word, Excel, CSV, ข้อความ)
🌐 Search the web ให้ ChatGPT ค้นเว็บแบบ real-time
🎨 Create image สร้างรูปด้วย DALL·E 3 (Plus)
💻 Code Code Interpreter / รันโค้ด Python (Plus)
🎙️ Voice พูดคุยด้วยเสียง (Advanced Voice Mode)
📷 Camera / Vision ถ่ายรูป/อัปโหลดรูปให้ AI ดู
⚙️ Settings ตั้งค่าบัญชี, Theme, Memory, Data Controls

4. GPT Store — ร้านค้า GPT เฉพาะทาง

GPT Store คือคลัง "Custom GPT" ที่คนอื่นสร้างไว้ ใช้งานได้ฟรี เช่น:

  • Canva GPT — ออกแบบกราฟิกผ่าน ChatGPT
  • Data Analyst — วิเคราะห์ข้อมูลเชิงลึก
  • Consensus — ค้นหางานวิจัยทางวิชาการ
  • Write for Me — ช่วยเขียนบทความ/เอกสาร

💡 คำแนะนำสำหรับหน่วยงานราชการ
  • เริ่มต้น: ใช้แผน Free ทดลองก่อน — ได้ GPT-4o mini ไม่จำกัด + GPT-4o จำกัด/วัน
  • ใช้จริงทุกวัน: Plus คุ้มค่า ($20 ≈ 700 บาท/เดือน) ได้ DALL·E + Code Interpreter
  • ทั้งหน่วยงาน: Team เหมาะถ้ามี 5+ คน มี Admin + ข้อมูลไม่ถูกนำไป Train

1.6 ความสามารถของ ChatGPT — 7 โหมดหลัก

ภาพรวมทุกอย่างที่ ChatGPT ทำได้

โหมด คำอธิบาย ตัวอย่างการใช้งาน แผน
💬 Text ถาม-ตอบ ข้อความ ร่างเอกสาร, สรุป, แปล, ถามความรู้ ทุกแผน
🌐 Browse ค้นหาข้อมูลจากอินเทอร์เน็ต หาข่าวล่าสุด, ราคาหุ้น, สภาพอากาศ ทุกแผน
🎨 DALL·E สร้างรูปภาพจากข้อความ โปสเตอร์, โลโก้, Infographic, ภาพประกอบ Plus+
💻 Code เขียน/รันโค้ด Python วิเคราะห์ CSV, สร้างกราฟ, คำนวณ Plus+
📎 Files อัปโหลดไฟล์ให้อ่าน สรุป PDF, อ่านรูป, วิเคราะห์ Excel ทุกแผน
🎙️ Voice สนทนาด้วยเสียง ฝึกภาษา, ถามตอบแบบ hands-free ทุกแผน (จำกัด)
👁️ Vision อ่าน/วิเคราะห์รูปภาพ OCR, อธิบายกราฟ, อ่านป้าย, แก้สมการ ทุกแผน
⚠️ ข้อควรระวังเรื่องข้อมูลราชการ
  • ห้ามส่งข้อมูลลับ / ข้อมูลส่วนบุคคล / ข้อมูลมั่นคง ขึ้น AI ทุกตัว
  • ข้อมูลที่พิมพ์อาจถูกเก็บไว้ในเซิร์ฟเวอร์ต่างประเทศ
  • ใช้ข้อมูลทั่วไป ข้อมูลสาธารณะ หรือข้อมูลตัวอย่างเท่านั้น
  • ตรวจสอบนโยบายหน่วยงานก่อนใช้ AI กับงานราชการ
  • ปิด "Improve the model for everyone" ใน Settings → Data Controls ถ้าไม่อยากให้ข้อมูลถูกนำไป Train

✏️ แบบฝึกหัดบทที่ 1

  1. สมัคร ChatGPT — สมัครบัญชีด้วย Google Account ให้สำเร็จ
  2. ตั้งค่า — ตั้ง Language, Custom Instructions ("ตอบภาษาไทย กระชับ สุภาพ")
  3. ทดลอง 5 คำถาม:
    • "สวัสดีครับ ChatGPT แนะนำตัวหน่อย"
    • "อธิบายเรื่อง AI ให้เข้าใจง่ายๆ เหมือนอธิบายให้เด็กฟัง"
    • "ช่วยร่างหนังสือเชิญประชุมสั้นๆ"
    • "แปลคำว่า 'สำนักงานพัฒนาฝีมือแรงงาน' เป็นอังกฤษ"
    • "สรุปข้อดีของ ChatGPT ในงานราชการ 5 ข้อ"
  4. สำรวจ GPT Store — เปิด Explore GPTs ค้นหา "Data Analysis" แล้วลองใช้
  5. ทดลอง Voice — กดไอคอนไมค์แล้วพูด "ช่วยอธิบายเรื่อง AI หน่อย"
