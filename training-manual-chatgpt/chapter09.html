9.1 Google Apps Script — เขียนสคริปต์อัตโนมัติ

ใช้ ChatGPT เขียนสคริปต์ที่ทำงานร่วมกับ Google Workspace

Google Apps Script คืออะไร?

Google Apps Script (GAS) คือภาษาสคริปต์ที่ใช้ JavaScript เป็นพื้นฐาน ทำงานร่วมกับ Google Workspace ได้อย่างสมบูรณ์ เช่น Google Sheets, Gmail, Drive, Calendar, Forms — สามารถสร้างระบบอัตโนมัติได้โดยไม่ต้องติดตั้งซอฟต์แวร์เพิ่มเติม

ตัวอย่างงานที่ทำได้ด้วย Google Apps Script

# งานที่ทำได้ รายละเอียด ระดับความยาก
1 ส่งอีเมลอัตโนมัติ ส่งอีเมลแจ้งเตือนจาก Sheet ตามเงื่อนไข เช่น ใกล้ครบกำหนด ง่าย
2 สร้างเอกสาร PDF ดึงข้อมูลจาก Sheet มาสร้างเป็น PDF อัตโนมัติ เช่น ใบประกาศ ใบเสร็จ กลาง
3 อ่านข้อมูลจาก Google Sheet ดึงข้อมูล คำนวณ สรุปผล แล้วเขียนกลับลง Sheet ง่าย
4 สร้าง Google Form อัตโนมัติ สร้างฟอร์มจาก Sheet โดยกำหนดคำถามและตัวเลือกจากข้อมูลที่มี กลาง
5 เชื่อมต่อ API ภายนอก ดึงข้อมูลจาก API ภายนอก เช่น ค่าเงิน สภาพอากาศ หรือส่งข้อมูลไป Line สูง
Prompt ที่ 1 — สร้างสคริปต์ส่งอีเมลอัตโนมัติ

ช่วยเขียน Google Apps Script ที่ทำงานดังนี้:

1. อ่านข้อมูลจาก Google Sheet ชื่อ "รายชื่อ" ที่มีคอลัมน์: ชื่อ, อีเมล, วันครบกำหนด

2. ตรวจสอบว่ามีรายการไหนที่วันครบกำหนดอยู่ภายใน 7 วัน

3. ส่งอีเมลแจ้งเตือนไปยังคนที่ใกล้ครบกำหนด

4. บันทึกสถานะว่าส่งแล้วในคอลัมน์ "สถานะ"

ต้องมี comment อธิบายทุกขั้นตอน เขียนเป็นภาษาไทย

Prompt ที่ 2 — สคริปต์อ่านข้อมูลจาก Sheet แล้วสรุป

ช่วยเขียน Google Apps Script ที่:

1. เปิด Google Sheet ID: "xxxxxxxx" ชีต "ข้อมูลฝึกอบรม"

2. อ่านข้อมูลทั้งหมด (คอลัมน์: ชื่อ, หลักสูตร, คะแนน, ผ่าน/ไม่ผ่าน)

3. นับจำนวนผู้ผ่านและไม่ผ่าน คำนวณคะแนนเฉลี่ย

4. เขียนสรุปผลลง Sheet ใหม่ชื่อ "สรุป"

ใส่ comment อธิบายทุกบรรทัด เหมาะสำหรับผู้เริ่มต้น

9.2 สร้างเว็บไซต์ HTML/CSS ด้วย ChatGPT

ให้ ChatGPT สร้างหน้าเว็บสมบูรณ์พร้อมใช้งาน

ChatGPT สามารถสร้างหน้าเว็บ HTML/CSS ที่สมบูรณ์ได้ในคำสั่งเดียว — รวมโครงสร้าง, การจัดวาง, สีสัน, และ Responsive Design สิ่งสำคัญคือต้องระบุรายละเอียดให้ชัดเจนว่าต้องการอะไรบ้าง

เทคนิคสั่งสร้างเว็บให้ได้ผลดี

  • ระบุ Responsive — บอกว่าต้องรองรับมือถือและคอมพิวเตอร์
  • ระบุสไตล์ — Modern, Minimal, Corporate, ราชการ ฯลฯ
  • ระบุสีธีม — บอกสีหลัก สีรอง หรือให้ใช้สีของหน่วยงาน
  • ระบุส่วนประกอบ — Navbar, Hero, Features, Footer ฯลฯ
  • ใช้ Bootstrap หรือ Tailwind — ระบุ CSS Framework ที่ต้องการ
Prompt ที่ 1 — หน้า Landing Page หน่วยงาน

สร้างหน้าเว็บ Landing Page สำหรับ "สำนักงานพัฒนาฝีมือแรงงานสุรินทร์" โดย:

- ใช้ Bootstrap 5, ภาษาไทย, Responsive

- สีธีม: น้ำเงินเข้ม #003366 กับทอง #FFD700

- ส่วนประกอบ: Navbar + Logo, Hero Section (ชื่อหน่วยงาน + คำขวัญ), บริการหลัก 4 ข้อ (ฝึกอบรม/ทดสอบ/ส่งเสริม/ให้คำปรึกษา), ข่าวสาร 3 รายการ, ติดต่อ (แผนที่ + เบอร์โทร), Footer

- เขียนเป็นไฟล์เดียว (HTML + CSS inline) พร้อมใช้งานทันที

Prompt ที่ 2 — แบบฟอร์มลงทะเบียนฝึกอบรม

สร้างหน้าแบบฟอร์มลงทะเบียนฝึกอบรมออนไลน์ โดย:

- ใช้ Bootstrap 5, Responsive, ดีไซน์สะอาด

- ฟิลด์: ชื่อ-นามสกุล, เลขบัตรประชาชน, หน่วยงาน, ตำแหน่ง, อีเมล, โทรศัพท์, หลักสูตรที่สมัคร (Dropdown), อาหาร (ปกติ/มังสวิรัติ/อิสลาม)

- มี Validation: ตรวจเลขบัตร 13 หลัก, อีเมลถูกต้อง, เบอร์โทร 10 หลัก

- ปุ่มส่ง + ปุ่มล้างข้อมูล

- แสดง SweetAlert เมื่อส่งสำเร็จ

9.3 Code Interpreter — Python ใน ChatGPT

ใช้ Python รันโค้ดตรงใน ChatGPT เพื่อวิเคราะห์ข้อมูลและสร้างกราฟ

สิ่งที่ Code Interpreter ทำได้

ความสามารถ ตัวอย่างการใช้งาน ไลบรารีที่ใช้
สร้างกราฟ/แผนภูมิ กราฟแท่ง วงกลม เส้น จากข้อมูลที่อัปโหลด Matplotlib, Seaborn, Plotly
แปลงไฟล์ CSV → Excel, PDF → ข้อความ, รูป → ขนาดอื่น Pandas, Pillow, ReportLab
คำนวณซับซ้อน สถิติ, สมการ, พยากรณ์, Machine Learning NumPy, SciPy, Scikit-learn
จัดการข้อมูล กรอง, เรียง, Pivot, รวมไฟล์, ล้างข้อมูล Pandas, OpenPyXL
Prompt ที่ 1 — สร้างกราฟจากข้อมูล

จากไฟล์ Excel ที่แนบ (ข้อมูลผู้เข้าอบรมรายเดือน) ช่วย:

1. สร้างกราฟแท่งแสดงจำนวนผู้อบรมแต่ละเดือน

2. สร้างกราฟวงกลมแสดงสัดส่วนตามหลักสูตร

3. สร้างกราฟเส้นเปรียบเทียบเป้า vs ผลงาน

ใช้ฟอนต์ที่รองรับภาษาไทย สีสวยงาม มี Title และ Legend

ส่งออกเป็นไฟล์ PNG ความละเอียดสูง (300 DPI)

Prompt ที่ 2 — วิเคราะห์และล้างข้อมูล

จากไฟล์ CSV ที่แนบ ช่วย:

1. ตรวจสอบข้อมูลซ้ำ (Duplicate) และลบออก

2. ตรวจหาค่าว่าง (Missing) แจ้งว่ามีกี่เซลล์

3. คำนวณสถิติพื้นฐาน (ค่าเฉลี่ย, มัธยฐาน, ค่าสูงสุด-ต่ำสุด)

4. สร้าง Summary Table สรุปผล

5. ส่งออกเป็นไฟล์ Excel ที่ล้างข้อมูลแล้ว พร้อม Sheet สรุป

9.4 Debug & แก้ไขโค้ด

วิธีส่งโค้ดที่มีปัญหาให้ ChatGPT ช่วยหาข้อผิดพลาดและแก้ไข

3 ขั้นตอนส่งโค้ดให้ ChatGPT แก้

ขั้นที่ 1: วางโค้ดที่มีปัญหา — Copy โค้ดทั้งหมด (หรือเฉพาะส่วนที่ error) วางใน ChatGPT พร้อมระบุภาษาโปรแกรม เช่น "นี่คือ Google Apps Script" หรือ "นี่คือ HTML/JavaScript"
ขั้นที่ 2: บอก Error Message — วาง Error Message ที่ได้จากระบบ เช่น "TypeError: Cannot read properties of undefined" หรือ "Exception: You do not have permission" — ยิ่งบอก Error ชัดเจน AI ยิ่งแก้ได้แม่น
ขั้นที่ 3: อธิบายสิ่งที่ต้องการ — บอกว่าโค้ดควรทำอะไร (Expected Behavior) vs สิ่งที่เกิดขึ้นจริง (Actual Behavior) เช่น "ต้องการให้ส่งอีเมล แต่มันส่งไม่ออก" หรือ "ควรแสดงตารางแต่หน้าว่างเปล่า"
Prompt ที่ 1 — แก้ Google Apps Script Error

Google Apps Script ด้านล่างมีปัญหา ช่วยแก้ไขให้หน่อย:

[วางโค้ด]

Error ที่ได้: "TypeError: Cannot read properties of null (reading 'getRange')"

สิ่งที่ต้องการ: สคริปต์ควรอ่านข้อมูลจาก Sheet "รายชื่อ" แล้วส่งอีเมลแจ้งเตือน

ช่วยอธิบายสาเหตุของ Error และแก้ไขโค้ดให้ด้วย

Prompt ที่ 2 — แก้ HTML ที่แสดงผลผิด

HTML ด้านล่างแสดงผลผิดพลาด ช่วยแก้ไข:

[วางโค้ด HTML/CSS]

ปัญหา: ตารางล้นออกนอกจอมือถือ, ปุ่มกดไม่ทำงาน, รูปไม่แสดง

ต้องการ: หน้าเว็บ Responsive, ปุ่มกดแล้วแสดง Alert, รูปแสดงครบ

ช่วยแก้โค้ดพร้อมอธิบายว่าแก้ตรงไหนบ้างและทำไม

9.5 Excel VBA & Macro

สร้าง Macro อัตโนมัติใน Excel ด้วย AI

VBA (Visual Basic for Applications) คือภาษาสคริปต์ใน Microsoft Office ที่ช่วยสร้างระบบอัตโนมัติ — ไม่ต้องเก่งเขียนโค้ด เพียงบอก ChatGPT ว่าต้องการทำอะไร AI จะเขียนโค้ด VBA ให้พร้อมใช้

Prompt ที่ 1 — Macro จัดรูปแบบตาราง

ช่วยเขียน Excel VBA Macro ที่ทำงานดังนี้:

1. เลือกข้อมูลทั้งหมดใน Sheet ปัจจุบัน

2. จัดรูปแบบเป็นตาราง: หัวตารางสีน้ำเงินเข้มตัวอักษรขาว, แถวสลับสี (ขาว/ฟ้าอ่อน)

3. ปรับความกว้างคอลัมน์อัตโนมัติ (AutoFit)

4. เพิ่มกรอบตาราง (Border) ทุกเซลล์

5. ฟอนต์ TH SarabunPSK ขนาด 14

Prompt ที่ 2 — Macro ส่งอีเมลจาก Excel

ช่วยเขียน Excel VBA Macro ที่:

1. อ่านรายชื่อจากคอลัมน์ A (ชื่อ), คอลัมน์ B (อีเมล), คอลัมน์ C (เนื้อหา)

2. ส่งอีเมลผ่าน Outlook ไปทีละคน

3. หัวข้อ: "แจ้งเตือนจากสำนักงานพัฒนาฝีมือแรงงานสุรินทร์"

4. บันทึกสถานะ "ส่งแล้ว" พร้อมวันที่ ในคอลัมน์ D

ต้องมี Error Handling กรณีอีเมลไม่ถูกต้อง

9.6 สร้าง Automation / Bot

ตัวอย่างระบบอัตโนมัติที่ ChatGPT ช่วยเขียนได้

ตัวอย่างระบบ Automation ที่สร้างได้

ระบบ รายละเอียด เทคโนโลยี ระดับ
Line Bot ตอบ FAQ บอทตอบคำถามที่พบบ่อยอัตโนมัติ เช่น เวลาทำการ สถานที่ ค่าธรรมเนียม Google Apps Script + Line API กลาง
Email Auto-Reply ตอบอีเมลอัตโนมัติตามเงื่อนไข เช่น ส่งเอกสาร ยืนยันการรับสมัคร Google Apps Script + Gmail ง่าย
Scheduler / Reminder แจ้งเตือนงานอัตโนมัติตามตาราง เช่น ก่อนประชุม 30 นาที Google Apps Script + Calendar ง่าย
Form → Sheet → Email กรอกฟอร์ม → บันทึกลง Sheet → ส่งอีเมลยืนยัน → แจ้ง Admin Google Forms + GAS + Gmail กลาง
Prompt — สร้าง Line Bot ตอบ FAQ

ช่วยเขียน Google Apps Script สำหรับ Line Bot ตอบ FAQ อัตโนมัติ:

1. รับ Webhook จาก Line Messaging API

2. อ่านคำถาม-คำตอบจาก Google Sheet "FAQ" (คอลัมน์ A: คำถาม, คอลัมน์ B: คำตอบ)

3. เมื่อผู้ใช้ส่งข้อความ ให้ค้นหาคำที่ตรงกันใน Sheet แล้วตอบกลับ

4. ถ้าไม่พบคำตอบ ให้ตอบว่า "ขออภัย ไม่พบข้อมูล กรุณาติดต่อ 044-511073"

5. มี Log บันทึกทุกข้อความที่เข้ามาลง Sheet "Log"

✏️ แบบฝึกหัดบทที่ 9

  1. สร้าง Google Apps Script ง่ายๆ 1 ตัว

    ใช้ ChatGPT เขียนสคริปต์ที่อ่านข้อมูลจาก Google Sheet แล้วทำอะไรบางอย่าง เช่น ส่งอีเมล, สร้างรายงาน, หรือคำนวณสรุปผล — ทดสอบรันใน Apps Script Editor จริง

  2. สร้างหน้าเว็บ HTML 1 หน้า

    สั่ง ChatGPT สร้างหน้าเว็บสำหรับหน่วยงานหรือโครงการของท่าน ต้อง Responsive ดูได้ทั้งมือถือและคอมพิวเตอร์ — บันทึกเป็น .html แล้วเปิดดูในเบราว์เซอร์

  3. ส่งโค้ดที่มี Error ให้ ChatGPT แก้ไข

    หาโค้ดที่มีปัญหา (จากงานจริง หรือสร้างขึ้นมา) ส่งให้ ChatGPT พร้อม Error Message — ตรวจสอบว่า AI แก้ได้ถูกต้องหรือไม่

  4. สร้าง Excel Macro 1 ตัว

    ใช้ ChatGPT เขียน VBA Macro สำหรับงานที่ทำซ้ำบ่อยๆ ใน Excel เช่น จัดรูปแบบตาราง, สรุปข้อมูล, หรือส่งอีเมล — ทดสอบรันใน Excel จริง
