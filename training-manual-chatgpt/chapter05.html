5.1 สร้างสูตร Excel ด้วย ChatGPT

ให้ AI เขียนสูตร VLOOKUP, IF, SUMIFS, INDEX-MATCH แทนเรา

สูตร Excel ที่ใช้บ่อยในงานราชการ

หลายคนรู้ว่าอยากได้ผลลัพธ์อะไร แต่ไม่รู้ว่าต้องเขียนสูตรอย่างไร — ChatGPT ช่วยได้ทันที เพียงแค่อธิบายสิ่งที่ต้องการเป็นภาษาคน

สูตร ใช้ทำอะไร ตัวอย่างงานราชการ
VLOOKUP ค้นหาข้อมูลจากตารางอ้างอิง ค้นชื่อหลักสูตรจากรหัส, ดึงเงินเดือนจากตำแหน่ง
IF / IFS ตัดสินใจตามเงื่อนไข ตัดเกรดผู้เข้าอบรม, แจ้งสถานะผ่าน/ไม่ผ่าน
SUMIFS รวมยอดตามเงื่อนไขหลายข้อ รวมงบประมาณเฉพาะหมวดฝึกอบรม ไตรมาส 1
INDEX-MATCH ค้นหาขั้นสูง ยืดหยุ่นกว่า VLOOKUP ค้นข้อมูลจากหลายเงื่อนไข, ค้นจากขวาไปซ้าย
COUNTIFS นับจำนวนตามเงื่อนไข นับจำนวนผู้ผ่านการทดสอบ แยกตามจังหวัด

ตัวอย่างตารางข้อมูล

รหัส ชื่อ-สกุล หลักสูตร คะแนนก่อนอบรม คะแนนหลังอบรม ผลการประเมิน

T002สมหญิง รักงานExcel ขั้นสูง6078?
T003ประสิทธิ์ ทำดีAI เบื้องต้น3555?
T004วิไล สุขสมPower BI7092?
T005อนุชา พัฒนาExcel ขั้นสูง5048?
Prompt ที่ 1 — สร้างสูตร IF ตัดเกรด

ฉันมีตาราง Excel ข้อมูลผู้เข้าอบรม มีคอลัมน์ "คะแนนหลังอบรม" อยู่ที่ E2

ต้องการสูตรตัดเกรดในคอลัมน์ F2 ตามเกณฑ์:

- 80 ขึ้นไป = "ดีมาก"

- 60-79 = "ดี"

- 50-59 = "พอใช้"

- ต่ำกว่า 50 = "ต้องปรับปรุง"

Prompt ที่ 2 — สร้างสูตร SUMIFS รวมงบตามเงื่อนไข

ฉันมีตาราง Excel งบประมาณ คอลัมน์ A=หมวดรายจ่าย, B=ไตรมาส, C=จำนวนเงิน

ต้องการสูตร SUMIFS ที่รวมเงินเฉพาะ:

- หมวด "ฝึกอบรม" (คอลัมน์ A)

- ไตรมาส "Q1" (คอลัมน์ B)

ช่วยเขียนสูตรพร้อมอธิบายแต่ละส่วนของสูตรด้วย

5.2 วิเคราะห์ข้อมูลจากตาราง

Copy-Paste ตารางให้ AI อ่าน แล้วสั่งวิเคราะห์ได้ทันที

วิธี Copy-Paste ข้อมูลให้ ChatGPT

ChatGPT สามารถอ่านข้อมูลจากตารางที่เรา Copy มาวางได้ทันที — ไม่ต้องอัปโหลดไฟล์ เพียงเลือกข้อมูลใน Excel → Ctrl+C → วางใน ChatGPT

💡 เคล็ดลับ
  • ถ้าข้อมูลมากกว่า 50 แถว ให้ตัดมาเฉพาะส่วนที่ต้องการวิเคราะห์
  • บอก ChatGPT ว่าแต่ละคอลัมน์คืออะไร เพื่อให้ AI เข้าใจบริบท
  • ข้อมูลที่วางจะอยู่ในรูปแบบ Tab-separated (แยกด้วย Tab)

5.3 Code Interpreter — วิเคราะห์ไฟล์และสร้างกราฟ

อัปโหลด CSV/Excel ให้ ChatGPT วิเคราะห์และสร้างกราฟอัตโนมัติ

Code Interpreter คืออะไร?

Code Interpreter (หรือ Advanced Data Analysis) คือความสามารถของ ChatGPT Plus ที่สามารถรันโค้ด Python เพื่อวิเคราะห์ข้อมูล สร้างกราฟ และประมวลผลไฟล์ได้จริง — ไม่ใช่แค่ "เดา" คำตอบ

✅ ข้อดีของ Code Interpreter
  • แม่นยำ — คำนวณจากข้อมูลจริง ไม่ใช่การเดา
  • สร้างกราฟได้ — Bar chart, Pie chart, Line chart, Heatmap
  • รองรับหลายไฟล์ — CSV, Excel (.xlsx), PDF, รูปภาพ
  • ดาวน์โหลดผลลัพธ์ได้ — ส่งออกเป็น Excel, CSV, รูปภาพ

ขั้นตอนใช้งาน Code Interpreter

ขั้นที่ 1: อัปโหลดไฟล์ — คลิกไอคอน 📎 (แนบไฟล์) ในช่อง ChatGPT แล้วเลือกไฟล์ CSV หรือ Excel ที่ต้องการวิเคราะห์ (รองรับไฟล์สูงสุด 512 MB)
ขั้นที่ 2: สั่งวิเคราะห์ — พิมพ์คำสั่งบอกว่าต้องการให้ทำอะไร เช่น "วิเคราะห์ข้อมูลนี้ สรุปภาพรวม แสดงสถิติเบื้องต้น และสร้าง Bar Chart เปรียบเทียบแต่ละหลักสูตร"
ขั้นที่ 3: ดาวน์โหลดผลลัพธ์ — ChatGPT จะแสดงกราฟ ตาราง และสรุปผล พร้อมลิงก์ดาวน์โหลดไฟล์ผลลัพธ์ (รูปภาพ .png หรือ Excel .xlsx)

5.4 Pivot Table & สรุปข้อมูล

สั่งให้ AI จัดกลุ่ม สรุปภาพรวมข้อมูล แบบ Pivot Table

ทำ Pivot Table ด้วย ChatGPT

แทนที่จะสร้าง Pivot Table ใน Excel เอง เราสามารถสั่งให้ ChatGPT จัดกลุ่มและสรุปข้อมูลให้ได้ทันที — เหมาะสำหรับคนที่ไม่ถนัด Pivot Table หรือต้องการผลลัพธ์เร็วๆ

ช่วยสร้างตาราง Pivot สรุปดังนี้:

1. สรุปจำนวนผู้เข้าอบรมรวม แยกตามหลักสูตร

2. สรุปงบประมาณรวม แยกตามหลักสูตร

3. คำนวณอัตราการผ่านเฉลี่ย (%) ของแต่ละหลักสูตร

💡 คำสั่งเพิ่มเติมที่ใช้ได้

  • "เปรียบเทียบ..." — ระหว่างไตรมาส / ระหว่างปี / ระหว่างหลักสูตร
  • "หาแนวโน้ม..." — เพิ่มขึ้น-ลดลง / pattern / ค่าผิดปกติ (outlier)

5.5 Google Sheets Formula

สูตร QUERY, IMPORTRANGE, ARRAYFORMULA สำหรับ Google Sheets

สูตร Google Sheets ที่ต่างจาก Excel

Google Sheets มีสูตรเฉพาะที่ Excel ไม่มี — ChatGPT สามารถสร้างสูตรเหล่านี้ได้ เพียงบอกว่าใช้ Google Sheets

สูตร ใช้ทำอะไร ตัวอย่างการใช้งาน
QUERY ค้นหา กรอง จัดเรียงข้อมูล (คล้าย SQL) ดึงรายชื่อผู้อบรมที่คะแนน > 80 เรียงจากมากไปน้อย
IMPORTRANGE ดึงข้อมูลข้าม Google Sheets ดึงรายชื่อจาก Sheet ฝ่ายบุคคลมาใช้ใน Sheet ฝึกอบรม
ARRAYFORMULA ใช้สูตร 1 ตัวกับทุกแถวอัตโนมัติ คำนวณ % ให้ทุกแถวโดยไม่ต้อง copy สูตรลง
FILTER กรองข้อมูลแบบ dynamic แสดงเฉพาะผู้อบรมที่ "ไม่ผ่าน" ทั้งหมด
UNIQUE + SORT หาค่าไม่ซ้ำ + เรียงลำดับ รายชื่อหลักสูตรทั้งหมดไม่ซ้ำ เรียง ก-ฮ
Prompt สร้างสูตร QUERY ใน Google Sheets

ฉันใช้ Google Sheets มีข้อมูลอยู่ใน Sheet "ฝึกอบรม" ช่วง A1:G100

คอลัมน์: A=รหัส, B=ชื่อ, C=หลักสูตร, D=วันที่อบรม, E=คะแนนก่อน, F=คะแนนหลัง, G=สถานะ

ต้องการสูตร QUERY ที่:

1. ดึงเฉพาะคน "ไม่ผ่าน" (คอลัมน์ G = "ไม่ผ่าน")

2. แสดงเฉพาะคอลัมน์ ชื่อ, หลักสูตร, คะแนนหลัง

3. เรียงตามคะแนนหลังจากน้อยไปมาก

5.6 สร้าง Dashboard Data

จัดข้อมูลให้อยู่ในรูปแบบพร้อมนำเสนอ

Dashboard Data คืออะไร?

Dashboard Data คือการจัดข้อมูลให้อยู่ในรูปแบบที่พร้อมนำเสนอ — ไม่ว่าจะเป็นตาราง KPI, ข้อมูลสรุปผู้บริหาร, หรือตัวเลขสำหรับทำกราฟ โดยให้ ChatGPT ช่วยจัดข้อมูลดิบให้เป็นระเบียบ

สิ่งที่ ChatGPT ช่วยได้

  • จัดข้อมูลดิบเป็นตาราง KPI — เป้า vs ผลงาน vs % ความสำเร็จ
  • สรุปตัวเลขสำคัญ — Top 3, ค่าเฉลี่ย, แนวโน้ม
  • จัดรูปแบบสำหรับ Power BI / Google Data Studio — ข้อมูลที่พร้อม import
  • สร้าง Summary Card — ตัวเลขหลัก + การเปลี่ยนแปลง + สัญลักษณ์ ↑↓
Prompt สร้าง Dashboard Data

ช่วยจัดข้อมูลเป็น Dashboard สำหรับรายงานผู้บริหาร:

1. ตาราง KPI (เป้าหมาย / ผลงาน / %ความสำเร็จ / สถานะ 🟢🟡🔴)

2. สรุป Top 3 หลักสูตรที่มีผู้เข้าร่วมมากที่สุด

3. สรุปตัวเลขสำคัญ 4 ตัว (จำนวนหลักสูตร, ผู้เข้าร่วมรวม, อัตราผ่านเฉลี่ย, งบที่ใช้)

4. ข้อเสนอแนะ 3 ข้อสำหรับปรับปรุงในไตรมาสหน้า

Comparisonหลายหมวดข้อมูลสำหรับ Bar Chart

✏️ แบบฝึกหัดบทที่ 5

  1. สร้างสูตร Excel 3 สูตร ด้วย ChatGPT

    ใช้ข้อมูลงานจริงของท่าน (หรือข้อมูลตัวอย่างข้างต้น) สั่งให้ ChatGPT เขียนสูตร:

    • สูตร IF หรือ IFS สำหรับตัดสินผล (ผ่าน/ไม่ผ่าน/ดีมาก/ดี/พอใช้)
    • สูตร SUMIFS หรือ COUNTIFS รวมยอดตามเงื่อนไข
    • สูตร VLOOKUP หรือ INDEX-MATCH ค้นหาข้อมูล
  2. Copy ข้อมูลจริงให้ ChatGPT วิเคราะห์

    เปิดไฟล์ Excel ที่มีอยู่ เลือกข้อมูล 10-20 แถว Copy แล้ววางใน ChatGPT พร้อมสั่ง:

    • สรุปภาพรวมของข้อมูล
    • หาค่าผิดปกติ (ถ้ามี)
    • ให้ข้อเสนอแนะ
  3. ใช้ Code Interpreter สร้างกราฟ 1 ชิ้น

    อัปโหลดไฟล์ CSV หรือ Excel ให้ ChatGPT แล้วสั่งสร้างกราฟแท่งหรือกราฟวงกลม พร้อมดาวน์โหลดผลลัพธ์

  4. สร้าง Dashboard Data สำหรับรายงานผู้บริหาร

    ใช้ข้อมูลผลงานหน่วยงาน สั่งให้ ChatGPT จัดรูปแบบเป็น:

    • ตาราง KPI พร้อมสถานะสี (🟢🟡🔴)
    • สรุปตัวเลขสำคัญ 4 ตัว
    • ข้อเสนอแนะ 3 ข้อ
