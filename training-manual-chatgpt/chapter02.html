2.1 Prompt คืออะไร?

ทำไมถึงสำคัญกว่าเครื่องมือ

ความหมายของ Prompt

Prompt คือ ข้อความ/คำสั่งที่เราพิมพ์ให้ AI เพื่อบอกว่าต้องการอะไร — เปรียบเสมือนการสั่งงานลูกน้อง ถ้าสั่งชัด ลูกน้องทำได้ดี ถ้าสั่งกว้างๆ ลูกน้องก็เดาไปเอง

หลักการง่ายๆ: Prompt ดี = ผลลัพธ์ดี | Prompt แย่ = ผลลัพธ์แย่

ทำไม Prompt ถึงสำคัญกว่าเครื่องมือ?

  • AI ทุกตัว (ChatGPT, Gemini, Claude) ใช้ Prompt เป็นตัวขับเคลื่อน
  • คนที่เขียน Prompt ดี จะได้ผลลัพธ์ดีกว่าคนที่จ่ายเงินซื้อแผน Plus แต่เขียน Prompt ไม่เป็น
  • Prompt Engineering คือทักษะที่ติดตัว — ใช้ได้กับ AI ทุกตัว ทุกยุค
  • เปรียบเหมือนการถ่ายรูป: กล้องดีอย่างเดียวไม่พอ ต้องรู้วิธีจัดมุมด้วย
❌ Prompt แย่

"เขียนหนังสือ"

→ AI ไม่รู้ว่าหนังสืออะไร เขียนให้ใคร ยาวแค่ไหน
✅ Prompt ดี

"คุณเป็นเจ้าหน้าที่ธุรการราชการ ช่วยร่างบันทึกข้อความ เรื่องขออนุมัติจัดซื้อคอมพิวเตอร์ 5 เครื่อง ถึงผู้อำนวยการ รูปแบบหนังสือราชการ"

→ AI รู้บทบาท รู้บริบท รู้คำสั่ง รู้รูปแบบ

2.2 โครงสร้าง Prompt ที่ดี — สูตร RCIF

สูตรลับในการเขียน Prompt ให้ได้ผลลัพธ์ตรงใจ

RCIF คืออะไร?

RCIF คือสูตรที่ช่วยให้เขียน Prompt ได้ครบถ้วน ประกอบด้วย 4 องค์ประกอบ:

องค์ประกอบ ความหมาย คำอธิบาย ตัวอย่าง
R — Role บทบาท กำหนดว่า AI เป็นใคร มีความเชี่ยวชาญอะไร "คุณเป็นนักเขียนคอนเทนต์มืออาชีพ"
C — Context บริบท ให้ข้อมูลพื้นหลัง สถานการณ์ เงื่อนไข "สำหรับหน่วยงานราชการ กลุ่มเป้าหมายคือข้าราชการอายุ 35-50 ปี"
I — Instruction คำสั่ง บอกชัดเจนว่าต้องทำอะไร "เขียนบทความ 500 คำ เรื่องการใช้ AI ในงานราชการ"
F — Format รูปแบบ กำหนดรูปแบบผลลัพธ์ที่ต้องการ "แบ่งเป็นหัวข้อ 5 ข้อ แต่ละข้อมีคำอธิบาย 2-3 ประโยค"

ตัวอย่าง Prompt เต็มรูปแบบ (สูตร RCIF)

ตัวอย่างที่ 1 — ร่างบันทึกข้อความ

[R] คุณเป็นเจ้าหน้าที่ธุรการที่มีประสบการณ์ 10 ปีในหน่วยงานราชการ

[C] สำนักงานพัฒนาฝีมือแรงงานสุรินทร์ ต้องการจัดซื้อคอมพิวเตอร์โน้ตบุ๊ก 10 เครื่อง สำหรับฝ่ายสารสนเทศ งบประมาณไม่เกิน 200,000 บาท

[I] ช่วยร่างบันทึกข้อความถึงผู้อำนวยการ เพื่อขออนุมัติจัดซื้อ

[F] ใช้รูปแบบหนังสือราชการ มีเลขที่หนังสือ วันที่ เรื่อง เรียน สิ่งที่ส่งมาด้วย และลงท้ายด้วยช่องลงนาม

ตัวอย่างที่ 2 — สร้างแผนฝึกอบรม

[R] คุณเป็นนักฝึกอบรมมืออาชีพที่เชี่ยวชาญด้านการพัฒนาทักษะดิจิทัล

[C] หน่วยงานต้องการจัดอบรม "การใช้ AI ในงานราชการ" สำหรับเจ้าหน้าที่ 30 คน ระยะเวลา 2 วัน

[I] ออกแบบกำหนดการอบรมแบบละเอียด รวมเนื้อหาและกิจกรรม

[F] จัดเป็นตารางแบ่งเป็นช่วงเช้า-บ่าย แต่ละช่วงระบุหัวข้อ ระยะเวลา วิทยากร และกิจกรรม

ตัวอย่างที่ 3 — สรุปรายงานประชุม

[R] คุณเป็นเลขานุการที่ประชุมที่มีความชำนาญในการจดบันทึก

[C] ประชุมคณะกรรมการบริหารครั้งที่ 3/2569 วันที่ 15 มกราคม 2569 ผู้เข้าร่วม 12 คน วาระหลัก: แผนงบประมาณปี 2570

[I] สรุปรายงานการประชุมจากข้อมูลที่ให้ไว้

[F] รูปแบบรายงานการประชุมราชการ มีวาระที่ 1-5 แต่ละวาระมีผู้เสนอ สาระสำคัญ และมติ

💡 เคล็ดลับ

ไม่จำเป็นต้องใส่ครบทุกตัว RCIF ทุกครั้ง — แต่ยิ่งใส่ครบ ผลลัพธ์ยิ่งตรงใจ โดยเฉพาะ I (คำสั่ง) ที่ขาดไม่ได้เลย

2.3 ตัวอย่าง Prompt ดี vs Prompt แย่

เปรียบเทียบให้เห็นชัดว่าอะไรทำให้ Prompt ดีขึ้น

# ❌ Prompt แย่ ✅ Prompt ดี ทำไมดีกว่า
1 "เขียนหนังสือ" "คุณเป็นเจ้าหน้าที่ธุรการ ร่างบันทึกข้อความ เรื่องขออนุมัติเดินทางไปราชการจังหวัดนครราชสีมา วันที่ 20-21 ม.ค. 2569 ถึงผู้อำนวยการ รูปแบบหนังสือราชการ" มีบทบาท + บริบทครบ + ระบุรูปแบบชัด
2 "สรุปให้หน่อย" "สรุปบทความนี้เป็นภาษาไทย 5 หัวข้อหลัก แต่ละข้อไม่เกิน 2 บรรทัด เน้นประเด็นที่เกี่ยวกับงานราชการ" บอกจำนวน ความยาว และเงื่อนไข
3 "แปลภาษา" "แปลข้อความต่อไปนี้จากไทยเป็นอังกฤษ ใช้ภาษาทางการสำหรับหนังสือราชการ: [ข้อความ]" ระบุภาษาต้นทาง-ปลายทาง + ระดับภาษา
4 "ทำ PowerPoint" "คุณเป็นนักออกแบบสื่อ ช่วยเขียนเนื้อหาสำหรับ PowerPoint 10 สไลด์ เรื่อง 'AI กับงานราชการ' แต่ละสไลด์มีหัวข้อ + bullet 3-4 ข้อ + คำแนะนำรูปภาพ" ระบุจำนวนสไลด์ + โครงสร้างแต่ละหน้า
5 "เขียนอีเมล" "ร่างอีเมลเชิญวิทยากรภายนอก (ผศ.ดร.สมศรี) มาบรรยายเรื่อง Digital Transformation วันที่ 15 ก.พ. 2569 เวลา 09:00-12:00 น. ที่ สนพ.สุรินทร์ น้ำเสียงสุภาพเป็นทางการ ความยาวไม่เกิน 200 คำ" มีชื่อ วันเวลา สถานที่ + กำหนดน้ำเสียง + ความยาว
6 "วิเคราะห์ข้อมูล" "วิเคราะห์ข้อมูลผู้เข้าอบรมจากตารางที่แนบ โดย: 1) สรุปจำนวนผู้เข้าอบรมแยกตามหลักสูตร 2) หาหลักสูตรที่มีผู้สมัครมากที่สุด 3) เปรียบเทียบกับปีก่อน แสดงผลเป็นตาราง" บอกชัดว่าวิเคราะห์อะไร + แบ่งเป็นข้อ + ระบุรูปแบบผลลัพธ์
⚠️ ข้อสังเกต

Prompt แย่ไม่ได้ "ผิด" — AI ยังตอบได้ แต่ผลลัพธ์จะกว้าง ไม่ตรงใจ ต้องแก้หลายรอบ ในขณะที่ Prompt ดี ตอบได้ตรงใจตั้งแต่ครั้งแรก

2.4 เทคนิค Zero-shot, One-shot, Few-shot

เทคนิคการให้ตัวอย่างเพื่อควบคุมรูปแบบผลลัพธ์

Zero-shot — ถามตรงๆ ไม่ให้ตัวอย่าง

เหมาะกับงานง่ายๆ ที่ AI เข้าใจได้เลย ไม่ต้องชี้แนะ

ตัวอย่าง Zero-shot

"จัดหมวดหมู่คำต่อไปนี้เป็นหมวด ผลไม้ หรือ ผัก: แตงโม, ผักกาด, มะม่วง, บร็อคโคลี, ทุเรียน, แครอท"

One-shot — ให้ตัวอย่าง 1 ข้อ

เหมาะเมื่อต้องการให้ AI เข้าใจ "รูปแบบ" หรือ "สไตล์" ที่ต้องการ

ตัวอย่าง One-shot

เขียนคำขวัญหน่วยงานตามรูปแบบนี้:

ตัวอย่าง: "พัฒนาฝีมือ เสริมทักษะ สร้างอาชีพ สู่ความยั่งยืน"

เขียนให้: คำขวัญสำหรับหน่วยงานส่งเสริมการท่องเที่ยวจังหวัด 3 ข้อ

Few-shot — ให้ตัวอย่าง 2-3 ข้อ

เหมาะเมื่อต้องการควบคุมรูปแบบอย่างเข้มงวด หรืองานที่มีรูปแบบเฉพาะ

ตัวอย่าง Few-shot

เขียนคำอธิบายหลักสูตรอบรมตามรูปแบบนี้:

ตัวอย่าง 1:
หลักสูตร: การใช้ Excel ขั้นสูง
คำอธิบาย: เรียนรู้สูตรขั้นสูง Pivot Table และ Macro เพื่อจัดการข้อมูลได้เร็วขึ้น 10 เท่า เหมาะสำหรับผู้ที่ใช้ Excel ทุกวัน

ตัวอย่าง 2:
หลักสูตร: การถ่ายภาพด้วยสมาร์ทโฟน
คำอธิบาย: เทคนิคถ่ายรูปสวยด้วยมือถือ ตั้งแต่จัดองค์ประกอบ แสง จนถึงแต่งรูป เหมาะสำหรับงาน PR และสื่อออนไลน์

ตัวอย่าง 3:
หลักสูตร: การเขียนโปรแกรม Python เบื้องต้น
คำอธิบาย: เริ่มต้นเขียนโปรแกรมตั้งแต่ศูนย์ ครอบคลุมตัวแปร เงื่อนไข ลูป และการจัดการไฟล์ เหมาะสำหรับผู้เริ่มต้นที่ไม่มีพื้นฐาน

เขียนให้: หลักสูตร "การใช้ AI ในงานราชการ"

เปรียบเทียบ 3 เทคนิค

เทคนิค ให้ตัวอย่าง เหมาะกับ ข้อดี ข้อจำกัด
Zero-shot 0 ข้อ งานทั่วไป คำถามตรงๆ เร็ว กระชับ ผลลัพธ์อาจไม่ตรงรูปแบบ
One-shot 1 ข้อ ต้องการสไตล์เฉพาะ AI เข้าใจรูปแบบได้ ตัวอย่างเดียวอาจไม่พอ
Few-shot 2-5 ข้อ งานที่มี pattern ชัด ควบคุมรูปแบบได้ดีมาก Prompt ยาว ใช้ Token มาก

2.5 การกำหนดรูปแบบผลลัพธ์

บอก AI ว่าต้องการผลลัพธ์ในรูปแบบไหน

การระบุ Format ช่วยให้ AI ตอบในรูปแบบที่พร้อมใช้งาน ไม่ต้องมานั่งจัดเองทีหลัง:

รูปแบบ วิธีสั่ง เหมาะกับ
ตาราง "แสดงผลเป็นตาราง มีคอลัมน์: ลำดับ, หัวข้อ, รายละเอียด" เปรียบเทียบข้อมูล, รายการ
หัวข้อ (Bullet) "ตอบเป็นหัวข้อ 5 ข้อ แต่ละข้อ 1-2 ประโยค" สรุป, ไอเดีย, ข้อดี-ข้อเสีย
ย่อหน้า "เขียนเป็นย่อหน้า ความยาว 200 คำ" บทความ, รายงาน, คำอธิบาย
ขั้นตอน (Step) "อธิบายเป็นขั้นตอน ขั้นที่ 1, 2, 3..." วิธีทำ, คู่มือ, Tutorial
JSON "ตอบเป็น JSON format มี key: name, description, category" ข้อมูลสำหรับโปรแกรม/ระบบ
Markdown "ตอบเป็น Markdown ใช้ heading, bold, bullet" เอกสาร, README, บทความออนไลน์
อีเมล/จดหมาย "ร่างเป็นอีเมล มีหัวเรื่อง คำขึ้นต้น เนื้อหา ลงท้าย" การสื่อสารทางการ

ตัวอย่าง Prompt ที่ระบุ Format ต่างๆ

ตัวอย่าง — ขอผลลัพธ์เป็นตาราง

"เปรียบเทียบ ChatGPT, Gemini และ Claude ในด้าน: ราคา, ข้อดี, ข้อจำกัด, เหมาะกับงาน — แสดงผลเป็นตาราง"

ตัวอย่าง — ขอผลลัพธ์เป็นขั้นตอน

"อธิบายวิธีสร้าง Google Form สำหรับแบบประเมินความพึงพอใจ เป็นขั้นตอน step-by-step พร้อมคำอธิบายแต่ละขั้น"

ตัวอย่าง — ขอผลลัพธ์เป็น JSON

"สร้างข้อมูลหลักสูตรอบรม 3 หลักสูตร ตอบเป็น JSON array โดยแต่ละ object มี: course_name, duration_hours, level, description"

ตัวอย่าง — ขอผลลัพธ์แบบผสม

"สรุปรายงานประจำปีต่อไปนี้ โดย: 1) ย่อหน้าบทสรุปผู้บริหาร 3 บรรทัด 2) ตารางตัวชี้วัด KPI 5 ข้อ 3) bullet ข้อเสนอแนะ 3 ข้อ"

2.6 ข้อผิดพลาดที่พบบ่อย

หลุมพรางที่มือใหม่มักพลาด และวิธีแก้ไข

# ข้อผิดพลาด ตัวอย่าง วิธีแก้ไข
1 Prompt กว้างเกินไป "ช่วยเรื่องงาน" — AI ไม่รู้ว่างานอะไร ระบุให้ชัด: ทำอะไร + เรื่องอะไร + สำหรับใคร
2 ไม่ระบุบริบท "เขียนจดหมาย" — ไม่รู้จดหมายอะไร ถึงใคร ใส่ Context: ใคร ทำไม เมื่อไหร่ ที่ไหน
3 ยาวเกินไป (ข้อมูลไม่จำเป็น) ใส่ประวัติหน่วยงานตั้งแต่ก่อตั้ง 50 ปีก่อนเพื่อขอเขียนอีเมลสั้นๆ ใส่เฉพาะข้อมูลที่เกี่ยวข้องกับสิ่งที่ต้องการ
4 ไม่บอกรูปแบบผลลัพธ์ "สรุปให้หน่อย" — AI อาจตอบยาว 3 หน้า ระบุ: กี่ข้อ กี่คำ รูปแบบอะไร
5 ถามหลายเรื่องใน Prompt เดียว "เขียนจดหมาย + สร้าง PPT + วิเคราะห์ข้อมูล" แยก 1 Prompt = 1 งาน แล้วทำทีละขั้น
6 ไม่กำหนดบทบาทให้ AI "เขียนบทความ" — AI ตอบแบบทั่วๆ ไป เริ่มด้วย "คุณเป็น..." เพื่อให้ AI ตอบเฉพาะทาง
💡 Tips รวม — ทำให้ Prompt ดีขึ้นทันที
  • เริ่มด้วย Role: "คุณเป็น..." ช่วยให้ AI เข้าถึงความเชี่ยวชาญที่ต้องการ
  • 1 Prompt = 1 งาน: อย่ายัดหลายเรื่องรวมกัน
  • ระบุตัวเลข: "5 ข้อ", "200 คำ", "3 ย่อหน้า" — ชัดเจนกว่า "สั้นๆ"
  • บอกสิ่งที่ไม่ต้องการ: "ห้ามใช้ศัพท์เทคนิค", "ไม่ต้องมีคำนำ"
  • ถ้าไม่ได้ผลตรงใจ: ลองเพิ่มบริบท หรือให้ตัวอย่าง (One-shot/Few-shot)
  • Iterate: ถามต่อเนื่องได้ เช่น "ปรับให้สั้นกว่านี้" หรือ "เพิ่มตัวอย่างอีก"

✏️ แบบฝึกหัดบทที่ 2

  1. เขียน Prompt ตามสูตร RCIF 3 ข้อ

    เลือกจากงานจริงของท่าน เช่น ร่างหนังสือราชการ สรุปรายงาน เขียนประกาศ — ระบุ R, C, I, F ให้ครบ

  2. แปลง Prompt แย่ให้เป็น Prompt ดี

    ปรับปรุง Prompt ต่อไปนี้ให้ดีขึ้น:

    • "ช่วยทำตาราง"
    • "เขียนข่าว"
    • "สรุปเรื่องนี้"
  3. ทดลองเขียน Few-shot Prompt 1 ข้อ

    สร้าง Prompt ที่มีตัวอย่าง 2-3 ข้อ เพื่อให้ AI สร้างผลลัพธ์ในรูปแบบเดียวกัน (เช่น คำอธิบายหลักสูตร, ประกาศรับสมัคร, สโลแกน)

  4. ถามคำถามเดียวกัน 3 แบบ แล้วเปรียบเทียบ

    ถาม ChatGPT คำถามเดียวกัน (เช่น "เขียนอีเมลเชิญประชุม") ด้วย:

    • แบบที่ 1: Prompt สั้นๆ (Zero-shot ไม่มี Format)
    • แบบที่ 2: Prompt แบบ RCIF เต็มรูปแบบ
    • แบบที่ 3: Prompt แบบ Few-shot + ระบุ Format

    บันทึกผลลัพธ์และเปรียบเทียบความแตกต่าง
