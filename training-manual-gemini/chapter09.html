9.1 Gemini Code Assist คืออะไร

Gemini Code Assist เป็น AI Coding Assistant ของ Google ที่ทำงานใน IDE (VS Code, JetBrains) ช่วยเขียนโค้ดอัตโนมัติ อธิบายโค้ด แก้ bug และ generate code — คล้ายกับ GitHub Copilot แต่ใช้ Gemini model

🆓 Free Tier
  • Code completions: ไม่จำกัด
  • Chat: 240 ข้อความ/วัน (รีเซ็ตเที่ยงคืน Pacific)
  • ใช้ได้กับ Google Account ฟรี
  • รองรับ 20+ ภาษา
💼 Enterprise
  • Code completions: ไม่จำกัด
  • Chat: ไม่จำกัด
  • Code customization (private codebase)
  • IP indemnification
  • $19/user/เดือน
💡 สำหรับงานราชการ: Free tier เพียงพอสำหรับการเรียนรู้และใช้งานทั่วไป ไม่ต้องเสียเงิน!

9.2 ติดตั้ง Gemini Code Assist Extension ใน VS Code

📥 ขั้นตอนติดตั้ง (Step-by-Step)
ขั้นตอนที่ 1: ติดตั้ง VS Code
  1. ดาวน์โหลดจาก https://code.visualstudio.com
  2. ติดตั้งตามขั้นตอนปกติ (Next → Next → Install)
  3. เปิด VS Code
ขั้นตอนที่ 2: ติดตั้ง Extension
  1. คลิกไอคอน Extensions (□ สี่เหลี่ยม) ที่ sidebar ซ้าย หรือกด Ctrl+Shift+X
  2. พิมพ์ค้นหา: Gemini Code Assist
  3. เลือก "Gemini Code Assist" โดย Google (ตรวจสอบ Publisher: Google)
  4. คลิก "Install"
  5. รอ 10-20 วินาที จนติดตั้งเสร็จ
ขั้นตอนที่ 3: Sign in กับ Google Account
  1. จะเห็นไอคอน Gemini (ดาว) ที่ sidebar ซ้าย → คลิก
  2. คลิก "Sign in with Google"
  3. เบราว์เซอร์จะเปิดขึ้น → เลือก Google Account → อนุญาต
  4. กลับมาที่ VS Code → จะเห็นสถานะ "Signed in"
ขั้นตอนที่ 4: ทดสอบ
  1. สร้างไฟล์ใหม่: test.js
  2. พิมพ์: // ฟังก์ชันคำนวณ BMI แล้วกด Enter
  3. Gemini จะเสนอโค้ดสีเทา (ghost text)
  4. กด Tab เพื่อยอมรับ หรือ Esc เพื่อปฏิเสธ
⚠️ ปัญหาที่อาจพบ:
  • ไม่เห็นไอคอน Gemini: Reload VS Code (Ctrl+Shift+P → "Reload Window")
  • Sign in ไม่ได้: ตรวจสอบ Firewall/Proxy ขององค์กร
  • ไม่ suggest code: ตรวจว่าไฟล์เป็นภาษาที่รองรับ (.js, .py, .html)

9.3 ฟีเจอร์หลัก Gemini Code Assist

⌨️ 1. Code Completion — เติมโค้ดอัตโนมัติ

Gemini จะเสนอโค้ดถัดไปแบบ real-time ขณะที่คุณพิมพ์

วิธีใช้:
  • พิมพ์โค้ดปกติ → Gemini จะเสนอส่วนที่เหลือ (ghost text สีเทา)
  • Tab = ยอมรับทั้งหมด
  • Ctrl+→ = ยอมรับทีละคำ
  • Esc = ปฏิเสธ
ตัวอย่าง: พิมพ์ comment แล้ว Gemini สร้างโค้ดให้
// ฟังก์ชันตรวจสอบเลขบัตรประชาชน 13 หลัก
function validateThaiID(id) {
  // ← Gemini จะเสนอโค้ดทั้งฟังก์ชันให้!
  if (id.length !== 13) return false;
  var sum = 0;
  for (var i = 0; i < 12; i++) {
    sum += parseInt(id[i]) * (13 - i);
  }
  var checkDigit = (11 - (sum % 11)) % 10;
  return checkDigit === parseInt(id[12]);
}
💬 2. Chat — ถามตอบเกี่ยวกับโค้ด

เปิด Chat panel เพื่อถามคำถามเกี่ยวกับโค้ด ขอคำแนะนำ หรือให้ช่วยเขียนโค้ด

วิธีเปิด Chat:
  • คลิกไอคอน Gemini ที่ sidebar → เปิด Chat panel
  • หรือกด Ctrl+Shift+P → พิมพ์ "Gemini: Open Chat"
ตัวอย่างคำถามใน Chat:
  • "เขียนฟังก์ชัน GAS ที่ส่ง Email พร้อม PDF attachment"
  • "อธิบายโค้ดนี้ทำงานอย่างไร" (select code แล้วถาม)
  • "แก้ bug ในโค้ดนี้ — มันส่ง email ซ้ำ 2 ครั้ง"
  • "เขียน unit test สำหรับฟังก์ชัน validateThaiID"
  • "refactor โค้ดนี้ให้อ่านง่ายขึ้น"
🔨 3. Code Generation — สร้างโค้ดจากคำอธิบาย

ใช้ comment หรือ Chat เพื่อให้ Gemini สร้างโค้ดทั้งไฟล์ได้

📖 4. Explain Code — อธิบายโค้ด

Select โค้ดที่ไม่เข้าใจ → คลิกขวา → "Gemini: Explain this code"

ตัวอย่าง: Select โค้ดนี้แล้วให้อธิบาย
var hash = Utilities.computeDigest(
  Utilities.DigestAlgorithm.SHA_256, 
  password + salt
).map(function(b) { 
  return ('0' + (b & 0xFF).toString(16)).slice(-2); 
}).join('');
Gemini จะอธิบาย:

โค้ดนี้ทำ Password Hashing ด้วย SHA-256:

  1. Utilities.computeDigest() — คำนวณ hash จาก password + salt
  2. .map(function(b)...) — แปลง byte array เป็น hex string
  3. (b & 0xFF) — แปลง signed byte เป็น unsigned (0-255)
  4. .toString(16) — แปลงเป็น hexadecimal
  5. .slice(-2) — เติม 0 นำหน้าให้ครบ 2 หลัก
  6. .join('') — รวมเป็น string เดียว
🐛 5. Fix Bugs — หา + แก้ Bug

Select โค้ดที่มีปัญหา → "Gemini: Fix this code" หรือถามใน Chat

ตัวอย่าง: โค้ดที่มี Bug
// Bug: ฟังก์ชันนี้ไม่ส่ง email เมื่อชื่อมีอักขระพิเศษ
function sendWelcomeEmail(name, email) {
  var subject = 'ยินดีต้อนรับ ' + name + ' สู่ สนพ.สุรินทร์';
  var body = '<h1>สวัสดี ' + name + '</h1>';
  GmailApp.sendEmail(email, subject, '', {htmlBody: body});
}
Gemini แนะนำแก้ไข:
// Fixed: เพิ่ม HTML escape ป้องกัน XSS + จัดการ error
function sendWelcomeEmail(name, email) {
  try {
    // Escape HTML entities ป้องกัน XSS
    var safeName = name.replace(/&/g, '&amp;')
                       .replace(//g, '&gt;');
    var subject = 'ยินดีต้อนรับ ' + safeName + ' สู่ สนพ.สุรินทร์';
    var body = '<h1>สวัสดี ' + safeName + '</h1>';
    GmailApp.sendEmail(email, subject, '', {htmlBody: body});
    return true;
  } catch (e) {
    Logger.log('Email error: ' + e.toString());
    return false;
  }
}

9.4 ใช้ Gemini Code Assist เขียน GAS + HTML + JavaScript

🏋️ Workshop (60 นาที)

ใช้ Gemini Code Assist สร้างระบบลงทะเบียนฝึกอบรมออนไลน์แบบง่าย

📋 ขั้นตอนที่ 1: สร้างโปรเจกต์ (5 นาที)
  1. สร้างโฟลเดอร์ใหม่: training-registration
  2. เปิดใน VS Code: File → Open Folder
  3. สร้างไฟล์ 3 ไฟล์: Code.gs, index.html, appsscript.json
💻 ขั้นตอนที่ 2: ใช้ Gemini สร้าง Code.gs (20 นาที)

เปิดไฟล์ Code.gs → พิมพ์ comment → ให้ Gemini generate

// ⬇️ พิมพ์ comment นี้ แล้ว Gemini จะ generate ให้ ⬇️

// ระบบลงทะเบียนฝึกอบรม สนพ.สุรินทร์
// - doGet() แสดงหน้า HTML
// - doPost() รับข้อมูลลงทะเบียน บันทึกลง Google Sheets
// - Sheet: REGISTRATIONS (id, name, idcard, phone, email, course, date, status)
// - ส่ง email ยืนยันหลังลงทะเบียน
// - ส่ง Line Notify แจ้ง admin

function doGet() {
  return HtmlService.createHtmlOutputFromFile('index')
    .setTitle('ลงทะเบียนฝึกอบรม - สนพ.สุรินทร์')
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}

function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  
  // validate
  if (!data.name || !data.idcard || !data.phone || !data.course) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false, message: 'กรุณากรอกข้อมูลให้ครบ'
    })).setMimeType(ContentService.MimeType.JSON);
  }
  
  // บันทึกลง Sheet
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('REGISTRATIONS');
  var id = 'REG-' + new Date().getTime();
  
  sheet.appendRow([
    id, data.name, data.idcard, data.phone, 
    data.email, data.course, new Date(), 'รอดำเนินการ'
  ]);
  
  // ส่ง email ยืนยัน
  if (data.email) {
    sendConfirmEmail(data.email, data.name, data.course, id);
  }
  
  return ContentService.createTextOutput(JSON.stringify({
    success: true, message: 'ลงทะเบียนสำเร็จ', id: id
  })).setMimeType(ContentService.MimeType.JSON);
}

// ← Gemini จะช่วยเขียน sendConfirmEmail() ต่อ

🐛 ขั้นตอนที่ 4: ใช้ Gemini Fix Bug (10 นาที)
  1. ทดสอบระบบ → หาจุดที่ผิดพลาด
  2. Select โค้ดที่มีปัญหา → คลิกขวา → "Gemini: Fix this"
  3. หรือถามใน Chat: "ทำไมฟอร์มส่งข้อมูลแล้ว error 403"
  4. Gemini จะแนะนำวิธีแก้ไข
💡 เคล็ดลับใช้ Gemini Code Assist อย่างมีประสิทธิภาพ
  • เขียน comment ก่อน: comment ที่ละเอียดจะได้โค้ดที่แม่นยำ
  • ใช้ภาษาอังกฤษใน comment: ได้ผลดีกว่าภาษาไทย (สำหรับ code generation)
  • ตั้งชื่อฟังก์ชันให้สื่อ: Gemini จะเข้าใจ context จากชื่อ
  • เปิดไฟล์ที่เกี่ยวข้อง: Gemini ดูจากไฟล์ที่เปิดอยู่ด้วย
  • อย่ายอมรับโค้ดโดยไม่ตรวจ: ต้อง review ก่อนเสมอ!
