7.1 Google AI Studio คืออะไร

Google AI Studio เป็น Web-based IDE/Playground ที่ให้ทดลองใช้งาน Gemini API ฟรี โดยไม่ต้องเขียนโค้ด เหมาะสำหรับ:

🧪

ทดลอง Prompt

ทดสอบ prompt ต่างๆ กับ Gemini โดยตรง ปรับแต่ง parameter ได้

🔑

สร้าง API Key

สร้าง API Key ฟรี เพื่อนำไปใช้ในโค้ดของเรา

🤖

สร้าง Chatbot

ออกแบบ Chatbot ง่ายๆ ทดสอบได้ทันที

🌐 เข้าใช้งานได้ที่

URL: https://aistudio.google.com — ใช้ Google Account ฟรี (ไม่ต้องเสียเงิน!)

📊 เปรียบเทียบ: Gemini Web vs Google AI Studio
หัวข้อ Gemini Web (gemini.google.com) Google AI Studio (aistudio.google.com)
กลุ่มเป้าหมาย ผู้ใช้ทั่วไป นักพัฒนา / ผู้ต้องการปรับแต่ง
ราคา ฟรี (Pro ต้องจ่าย) ฟรี (มี Quota)
ปรับ Parameter ❌ ไม่ได้ ✅ Temperature, Top-P, Top-K, Max tokens
API Key ❌ ไม่มี ✅ สร้างได้ฟรี
System Instruction ❌ ไม่มี (ใช้ Gems แทน) ✅ กำหนดได้เอง
Few-shot Examples ❌ ไม่รองรับ ✅ เพิ่ม input/output ตัวอย่างได้
Export Code ❌ ไม่มี ✅ Export เป็น Python, JavaScript, cURL
Fine-tuning ❌ ไม่มี ✅ Tune model ได้

🎮 ขั้นตอนที่ 3: ทดลอง Prompt ใน Playground

Playground มี 3 โหมด:

1. Freeform

พิมพ์ prompt อิสระ เหมือน Gemini Web แต่ปรับ parameter ได้

2. Structured

กำหนด input/output ตัวอย่าง (Few-shot)

3. Chat

สร้าง Chatbot มี System Instruction

Parameter ที่ปรับได้:
Parameter ค่าเริ่มต้น ช่วง อธิบาย
Temperature 1.0 0.0 – 2.0 ยิ่งสูง = ยิ่งสร้างสรรค์/สุ่ม | ยิ่งต่ำ = ยิ่งแม่นยำ/ซ้ำ
Top-P 0.95 0.0 – 1.0 ความหลากหลายของคำตอบ (nucleus sampling)
Top-K 40 1 – 100 จำนวน token ที่พิจารณาในแต่ละ step
Max Output Tokens 8192 1 – 65,536 ความยาวคำตอบสูงสุด
💡 เคล็ดลับ:
  • งานเขียนสร้างสรรค์: Temperature 1.0-1.5
  • งานสรุป/แปลภาษา: Temperature 0.2-0.5
  • งานถูก-ผิดชัดเจน (เช่น code): Temperature 0.0-0.3
🏋️ ฝึกปฏิบัติ: ทดลอง Playground (15 นาที)
  1. เข้า AI Studio → เลือก Freeform
  2. พิมพ์ prompt: "สรุปหน้าที่ของสำนักงานพัฒนาฝีมือแรงงานจังหวัด 5 ข้อ"
  3. ลองเปลี่ยน Temperature เป็น 0.2 → Run อีกครั้ง → เปรียบเทียบผลลัพธ์
  4. ลองเปลี่ยน Temperature เป็น 1.8 → Run อีกครั้ง → สังเกตความแตกต่าง
  5. ลองเพิ่ม System Instruction: "คุณเป็นผู้เชี่ยวชาญด้านพัฒนาฝีมือแรงงาน ตอบเป็นภาษาราชการ"

7.3 Structured Prompts — สร้าง Prompt แบบมีโครงสร้าง

Structured Prompts คือการสอน Gemini ด้วย ตัวอย่าง input/output (Few-shot learning) ทำให้ได้ผลลัพธ์ที่สม่ำเสมอและตรงตามรูปแบบที่ต้องการ

📝 โครงสร้างของ Structured Prompt
┌─────────────────────────────────────┐
│  System Instruction (คำสั่งระบบ)     │
├─────────────────────────────────────┤
│  Examples (ตัวอย่าง input → output)  │
│  ├── Example 1: Input → Output      │
│  ├── Example 2: Input → Output      │
│  └── Example 3: Input → Output      │
├─────────────────────────────────────┤
│  Test Input (ข้อมูลจริงที่ต้องการ)    │
└─────────────────────────────────────┘
ตัวอย่าง 1: จำแนกประเภทหนังสือราชการ
System Instruction:
"จำแนกประเภทหนังสือราชการ ตอบเฉพาะประเภท: คำสั่ง | ประกาศ | หนังสือเวียน | บันทึกข้อความ | หนังสือภายนอก"
ตัวอย่าง (Examples):
Input Output
เรื่อง แต่งตั้งคณะกรรมการดำเนินการฝึกอบรม คำสั่ง
เรื่อง รับสมัครบุคคลเพื่อเลือกสรรเป็นพนักงานราชการ ประกาศ
เรื่อง ขอเชิญประชุมประจำเดือน บันทึกข้อความ
เรื่อง ขอความร่วมมือสำรวจข้อมูลแรงงาน หนังสือภายนอก
Test Input:
"เรื่อง กำหนดวันหยุดราชการเพิ่มเติมเป็นกรณีพิเศษ"

→ Output: ประกาศ

ตัวอย่าง 2: สร้างรหัสหลักสูตรอัตโนมัติ
System Instruction:
"สร้างรหัสหลักสูตร สนพ.สุรินทร์ ตามรูปแบบ: [ประเภท 2 ตัว]-[ปี 2 หลัก]-[ลำดับ 3 หลัก] ประเภท: ฝก=ฝึกอบรม, ทด=ทดสอบ, ปม=ประเมิน"
ตัวอย่าง:
Input Output
หลักสูตรฝึกอบรมช่างเชื่อม ปี 2569 ลำดับที่ 5 ฝก-69-005
ทดสอบมาตรฐานฝีมือช่างไฟ ปี 2569 ลำดับที่ 12 ทด-69-012
ประเมินความรู้ผู้ประกอบอาหาร ปี 2570 ลำดับที่ 1 ปม-70-001
ตัวอย่าง 3: สกัดข้อมูลจากข้อความ (Entity Extraction)
System Instruction:
"สกัดข้อมูลจากข้อความ คืนผลเป็น JSON format: {name, position, organization, phone, email}"
ตัวอย่าง:

Input:

"ติดต่อ นายสมชาย ใจดี ตำแหน่งนักวิชาการพัฒนาฝีมือแรงงาน สนพ.สุรินทร์ โทร 044-511-259 อีเมล somchai@dsd.go.th"

Output:

{
  "name": "นายสมชาย ใจดี",
  "position": "นักวิชาการพัฒนาฝีมือแรงงาน",
  "organization": "สนพ.สุรินทร์",
  "phone": "044-511-259",
  "email": "somchai@dsd.go.th"
}
📝 ขั้นตอนสร้าง Structured Prompt ใน AI Studio:
  1. คลิก "Create new" → เลือก "Structured prompt"
  2. เขียน System Instruction ด้านบน
  3. เพิ่มตัวอย่าง: คลิก "Add example" → กรอก Input และ Output
  4. เพิ่มอย่างน้อย 3 ตัวอย่าง (ยิ่งมากยิ่งดี)
  5. ทดสอบด้วยข้อมูลใหม่ใน "Test" section
  6. ถ้าพอใจ คลิก "Get code" เพื่อ export เป็น Python/JS/cURL

7.4 Chat Prompts — สร้าง Chatbot ง่ายๆ

Chat Prompts ใน AI Studio ช่วยให้ออกแบบ Chatbot ที่มี "บุคลิก" เฉพาะตัว โดยกำหนด System Instruction เป็นตัวกำกับพฤติกรรม

🤖 ตัวอย่าง: Chatbot ตอบคำถามงาน สนพ.สุรินทร์
System Instruction:
คุณเป็น "น้องฝีมือ" ผู้ช่วย AI ของสำนักงานพัฒนาฝีมือแรงงานสุรินทร์

หน้าที่ของคุณ:
1. ตอบคำถามเกี่ยวกับหลักสูตรฝึกอบรม
2. ให้ข้อมูลการสมัครเรียน คุณสมบัติ เอกสาร
3. แนะนำหลักสูตรที่เหมาะสมตามความสนใจ
4. ให้ข้อมูลติดต่อ สนพ.สุรินทร์

กฎ:
- ตอบเป็นภาษาไทย สุภาพ เป็นกันเอง
- ถ้าไม่แน่ใจ ให้แนะนำโทรสอบถาม 044-511-259
- ห้ามให้ข้อมูลที่ไม่เกี่ยวกับงาน สนพ.
- ถ้าถูกถามเรื่องนอกเหนือหน้าที่ ให้บอกว่า "ขอโทษค่ะ น้องฝีมือช่วยได้เฉพาะเรื่องฝึกอบรมค่ะ"

ข้อมูลหลักสูตรปัจจุบัน:
- ช่างเชื่อม MAG (30 ชม.) — เปิดรับ
- คอมพิวเตอร์สำนักงาน (30 ชม.) — เปิดรับ
- ช่างไฟฟ้าภายในอาคาร (30 ชม.) — เต็ม
- ตัดผมชาย (60 ชม.) — เปิดรับ
ทดสอบบทสนทนา:
User: มีหลักสูตรอะไรเปิดรับบ้างครับ
น้องฝีมือ: สวัสดีค่ะ 😊 ตอนนี้ สนพ.สุรินทร์ มีหลักสูตรที่เปิดรับดังนี้ค่ะ:
1. ช่างเชื่อม MAG (30 ชม.) - ยังมีที่ว่าง
2. คอมพิวเตอร์สำนักงาน (30 ชม.) - ยังมีที่ว่าง
3. ตัดผมชาย (60 ชม.) - ยังมีที่ว่าง
ส่วนช่างไฟฟ้าภายในอาคารเต็มแล้วค่ะ สนใจหลักสูตรไหนคะ?
User: สมัครช่างเชื่อมต้องใช้เอกสารอะไรบ้าง
น้องฝีมือ: สำหรับการสมัครหลักสูตรช่างเชื่อม MAG ใช้เอกสารดังนี้ค่ะ:
1. สำเนาบัตรประชาชน 1 ฉบับ
2. สำเนาทะเบียนบ้าน 1 ฉบับ
3. รูปถ่าย 1 นิ้ว 2 รูป
4. วุฒิการศึกษา (ถ้ามี)
มาสมัครได้ที่ สนพ.สุรินทร์ วันจันทร์-ศุกร์ 08:30-16:30 น. หรือโทร 044-511-259 ค่ะ 📞
📝 ขั้นตอนสร้าง Chat Prompt ใน AI Studio:
  1. คลิก "Create new" → เลือก "Chat prompt"
  2. เขียน System Instruction (บุคลิก + กฎ + ข้อมูล)
  3. เพิ่ม Example Conversations (ถ้ามี)
  4. ทดสอบโดยพิมพ์คำถามในช่อง Chat
  5. ปรับแต่ง Temperature (แนะนำ 0.5-0.7 สำหรับ Chatbot)
  6. Export code เพื่อนำไปใช้จริง
🏋️ Workshop: สร้าง Chatbot ของหน่วยงาน (25 นาที)

โจทย์: สร้าง Chatbot สำหรับหน่วยงานของคุณ

  1. คิดชื่อ Chatbot (เช่น "น้องฝีมือ", "พี่ช่าง", "ผู้ช่วยฝึก")
  2. เขียน System Instruction ที่ครอบคลุม: บุคลิก, หน้าที่, กฎ, ข้อมูล
  3. ทดสอบด้วยคำถามอย่างน้อย 10 คำถาม
  4. ทดสอบ "edge cases" (ถามเรื่องนอกเหนือหน้าที่)
  5. ปรับปรุง System Instruction ตามผลทดสอบ

7.5 Tuning — ปรับแต่ง Model ด้วยข้อมูลของเรา

Tuning (Fine-tuning) คือการ "สอน" Gemini ด้วยข้อมูลเฉพาะทางของเรา เพื่อให้ตอบได้ตรงตามความต้องการมากขึ้น

📚 ทำความเข้าใจ Tuning
🔹 Prompt Engineering (ไม่ tune)
  • บอกทุกอย่างใน prompt
  • ง่าย ไม่ต้องเตรียมข้อมูล
  • Prompt ยาว = ค่าใช้จ่ายสูง
  • เหมาะกับงานทั่วไป
🔸 Fine-tuning (tune)
  • สอน model ล่วงหน้า
  • ต้องเตรียมข้อมูลฝึก 100+ ตัวอย่าง
  • Prompt สั้น ตอบได้ทันที
  • เหมาะกับงานเฉพาะทาง
🔧 ขั้นตอน Tuning ใน AI Studio
  1. เตรียมข้อมูลฝึก: สร้างไฟล์ CSV/JSON มีคู่ input-output อย่างน้อย 100 ตัวอย่าง
  2. สร้าง Tuning Job: ไปที่ AI Studio → "New tuned model"
  3. อัปโหลดข้อมูล: เลือกไฟล์ข้อมูลฝึก
  4. ตั้งค่า: เลือก base model, จำนวน epochs, learning rate
  5. เริ่ม Tuning: คลิก "Start tuning" (ใช้เวลา 15-60 นาที)
  6. ทดสอบ: ใช้ tuned model ทดสอบกับข้อมูลใหม่
ตัวอย่างข้อมูลฝึก (Training Data) สำหรับงาน สนพ.:
// training_data.jsonl
{"input": "อยากฝึกอาชีพช่างเชื่อม ต้องทำอย่างไร", "output": "สมัครที่ สนพ.สุรินทร์ ใช้บัตรประชาชน+ทะเบียนบ้าน+รูปถ่าย อายุ 18+ ไม่จำกัดวุฒิ ฝึกฟรี"}
{"input": "ค่าใช้จ่ายฝึกอบรมเท่าไหร่", "output": "ไม่มีค่าใช้จ่าย ฝึกฟรีทุกหลักสูตร รัฐบาลสนับสนุน"}
{"input": "ฝึกกี่วัน", "output": "ขึ้นอยู่กับหลักสูตร ปกติ 18-30 ชม. (3-5 วัน) บางหลักสูตร 60 ชม. (10 วัน)"}
{"input": "ได้ใบรับรองไหม", "output": "ได้ครับ เมื่อฝึกครบชั่วโมงและผ่านการประเมิน จะได้วุฒิบัตรจากกรมพัฒนาฝีมือแรงงาน"}
// ... (ต้องมีอย่างน้อย 100 ตัวอย่าง)
⚠️ ข้อจำกัดของ Tuning:
  • ต้องเตรียมข้อมูลคุณภาพ — garbage in = garbage out
  • Free tier: tune ได้ จำกัดจำนวน model
  • รองรับเฉพาะ Gemini Flash (ไม่ tune Pro ได้ใน free tier)
  • ผลลัพธ์ขึ้นกับคุณภาพและจำนวนตัวอย่าง
🤔 เมื่อไรควร Fine-tune vs ใช้ Prompt Engineering
สถานการณ์ แนะนำ เหตุผล
ตอบคำถามทั่วไป Prompt Engineering Gemini รู้อยู่แล้ว แค่สั่งให้ดี

  1. Playground: ทดลองเปลี่ยน Temperature 5 ค่า (0.0, 0.5, 1.0, 1.5, 2.0) กับ prompt เดียวกัน จด screenshots ผลลัพธ์
