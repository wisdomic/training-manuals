5.1 เปิดใช้ Gemini ใน Google Workspace

ข้อกำหนดสำคัญ

การใช้ Gemini ภายใน Google Docs / Gmail / Sheets / Slides ต้องมี:

  • ตัวเลือกที่ 1: Google Workspace Business Standard/Plus ที่ Admin เปิด Gemini ไว้
  • ตัวเลือกที่ 2: Google One AI Premium ($20/เดือน = Gemini Advanced)
  • ตัวเลือกที่ 3: Google Workspace Individual

💡 ถ้าหน่วยงานใช้ Google Workspace for Education — ขึ้นอยู่กับ Admin ว่าเปิดให้ใช้หรือไม่

ตรวจสอบว่าเปิดใช้ Gemini ได้หรือไม่

1
เปิด Google Docs → สร้างเอกสารใหม่
ไปที่ docs.google.com → คลิก "+" สร้างเอกสารเปล่า
2
มองหาไอคอน ✨ (Gemini) ที่แถบเครื่องมือ
ถ้าเห็นปุ่ม "Help me write" หรือไอคอนดาว ✨ = ใช้ได้!
ถ้าไม่เห็น = ยังไม่ได้เปิดใช้ (ต้องสมัคร Gemini Advanced หรือให้ Admin เปิด)
3
สำหรับ Gmail — ตรวจสอบเช่นกัน
เปิด Gmail → กด Compose → มองหา "Help me write" ✨ ที่ด้านล่างช่องเขียนอีเมล

📸 ภาพหน้าจอ: ตำแหน่งไอคอน Gemini ✨ ใน Google Docs และ Gmail

สำหรับการอบรม: แม้ผู้เข้าอบรมอาจไม่มี Gemini Advanced แต่สามารถเรียนรู้หลักการ + ดูตัวอย่างจากวิทยากร → นำไปใช้เมื่อหน่วยงานเปิดใช้งาน หรือสมัครใช้เองในอนาคต ทั้งนี้ สามารถใช้ gemini.google.com (ฟรี) แล้ว copy ผลลัพธ์ไปวางใน Docs/Gmail ได้เช่นกัน

5.2 Gemini ใน Google Docs — Help me write, Refine, Summarize

Gemini ใน Google Docs มีฟีเจอร์หลัก 3 อย่าง:

Help me write

สร้างเนื้อหาใหม่จากคำสั่ง

Refine

ปรับปรุงเนื้อหาที่มีอยู่

Summarize

สรุปเนื้อหายาวให้สั้น

✏️ Help me write — สร้างเนื้อหาใหม่

1
คลิกในตำแหน่งที่ต้องการเพิ่มข้อความ
วาง cursor ณ ตำแหน่งในเอกสาร
2
คลิกไอคอน ✨ "Help me write" ที่แถบข้าง
หรือพิมพ์ "/" แล้วเลือก "Help me write"
3
พิมพ์คำสั่ง → กด "Create"
เช่น: "เขียนบันทึกข้อความเรื่องขออนุมัติจัดฝึกอบรม AI 5 วัน 100 คน"
4
ตรวจสอบ → กด "Insert" หรือ "Refine"
- Insert: แทรกเข้าเอกสาร
- Refine: ปรับปรุงต่อ (สั้นลง/ยาวขึ้น/เป็นทางการ/ลำลอง)
- Retry: สร้างใหม่

📸 ภาพหน้าจอ: Help me write ใน Google Docs — ช่องพิมพ์คำสั่ง + ผลลัพธ์

ตัวอย่างคำสั่งใน Google Docs
คำสั่ง ผลลัพธ์
"ร่างหนังสือเชิญประชุม วันที่ 15 ก.ค. 2569 เรื่อง แผนอบรม AI" ได้หนังสือเชิญประชุมรูปแบบราชการ
"เขียนสรุปผลการดำเนินงานไตรมาส 3 จากข้อมูลต่อไปนี้: ..." ได้รายงานสรุปพร้อมตาราง
"สร้างตาราง timeline โครงการ AI ระยะ 6 เดือน" ได้ตาราง Timeline พร้อมกิจกรรม
"เขียน job description ตำแหน่ง นักวิชาการคอมพิวเตอร์" ได้ JD ครบถ้วน

🔮 Refine — ปรับปรุงเนื้อหา

เลือกข้อความที่มีอยู่แล้ว → คลิกขวา → "Help me write" → เลือกการปรับปรุง:

Shorten
ทำให้สั้นลง
Elaborate
เพิ่มรายละเอียด
Formalize
ทำให้ทางการ
Rephrase
เขียนใหม่
สถานการณ์: มีย่อหน้ายาว 10 บรรทัด ต้องการสรุปให้สั้น

1. เลือก (highlight) ข้อความทั้งย่อหน้า
2. คลิกไอคอน ✨ → เลือก "Shorten"
3. Gemini จะย่อให้สั้นลง → กด "Replace" เพื่อแทนที่

📋 Summarize — สรุปเอกสาร

สำหรับเอกสารยาว สามารถใช้ Gemini สรุปทั้งเอกสารได้:

1
เปิด Side Panel (ไอคอน Gemini ✨ ด้านขวา)
คลิกไอคอน Gemini ที่มุมขวาบนของ Docs
2
พิมพ์ "สรุปเอกสารนี้" หรือ "Summarize this document"
Gemini จะอ่านทั้งเอกสารแล้วสรุปให้
3
ถามคำถามต่อได้
เช่น "ประเด็นสำคัญที่สุดคืออะไร?" / "มีอะไรที่ต้องดำเนินการ?" / "สรุปเป็น 3 ข้อ"
ใช้ในงานราชการ: เมื่อได้รับหนังสือ/รายงานยาวๆ → เปิดใน Google Docs → ใช้ Gemini สรุปให้ก่อนอ่านรายละเอียด ช่วยประหยัดเวลาได้มาก!

5.3 Gemini ใน Gmail — Draft, Reply, Summarize

Gemini ใน Gmail ช่วยร่างอีเมล, ตอบกลับ และสรุป email thread ยาวๆ ได้:

📧 ร่างอีเมลใหม่ (Help me write)

1
คลิก "Compose" เขียนอีเมลใหม่
กรอก To:, Subject: ตามปกติ
2
คลิกไอคอน ✨ "Help me write" ที่ด้านล่าง
จะปรากฏช่องพิมพ์คำสั่ง
3
พิมพ์คำสั่ง
เช่น: "เขียนอีเมลเชิญผู้อำนวยการโรงเรียน 5 แห่ง มาร่วมงานอบรม AI วันที่ 1-5 ก.ค. ภาษาทางการ"
4
ตรวจสอบ → Insert / Refine
ปรับให้สั้น/ยาว/ทางการ ได้ก่อนส่ง

📸 ภาพหน้าจอ: Help me write ใน Gmail Compose window

ตัวอย่างคำสั่งร่างอีเมล
เขียนอีเมลตอบกลับ ขอบคุณที่ตอบรับเป็นวิทยากร แจ้งรายละเอียด:
- วันที่ 1 ก.ค. 2569 เวลา 09:00-12:00
- สถานที่: ห้องประชุมชั้น 3 สนพ.สุรินทร์
- ค่าตอบแทน: ชม.ละ 600 บาท
- ขอ bio + รูปภาพสำหรับป้ายไวนิล
ภาษาสุภาพ ทางการ

↩️ ตอบกลับอีเมล (Reply suggestions)

เมื่อได้รับอีเมล Gemini จะเสนอ "ร่างคำตอบ" ให้เลือก:

1
เปิดอีเมลที่ต้องการตอบ → กด Reply
Gemini จะวิเคราะห์เนื้อหาอีเมลต้นทาง
2
ดู Suggested replies หรือกด ✨ Help me write
Gemini จะเสนอร่างตอบกลับที่เหมาะสมกับเนื้อหา
3
เลือก / แก้ไข → Send
ตรวจสอบเนื้อหาก่อนส่งเสมอ!

📋 สรุป Email Thread

เมื่อมี email thread ยาว (หลาย reply) Gemini จะแสดงปุ่ม "Summarize this email" ที่ด้านบน:

สถานการณ์: Email thread 15 replies เรื่องการจัดประชุม

→ กดปุ่ม "Summarize this email" ที่ด้านบน
→ Gemini สรุป: ใคร, ตกลงอะไร, มีอะไรค้าง, action items

ผลลัพธ์: "สรุป: ประชุมตกลงวันที่ 15 ก.ค. เวลา 13:00 ห้อง A สมาชิก 10 คน รอ A ส่งวาระ รอ B จองห้อง"

5.4 Gemini ใน Google Meet — Notes, Summary, Action Items

Gemini ใน Google Meet ช่วยบันทึกการประชุมอัตโนมัติ:

Take notes

บันทึกสรุปการประชุมอัตโนมัติ

Meeting Summary

สรุปประเด็นหลักเมื่อประชุมเสร็จ

Action Items

ดึง action items + ผู้รับผิดชอบ

วิธีใช้ Gemini ใน Meet

1
เริ่มประชุมใน Google Meet
จะเห็นไอคอน ✨ Gemini ที่แถบเครื่องมือด้านล่าง
2
คลิก "Take notes with Gemini"
Gemini จะเริ่มบันทึกและสรุปสิ่งที่พูดในที่ประชุม
3
ระหว่างประชุม — ถามได้
พิมพ์คำถาม เช่น "มีอะไรที่ตกลงกันแล้วบ้าง?" Gemini จะสรุปให้
4
หลังประชุม — ได้รับ Meeting Summary
จะได้ Email สรุปการประชุม + Action items + บันทึก ส่งให้ผู้เข้าร่วมทุกคน
ประโยชน์สำหรับราชการ:
  • ไม่ต้องจดบันทึกเอง — Gemini จดให้อัตโนมัติ
  • ผู้ที่มาสายได้สรุป "ก่อนหน้านี้คุยอะไรไป" ได้ทันที
  • ได้ Action items ชัดเจน ใครทำอะไร ภายในเมื่อไร
  • สรุปส่ง email อัตโนมัติ — ใช้เป็นรายงานการประชุมได้เลย

5.5 Workshop: ร่างเอกสารราชการ + Email ด้วย Gemini

Workshop ปฏิบัติ: ทดลองใช้ Gemini ช่วยงานจริง — ถ้าไม่มี Gemini ใน Docs/Gmail ให้ใช้ gemini.google.com แล้ว copy-paste ผลลัพธ์

Workshop 1: ร่างบันทึกข้อความ

โจทย์:

ร่างบันทึกข้อความ เรื่อง ขออนุมัติโครงการ "พัฒนาทักษะ AI สำหรับบุคลากรภาครัฐ"

  • จาก: ฝ่ายแผนงานของหน่วยงานท่าน
  • ถึง: ผู้อำนวยการ/หัวหน้าหน่วยงาน
  • ระยะเวลา: 5 วัน
  • ผู้เข้าร่วม: 50-100 คน
  • งบประมาณ: 100,000-200,000 บาท
ขั้นตอน:
  1. เปิด Google Docs → Help me write
  2. พิมพ์คำสั่งให้ร่างบันทึกข้อความ
  3. ตรวจสอบ → ใช้ Refine ปรับปรุง (Formalize, Elaborate)
  4. ลองให้ "Shorten" สรุปให้สั้นลง
  5. ส่งออก / Copy ไปใช้งาน

Workshop 2: ร่าง Email เชิญประชุม

โจทย์:

เขียน email เชิญผู้เกี่ยวข้อง 10 คน เข้าร่วมประชุมเตรียมงานฝึกอบรม

  • วันเวลา: ตามสะดวก
  • วาระ: แบ่งงาน, ตารางเวลา, เตรียมอุปกรณ์
  • ภาษา: ทางการ แต่เป็นกันเอง
ขั้นตอน:
  1. เปิด Gmail → Compose → Help me write ✨
  2. พิมพ์คำสั่ง → ตรวจสอบ → Insert
  3. ลองกด Refine → "Make more formal" / "Make shorter"
  4. เปรียบเทียบ 2-3 เวอร์ชัน

Workshop 3: สรุปเอกสารยาว

โจทย์:

เปิดเอกสารยาว (ระเบียบ/รายงาน/แผน) ใน Google Docs → ใช้ Gemini สรุป

ขั้นตอน:
  1. อัปโหลด PDF ไป Google Drive → เปิดด้วย Google Docs
  2. คลิก Gemini Side Panel ✨ (ขวาบน)
  3. พิมพ์ "สรุปเอกสารนี้เป็น 5 ข้อ"
  4. ถามต่อ: "ส่วนไหนสำคัญที่สุดสำหรับ...?"
  5. ลองให้สรุปเป็น action items

Workshop 4: Proofread + ปรับภาษา

โจทย์:

นำเอกสารที่เขียนไว้ → ให้ Gemini ตรวจสอบภาษา + ปรับปรุง

ขั้นตอน:
  1. เลือกข้อความทั้งหมดในเอกสาร
  2. Help me write → "ตรวจคำผิด + ปรับภาษาให้ทางการ"
  3. ใช้ Gemini Side Panel: "ช่วยตรวจเอกสารนี้ มีจุดไหนควรปรับปรุง?"
  4. ลอง: "แปลเป็นภาษาอังกฤษ" / "เปลี่ยนเป็นภาษาลำลอง"
แบบฝึกหัดท้ายบทที่ 5
  1. Google Docs: สร้างเอกสารใหม่ → ใช้ Help me write ร่างหนังสือราชการ 1 ฉบับ → Refine ให้ทางการขึ้น → Summarize
    ✓ ลองทั้ง 3 ฟังก์ชัน: Write + Refine + Summarize
  2. Gmail: ร่าง email 2 ฉบับ — (1) ทางการ (เชิญผู้อำนวยการ) (2) ลำลอง (แจ้งทีมงาน)
    ✓ เปรียบเทียบ tone ของทั้ง 2 ฉบับ
  3. สรุป Email: เปิด email thread ยาว (ถ้ามี) → ใช้ Summarize → ดูว่าสรุปได้ดีแค่ไหน
    ✓ ถ้าไม่มี email ยาว ให้ใช้ gemini.google.com วาง email แล้วให้สรุป
  4. Proofread: นำเอกสารที่เคยเขียน → paste เข้า Docs → ให้ Gemini ตรวจ + ปรับปรุง
    ✓ สังเกตว่า Gemini แก้จุดไหนบ้าง — ตรงใจหรือไม่
  5. (Bonus) Meet: ถ้ามีโอกาสประชุมผ่าน Meet → เปิด Gemini Notes → ตรวจสอบสรุป
    ✓ ดูว่าสรุปได้ครบถ้วน/แม่นยำแค่ไหน
