10.3 Workshop สุดท้าย: เลือกสร้างงาน 4 ระดับ

🏋️ Workshop สุดท้าย (60 นาที)

เลือกโจทย์ 1 ข้อ ตามระดับความถนัด สร้างผลงานจริง แล้วนำเสนอ!

⭐ ระดับ 1: ง่าย — ใช้ Gemini Web (เลือก 1 จาก 4 โจทย์)
1A. ร่างหนังสือราชการ 3 ฉบับ
  • คำสั่งแต่งตั้งคณะกรรมการฝึกอบรม
  • หนังสือเชิญวิทยากร
  • รายงานผลการฝึกอบรม
1B. สรุปเอกสาร 3 ฉบับ
  • อัปโหลดเอกสารจริง → สรุป
  • สร้าง Mind Map จากเนื้อหา
  • เปรียบเทียบ 2 เอกสาร
1C. สร้าง Content + รูป AI
  • เขียน Facebook post 5 โพสต์ประชาสัมพันธ์หลักสูตร
  • สร้างรูป AI ประกอบแต่ละโพสต์
  • เขียน Caption + Hashtag
1D. วิเคราะห์ข้อมูล + สรุป
  • อัปโหลดข้อมูลผลฝึกอบรม
  • ให้ Gemini วิเคราะห์แนวโน้ม
  • เขียนข้อเสนอแนะเชิงนโยบาย
⭐⭐ ระดับ 2: ปานกลาง — ใช้ Gemini Workspace (เลือก 1 จาก 4 โจทย์)
2A. รายงาน + Presentation ผลงาน
  • สร้างตาราง Sheets ผลฝึกอบรม
  • สร้างสูตร + กราฟ
  • สร้าง Slides นำเสนอ 8 สไลด์
2B. ระบบติดตามผลฝึกอบรม
  • Sheets ตารางข้อมูลผู้ฝึก 30 คน
  • สูตร VLOOKUP + COUNTIF + IF
  • Dashboard กราฟ 3 แบบ
2C. คู่มือการใช้งานระบบ
  • เขียนคู่มือใน Google Docs (Gemini ช่วย)
  • สร้าง Infographic ขั้นตอน
  • แปลเป็นภาษาอังกฤษ
2D. แผนงบประมาณ + รายงาน
  • ตาราง Sheets งบ 5 โครงการ
  • สูตรคำนวณ + กราฟเปรียบเทียบ
  • สรุปเป็น Presentation
⭐⭐⭐ ระดับ 3: ยาก — ใช้ AI Studio + API (เลือก 1 จาก 4 โจทย์)
3A. Chatbot FAQ หน่วยงาน
  • ออกแบบ System Instruction ละเอียด
  • เพิ่ม FAQ 20+ ข้อ
  • ทดสอบ 15+ คำถาม + edge cases
3B. ระบบจำแนกเอกสารอัตโนมัติ
  • Structured Prompt จำแนก 8 ประเภท
  • ทดสอบ accuracy 20+ เอกสาร
  • Export code + สาธิต
3C. Gemini API ใน Sheets (Custom Function)
  • สร้าง =GEMINI() function
  • =GEMINI_TRANSLATE(), =GEMINI_SUMMARIZE()
  • ทดสอบกับข้อมูลจริง 10 แถว
3D. ระบบสรุปรายงานประชุม
  • อัปโหลด PDF → สรุปอัตโนมัติ
  • สกัดมติ/ผู้รับผิดชอบ/กำหนดส่ง
  • ส่ง Email สรุปให้ผู้เกี่ยวข้อง
⭐⭐⭐⭐ ระดับ 4: ท้าทาย — Code Assist + ระบบเต็ม (เลือก 1 จาก 4 โจทย์)
4A. Web App ลงทะเบียนฝึกอบรม
  • HTML Form + Validation + GAS Backend
  • บันทึก Sheets + Email ยืนยัน
  • Deploy เป็น Web App จริง
4B. AI Chatbot ใน Sheets (Full)
  • Sidebar Chat UI + Gemini API
  • อ่านข้อมูลจาก Sheet ตอบคำถาม
  • บันทึก log + Analytics
4C. ระบบประเมินผลอัจฉริยะ
  • ฟอร์มประเมิน + GAS + AI วิเคราะห์
  • Gemini สรุปข้อเสนอแนะ
  • สร้างรายงาน PDF อัตโนมัติ
4D. ระบบแจ้งเตือนอัจฉริยะ Line + AI
  • Trigger อ่านข้อมูล Sheets ทุกวัน
  • Gemini วิเคราะห์ + สร้างข้อความ
  • ส่ง Line Notify อัตโนมัติ

10.5 สรุป + Tips + แหล่งเรียนรู้ + ปิดอบรม

💡 10 Tips สำหรับใช้ Gemini AI ในงานราชการ
  1. เริ่มจาก Gemini Web: ใช้งานง่าย ฟรี ครอบคลุมงาน 80%
  2. เขียน Prompt ให้ชัดเจน: ยิ่งละเอียด ยิ่งได้ผลดี (บริบท + บทบาท + รูปแบบ)
  3. ตรวจสอบทุกครั้ง: AI ผิดได้! ต้อง fact-check ก่อนใช้จริง
  4. ห้ามใส่ข้อมูลลับ: ข้อมูลส่วนบุคคล ข้อมูลราชการลับ ห้ามส่งให้ AI
  5. ใช้ Gems/Custom Instructions: สร้างผู้ช่วยเฉพาะทางสำหรับงานประจำ
  1. ฝึกทุกวัน: ยิ่งใช้บ่อย ยิ่งเขียน prompt เก่งขึ้น
  2. ใช้ Iterative approach: ถ้าไม่ได้ผล ปรับ prompt แล้วลองใหม่
  3. เรียนรู้จากตัวอย่าง: ดูว่าคนอื่นใช้ AI อย่างไร แล้วประยุกต์
  4. อัปเดตอยู่เสมอ: AI พัฒนาเร็วมาก ฟีเจอร์ใหม่ออกทุกเดือน
  5. แชร์ความรู้: สอนเพื่อนร่วมงาน ยิ่งแชร์ ยิ่งเก่ง
