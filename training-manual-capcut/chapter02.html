📂 นำเข้าสื่อและ Timeline

Import วิดีโอ/ภาพ/เสียง, จัดการ Timeline, ลาก+วาง, ตัด/แบ่ง/ลบคลิป

🎯 วัตถุประสงค์การเรียนรู้
  • นำเข้าไฟล์วิดีโอ ภาพ และเสียงเข้า CapCut ได้
  • เข้าใจโครงสร้างและการทำงานของ Timeline
  • ลากวางไฟล์ลง Timeline ได้อย่างถูกต้อง
  • ตัด แบ่ง ลบ และจัดเรียงคลิปได้
  • ใช้ Keyboard Shortcut ทำงานได้เร็วขึ้น

2.1 การนำเข้าไฟล์ (Import)

ก่อนจะตัดต่อวิดีโอได้ ต้องนำไฟล์เข้ามาในโปรเจกต์ก่อน CapCut รองรับไฟล์หลากหลายรูปแบบ:

🎬 วิดีโอ

MP4, MOV, AVI, MKV, FLV, WebM, GIF

🖼️ ภาพ

JPG, PNG, WEBP, BMP, TIFF, SVG

🎵 เสียง

MP3, WAV, AAC, FLAC, M4A, OGG

วิธีการ Import มี 3 วิธี:

1 คลิกปุ่ม Import: คลิกปุ่ม "+" หรือ "Import" ใน Media Panel → เลือกไฟล์จากคอมพิวเตอร์
2 ลากวาง (Drag & Drop): ลากไฟล์จาก Windows Explorer มาวางใน Media Panel โดยตรง
3 Keyboard Shortcut: กด Ctrl + I จะเปิดหน้าต่างเลือกไฟล์
💡 Tips: สามารถเลือกหลายไฟล์พร้อมกันได้ โดยกด Ctrl ค้างแล้วคลิกเลือกทีละไฟล์ หรือกด Ctrl + A เพื่อเลือกทั้งหมด

2.2 ทำความรู้จัก Timeline

Timeline คือพื้นที่ด้านล่างของหน้าจอ Editor เปรียบเสมือน "กระดานเล่าเรื่อง" ที่เราจัดเรียงคลิปตามลำดับเวลา:

  • แกนนอน (Horizontal) — แทนเวลา จากซ้ายไปขวา (เริ่มต้น → จบ)
  • แกนตั้ง (Vertical) — แทนเลเยอร์ ซ้อนทับกันได้ (ล่างสุดคือพื้นหลัง)
  • Playhead (เส้นสีขาว) — ตำแหน่งปัจจุบันที่กำลังเล่น/แก้ไข
  • Track (แถว) — แต่ละแถวคือ 1 เลเยอร์ สามารถมีได้หลาย Track

ประเภทของ Track:

Track สีสัญลักษณ์ ใช้สำหรับ
Video Track ม่วง/น้ำเงิน วิดีโอ, ภาพ
Audio Track เขียว เสียง, เพลง, Voice
Text Track ชมพู ข้อความ, ซับไตเติ้ล
Effect Track ส้ม เอฟเฟกต์พิเศษ

2.3 ลากวางไฟล์ลง Timeline

วิธีนำไฟล์จาก Media Panel ลง Timeline:

1 ลากวาง: คลิกค้างที่ไฟล์ใน Media Panel แล้วลากมาวางใน Timeline
2 ปุ่ม +: เลื่อน Playhead ไปตำแหน่งที่ต้องการ → คลิกปุ่ม "+" บนไฟล์
3 Double-click: ดับเบิลคลิกที่ไฟล์ จะวางต่อท้ายคลิปสุดท้าย

หลักการจัดเรียง:

  • คลิปที่อยู่ ซ้าย จะเล่นก่อน คลิปที่อยู่ ขวา จะเล่นทีหลัง
  • คลิปที่อยู่ Track บน จะแสดงทับคลิปที่อยู่ Track ล่าง
  • สามารถลากย้ายตำแหน่งคลิปได้ตลอดเวลา

2.4 ตัด แบ่ง ลบ คลิป

เครื่องมือที่ใช้บ่อยที่สุดในการตัดต่อ:

🔪 Split (แบ่งคลิป)
  1. เลื่อน Playhead ไปตำแหน่งที่ต้องการแบ่ง
  2. คลิกที่คลิปให้เลือก (ขอบจะเป็นสีฟ้า)
  3. กด Ctrl + B หรือคลิกไอคอน ✂️ (กรรไกร)
  4. คลิปจะถูกแบ่งเป็น 2 ส่วน ณ ตำแหน่ง Playhead
🗑️ Delete (ลบคลิป)
  1. คลิกเลือกคลิปที่ต้องการลบ
  2. กด Delete หรือ Backspace
  3. คลิปจะถูกลบ และคลิปที่เหลือจะเลื่อนมาชิดกันอัตโนมัติ
✂️ Trim (ตัดหัว-ท้าย)
  1. เลื่อนเมาส์ไปที่ขอบซ้ายหรือขวาของคลิป (จะเห็นลูกศร ↔)
  2. คลิกค้างแล้วลากเข้ามา เพื่อตัดส่วนหัวหรือท้ายออก
  3. ปล่อยเมาส์ ส่วนที่ถูกตัดจะหายไป
📋 Copy & Paste (คัดลอก-วาง)
  • Ctrl + C — คัดลอกคลิปที่เลือก
  • Ctrl + V — วางคลิปที่ตำแหน่ง Playhead
  • Ctrl + D — Duplicate (สร้างสำเนาต่อท้ายทันที)

2.5 Zoom Timeline และ Navigation

เมื่อวิดีโอยาว จำเป็นต้องซูมเข้า-ออก Timeline:

  • ซูมเข้า: กด Ctrl + + หรือ Ctrl + Scroll Up
  • ซูมออก: กด Ctrl + - หรือ Ctrl + Scroll Down
  • Fit Timeline: กด Ctrl + 0 เพื่อให้เห็นทั้งโปรเจกต์
  • เลื่อน Timeline: กดปุ่มกลางเมาส์ค้าง แล้วลากซ้าย-ขวา

การเล่นวิดีโอ:

  • Space — Play/Pause
  • ← → — เลื่อนทีละ Frame
  • Home — กลับไปจุดเริ่มต้น
  • End — ไปจุดสิ้นสุด

🔬 Lab 2: นำเข้าสื่อและตัดต่อเบื้องต้น

เวลา: 20 นาที

ไฟล์ที่ใช้: วิทยากรเตรียมให้ (โฟลเดอร์ "Lab02")

  1. สร้างโปรเจกต์ใหม่ ชื่อ "Lab02_[ชื่อ]" อัตราส่วน 16:9
  2. Import วิดีโอ 3 คลิป + ภาพ 2 ภาพ + เพลง 1 เพลง จากโฟลเดอร์ Lab02
  3. ลากวิดีโอ 3 คลิป ลง Timeline เรียงต่อกัน
  4. ลากเพลงลง Audio Track
  5. ฝึก Split: แบ่งคลิปที่ 2 ออกเป็น 2 ส่วน ลบส่วนหลังทิ้ง
  6. ฝึก Trim: ตัดส่วนหัว 3 วินาทีของคลิปที่ 1 ออก
  7. ลากภาพ 1 ภาพ ไปวางแทรกระหว่างคลิปที่ 1 กับคลิปที่ 2
  8. Duplicate คลิปที่ 3 ไปวางท้ายสุด
  9. กด Space ดูตัวอย่างตั้งแต่ต้นจนจบ
  10. ซูมเข้า-ออก Timeline สังเกตความแตกต่าง

✅ สำเร็จเมื่อ: มีคลิปใน Timeline ≥ 5 ชิ้น เรียงต่อกัน เล่นได้ตั้งแต่ต้นจนจบ

💡 เคล็ดลับบทที่ 2
  • Snap (แม่เหล็ก): คลิปจะ "ดูด" เข้าหากันเมื่อลากใกล้ๆ ช่วยไม่ให้มีช่องว่าง
  • Undo: กด Ctrl + Z ย้อนกลับได้ไม่จำกัด ไม่ต้องกลัวพลาด
  • จัดโฟลเดอร์: ก่อน Import ควรจัดไฟล์ไว้ในโฟลเดอร์เดียวกัน จะหาง่าย
  • ลบช่องว่าง: คลิกขวาที่ช่องว่างใน Timeline → "Delete gap" เพื่อลบช่องว่าง
  • Lock Track: คลิกไอคอนกุญแจ 🔒 เพื่อล็อค Track ไม่ให้แก้ไขโดยบังเอิญ
