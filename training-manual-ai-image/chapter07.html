7.1 โครงสร้าง Prompt สร้างภาพที่ดี

🧩 5 องค์ประกอบของ Prompt ภาพ
# องค์ประกอบ คำอธิบาย ตัวอย่าง
1 Subject สิ่งที่ต้องการให้อยู่ในภาพ "แมวสีส้ม", "กลุ่มคนพิการ"
2 Style สไตล์/เทคนิคการวาด "watercolor", "flat design", "realistic"
3 Composition มุมกล้อง/การจัดองค์ประกอบ "close-up", "bird's eye view", "wide shot"
4 Mood/Lighting บรรยากาศ/แสง "warm lighting", "สดใส", "มืดลึกลับ"
5 Technical ขนาด/คุณภาพ "16:9", "high resolution", "4K"
📝 สูตรรวม
[Subject] + [Style] + [Composition] + [Mood] + [Technical]

ตัวอย่างเต็ม:

"กลุ่มคนพิการทางการมองเห็นกำลังใช้คอมพิวเตอร์อย่างมีความสุข, สไตล์ flat design สีสดใส, มุมกว้าง เห็นทั้งห้อง, บรรยากาศอบอุ่นแสงธรรมชาติ, อัตราส่วน 16:9 ความละเอียดสูง"

7.3 Negative Prompt คืออะไร

🚫 Negative Prompt = "สิ่งที่ไม่ต้องการ"

Negative Prompt คือการบอก AI ว่า "อย่าใส่สิ่งนี้" ในภาพ — ช่วยหลีกเลี่ยงสิ่งที่ไม่ต้องการ

❌ Negative Prompt ตัวอย่าง:
"ห้ามมีตัวอักษรในภาพ, ไม่ต้องมีกรอบ, ไม่ต้องมีลายน้ำ"

วิธีใช้กับ DALL-E (ChatGPT):

"สร้างภาพ... [เนื้อหา] ... ห้ามมีตัวอักษรในภาพ ห้ามมีลายน้ำ"

วิธีใช้กับ Firefly: มีช่อง "Exclude" ให้พิมพ์สิ่งที่ไม่ต้องการแยกต่างหาก

💡 Negative Prompt ที่ใช้บ่อย
  • no text, no letters — ไม่ต้องมีตัวอักษร
  • no watermark — ไม่ต้องมีลายน้ำ
  • no border, no frame — ไม่ต้องมีกรอบ
  • no blurry — ไม่เบลอ
  • no extra fingers — ไม่ต้องมีนิ้วเกิน (ปัญหาที่พบบ่อย)

7.4 ตัวอย่าง Prompt สำหรับงานราชการ

📋 Prompt พร้อมใช้ 10 แบบ
1. โปสเตอร์อบรม:
"สร้างภาพพื้นหลังโปสเตอร์อบรม AI สีน้ำเงินขาว มีไอคอนเทคโนโลยี สไตล์ flat design มีพื้นที่ว่างตรงกลางสำหรับใส่ข้อความ ขนาดแนวตั้ง"
2. Banner Facebook:
"สร้างภาพ Banner ขนาด 16:9 สำหรับ Facebook Page องค์กรคนพิการ สีสดใส มีภาพคนหลากหลายใช้เทคโนโลยี สไตล์ modern gradient"
3. โปสเตอร์รับสมัคร:
"สร้างภาพพื้นหลังประกาศรับสมัคร สีเขียวขาว สดใส มีไอคอนคนทำงาน สไตล์ flat design มืออาชีพ"
4. ภาพประกอบรายงาน:
"สร้างภาพ illustration คนพิการใช้ AI ในการทำงาน สไตล์ corporate สีน้ำเงินเทา มืออาชีพ"
5. ไอคอนโครงการ:
"สร้างไอคอนกลมๆ สำหรับโครงการ AI เพื่อคนพิการ สีม่วงน้ำเงิน มีภาพหุ่นยนต์จับมือคน สไตล์ flat minimal พื้นหลังขาว"

💡 Tips สำหรับบทนี้
  • เริ่มจาก Prompt สั้น → ดูผล → เพิ่มรายละเอียดทีละนิด
  • จด Prompt ที่ได้ผลดีไว้ — ใช้ซ้ำได้เป็น "Prompt Library"
  • ใช้ ChatGPT ช่วยเขียน Prompt ภาษาอังกฤษให้ได้ — "ช่วยเขียน prompt สร้างภาพ..."
  • ถ้า AI ใส่ตัวอักษรในภาพ — เพิ่ม "no text" ใน Prompt
📝 สรุปท้ายบท
  • 5 องค์ประกอบ Prompt ภาพ: Subject, Style, Composition, Mood, Technical
  • ยิ่งละเอียด ยิ่งตรงใจ — แต่อย่ายาวเกินไป
  • Negative Prompt: บอก AI ว่าไม่ต้องการอะไร
  • จด Prompt ที่ดีไว้ — สร้าง Prompt Library ของตัวเอง
  • ฝึกบ่อยๆ — ยิ่งฝึก ยิ่งเขียน Prompt เก่ง
