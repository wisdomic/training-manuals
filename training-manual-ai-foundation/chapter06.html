📖 พื้นฐาน AI ทุกมิติ บทที่ 6

บทที่ 6

Prompt Engineering ที่ใช้งานจริง

เรียนรู้เทคนิคการเขียนคำสั่งให้ AI เข้าใจ และได้ผลลัพธ์ตรงใจ

🎯 วัตถุประสงค์การเรียนรู้
  • เขียน Prompt ที่ดีได้ โดยใช้หลัก Role + Context + Task + Format
  • ใช้เทคนิค Zero-shot, Few-shot และ Chain-of-Thought ได้อย่างเหมาะสม
  • ประยุกต์ใช้ Prompt กับงานราชการและงานด้านคนพิการได้จริง

6.1 Prompt คืออะไร ทำไมสำคัญ

📝 Prompt = วิธีสั่งงานผู้ช่วย AI

ลองนึกภาพว่า AI คือ "ผู้ช่วยใหม่" ที่เก่งมากแต่ไม่รู้จักคุณ — ถ้าคุณสั่งงานไม่ชัด ผู้ช่วยก็ทำงานไม่ตรงใจ

Prompt ที่ดี = สั่งงานชัดเจน → ได้ผลลัพธ์ตรงใจ
Prompt ที่แย่ = สั่งงานคลุมเครือ → ได้ผลลัพธ์ไม่ตรง ต้องถามซ้ำหลายรอบ

🔄 เปรียบเทียบ Prompt ดี vs แย่
# ❌ Prompt แย่ ✅ Prompt ดี ทำไมดีกว่า
1 "เขียนหนังสือให้หน่อย" "เขียนหนังสือราชการถึงผู้ว่าราชการจังหวัดสุรินทร์ เรื่องขอสนับสนุนงบประมาณจัดอบรม AI สำหรับคนพิการ จำนวน 50 คน วันที่ 15 ก.ค. 2569" ระบุผู้รับ เรื่อง รายละเอียดครบ
2 "สรุปประชุมให้หน่อย" "สรุปรายงานการประชุมเป็น 5 หัวข้อหลัก: ผู้เข้าร่วม วาระ มติ ผู้รับผิดชอบ กำหนดส่ง โดยเขียนกระชับแต่ละหัวข้อไม่เกิน 3 บรรทัด" บอกรูปแบบ จำนวน และความยาว
3 "ช่วยแปลภาษา" "แปลข้อความนี้จากภาษาอังกฤษเป็นภาษาไทย ใช้ภาษาราชการ ไม่ทับศัพท์ ถ้าจำเป็นต้องใช้คำทับศัพท์ให้วงเล็บคำอังกฤษกำกับ" ระบุภาษาต้นทาง ปลายทาง ระดับภาษา

6.2 หลัก 4 อย่างของ Prompt ที่ดี: Role + Context + Task + Format

Prompt ที่ดีประกอบด้วย 4 ส่วน เหมือน "สูตรอาหาร" — ใส่ครบยิ่งอร่อย:

องค์ประกอบ ความหมาย เปรียบเทียบ ตัวอย่าง
🎭 Role กำหนดบทบาทให้ AI เหมือนบอกว่า "คุณเป็นใคร" "คุณเป็นผู้เชี่ยวชาญด้านสิทธิคนพิการ"
📋 Context ให้ข้อมูลพื้นฐาน/บริบท เหมือนบอก "สถานการณ์ตอนนี้เป็นยังไง" "กำลังจัดอบรม AI ให้คนพิการทางสายตา 30 คน"
✅ Task สั่งงานที่ต้องทำ เหมือนบอก "ต้องทำอะไร" "เขียนกำหนดการอบรม 1 วัน"
📐 Format กำหนดรูปแบบผลลัพธ์ เหมือนบอก "ส่งมาในรูปแบบไหน" "เป็นตาราง มี 3 คอลัมน์: เวลา กิจกรรม ผู้รับผิดชอบ"
📝 ตัวอย่าง Prompt ที่ใช้หลัก 4 อย่างครบ
ตัวอย่างที่ 1 — ร่างหนังสือราชการ

🎭 Role: คุณเป็นเจ้าหน้าที่ธุรการที่เชี่ยวชาญหนังสือราชการ
📋 Context: สนพ.สุรินทร์ จะจัดอบรม "AI สำหรับคนพิการ" วันที่ 20 ส.ค. 2569 ที่ห้องประชุมชั้น 3
✅ Task: เขียนหนังสือเชิญวิทยากรภายนอก (ผศ.ดร.สมชาย ใจดี มหาวิทยาลัยราชภัฏสุรินทร์)
📐 Format: รูปแบบหนังสือราชการ มีเลขที่หนังสือ วันที่ เรื่อง เรียน สิ่งที่ส่งมาด้วย

ตัวอย่างที่ 2 — สรุปรายงาน

🎭 Role: คุณเป็นเลขานุการที่ประชุม
📋 Context: ประชุมคณะทำงาน AI เพื่อคนพิการ ครั้งที่ 3/2569 มีผู้เข้าร่วม 12 คน เมื่อวันที่ 10 ก.ค. 2569
✅ Task: สรุปรายงานการประชุมจากบันทึกย่อที่แนบมา
📐 Format: แบ่งเป็นหัวข้อ: วาระ, มติ, ผู้รับผิดชอบ, กำหนดส่ง — ใช้ bullet points

ตัวอย่างที่ 3 — เขียนโครงการ

🎭 Role: คุณเป็นนักวิเคราะห์นโยบายและแผน
📋 Context: ต้องเขียนโครงการของบประมาณปี 2570 เรื่องจัดอบรม Digital Literacy ให้คนพิการ จำนวน 5 รุ่น รุ่นละ 30 คน
✅ Task: เขียนโครงการฉบับเต็ม
📐 Format: ครบทุกหัวข้อ: ชื่อโครงการ หลักการและเหตุผล วัตถุประสงค์ เป้าหมาย วิธีดำเนินงาน งบประมาณ ตัวชี้วัด ผลที่คาดว่าจะได้รับ

6.3 Zero-shot vs Few-shot vs Chain-of-Thought

เทคนิค 3 แบบในการเขียน Prompt — เรียงจากง่ายไปซับซ้อน:

🎯 Zero-shot
สั่งตรงๆ ไม่ให้ตัวอย่าง

คำอธิบาย: บอก AI ว่าต้องทำอะไร โดยไม่ต้องให้ตัวอย่าง

เหมาะกับ: งานง่ายๆ ตรงไปตรงมา

ข้อดี: เขียนสั้น รวดเร็ว

ข้อเสีย: ผลลัพธ์อาจไม่ตรงรูปแบบที่ต้องการ

📝 Few-shot
ให้ตัวอย่าง 2-3 ข้อ

คำอธิบาย: ให้ตัวอย่างผลลัพธ์ที่ต้องการ 2-3 ข้อ แล้ว AI จะทำตามรูปแบบเดียวกัน

เหมาะกับ: งานที่ต้องการรูปแบบเฉพาะ

ข้อดี: ผลลัพธ์ตรงรูปแบบ

ข้อเสีย: ต้องเขียน Prompt ยาวขึ้น

🔗 Chain-of-Thought
ให้ AI คิดทีละขั้น

คำอธิบาย: สั่งให้ AI อธิบายขั้นตอนการคิดก่อนตอบ เพื่อให้ได้คำตอบที่ถูกต้องมากขึ้น

เหมาะกับ: งานวิเคราะห์ ตัดสินใจ คำนวณ

ข้อดี: คำตอบแม่นยำ ตรวจสอบได้

ข้อเสีย: ผลลัพธ์ยาว ใช้เวลามากขึ้น

📝 ตัวอย่าง Prompt แต่ละแบบ
🎯 Zero-shot — สั่งตรงๆ

"จัดหมวดหมู่ข้อความนี้ว่าเป็น ร้องเรียน/สอบถาม/ชมเชย: 'ขอสอบถามเรื่องสิทธิบัตรคนพิการครับ ต้องใช้เอกสารอะไรบ้าง'"

📝 Few-shot — ให้ตัวอย่าง

"จัดหมวดหมู่ข้อความต่อไปนี้:

ตัวอย่าง 1: 'ทำไมรอนานมากยังไม่ได้บัตร' → ร้องเรียน
ตัวอย่าง 2: 'ขอบคุณเจ้าหน้าที่มากครับ บริการดี' → ชมเชย
ตัวอย่าง 3: 'อยากทราบว่าต้องเตรียมเอกสารอะไร' → สอบถาม

ข้อความ: 'ยื่นเรื่องไป 3 เดือนแล้วยังไม่ได้เงินเลย'"

🔗 Chain-of-Thought — คิดทีละขั้น

"วิเคราะห์ว่าโครงการนี้ควรได้รับอนุมัติหรือไม่:
โครงการอบรม AI ให้คนพิการ งบ 200,000 บาท 50 คน 3 วัน

ให้คิดทีละขั้นตอน:
1. ตรวจสอบความสอดคล้องกับนโยบาย
2. วิเคราะห์งบประมาณต่อหัว
3. ประเมินความคุ้มค่า
4. สรุปข้อเสนอแนะ
แล้วจึงให้คำตอบสุดท้าย"

6.4 เทคนิค Prompt สำหรับงานราชการโดยเฉพาะ

รวม 5 Prompt ที่ใช้บ่อยในงานราชการ — คัดลอกไปใช้ได้เลย:

📄 1. ร่างหนังสือราชการ

"คุณเป็นเจ้าหน้าที่ธุรการ สนพ.สุรินทร์ ช่วยร่างหนังสือราชการ เรื่อง ขอเชิญประชุมคณะทำงานโครงการ AI เพื่อคนพิการ ครั้งที่ 1/2569 ถึง คณะทำงานทุกท่าน กำหนดประชุมวันที่ 25 ก.ค. 2569 เวลา 09.00-12.00 น. ห้องประชุมชั้น 2 รูปแบบหนังสือราชการมาตรฐาน มีเลขที่ ลงวันที่ เรียน อ้างถึง สิ่งที่ส่งมาด้วย"

📋 2. สรุปรายงานการประชุม

"คุณเป็นเลขานุการที่ประชุม สรุปรายงานการประชุมจากข้อมูลต่อไปนี้ โดยแบ่งเป็น: 1) ข้อมูลการประชุม (วันเวลา สถานที่ ผู้เข้าร่วม) 2) วาระที่ 1-2-3 แต่ละวาระมี: เรื่อง สาระสำคัญ มติ 3) เรื่องอื่นๆ 4) กำหนดประชุมครั้งต่อไป — ใช้ภาษาราชการ กระชับ"

📊 3. เขียนโครงการ

"คุณเป็นนักวิเคราะห์นโยบายและแผน เขียนโครงการ 'พัฒนาทักษะดิจิทัลสำหรับคนพิการด้วย AI' ปีงบ 2570 กลุ่มเป้าหมาย: คนพิการ 150 คน (5 รุ่นๆ ละ 30 คน) งบประมาณ 500,000 บาท ครบทุกหัวข้อ: หลักการเหตุผล วัตถุประสงค์ เป้าหมาย วิธีดำเนินงาน งบประมาณ(แยกรายละเอียด) ตัวชี้วัด ผลที่คาดว่าจะได้รับ"

🌐 4. แปลเอกสาร

"คุณเป็นนักแปลเอกสารราชการ แปลข้อความต่อไปนี้จากภาษาอังกฤษเป็นภาษาไทย: [วางข้อความ] เงื่อนไข: 1) ใช้ภาษาราชการ 2) คำศัพท์เฉพาะทางให้ใส่วงเล็บภาษาอังกฤษกำกับ 3) ถ้ามีชื่อเฉพาะให้ทับศัพท์ 4) รักษาความหมายเดิมครบถ้วน"

📅 5. สร้างกำหนดการ

"คุณเป็นผู้จัดอบรม สร้างกำหนดการอบรม 'พื้นฐาน AI สำหรับคนพิการ' ระยะเวลา 2 วัน (09.00-16.30 น.) กลุ่มเป้าหมาย: เจ้าหน้าที่องค์กรคนพิการ ไม่มีพื้นฐาน IT รูปแบบตาราง: เวลา | หัวข้อ | รายละเอียด | วิทยากร รวมพัก tea break เช้า-บ่าย + พักเที่ยง"

6.5 ตัวอย่าง Prompt ที่ดีและไม่ดี

เปรียบเทียบ 5 คู่ — ลองสังเกตว่า Prompt ดีต่างจาก Prompt แย่อย่างไร:

# สถานการณ์ ❌ Prompt แย่ ✅ Prompt ดี ปัญหา
1 ร่างอีเมล "เขียนอีเมล" "เขียนอีเมลถึงผู้อำนวยการสำนักงานพัฒนาฝีมือแรงงานสุรินทร์ เรื่องขออนุมัติเดินทางไปราชการ วันที่ 5-7 ส.ค. 2569 ที่กรุงเทพฯ เพื่ออบรมหลักสูตร AI — ภาษาสุภาพเป็นทางการ" ไม่บอกรายละเอียด
2 วิเคราะห์ข้อมูล "วิเคราะห์ตัวเลขนี้ให้" "วิเคราะห์ข้อมูลจำนวนคนพิการที่ลงทะเบียน 5 ปีย้อนหลัง (2565-2569) หาแนวโน้มเพิ่ม/ลด สาเหตุที่เป็นไปได้ และข้อเสนอแนะเชิงนโยบาย สรุปเป็น 3 ย่อหน้า" ไม่บอกว่าวิเคราะห์อะไร
3 สร้าง Infographic "ทำรูปสวยๆ" "สร้างภาพ Infographic สิทธิคนพิการ 7 ด้าน สไตล์ flat design สีม่วง-เหลือง พื้นหลังขาว มี icon แต่ละด้าน ข้อความภาษาไทย ขนาด A4 แนวตั้ง" ไม่บอกเนื้อหา สไตล์ ขนาด
4 ตอบคำถาม "PDPA คืออะไร" "อธิบาย PDPA (พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล) สำหรับเจ้าหน้าที่องค์กรคนพิการที่ไม่มีพื้นฐานกฎหมาย สรุป 5 ข้อหลักที่ต้องรู้ ยกตัวอย่างสถานการณ์จริงในการทำงานประกอบ" กว้างเกินไป ไม่บอกระดับ
5 เขียนป้ายประชาสัมพันธ์ "เขียนป้ายอบรม" "เขียนข้อความสำหรับป้ายประชาสัมพันธ์อบรม 'AI เพื่อคนพิการ' ขนาด banner Facebook (1200x628px) ต้องมี: ชื่องาน วันเวลา สถานที่ กลุ่มเป้าหมาย สิ่งที่ได้รับ วิธีสมัคร — ข้อความสั้นกระชับอ่านจากมือถือได้" ไม่บอกรูปแบบ ช่องทาง

6.6 การปรับ Prompt เมื่อผลไม่ตรงใจ

ถ้า AI ตอบไม่ตรงใจ ไม่ต้องเริ่มใหม่ — ลอง 5 เทคนิคนี้:

1. ➕ เพิ่มบริบท

บอก AI เพิ่มว่า:

  • ผู้อ่านคือใคร
  • ใช้ที่ไหน
  • จุดประสงค์คืออะไร
"ผู้อ่านคือคนพิการทางสายตา ต้องสรุปสั้นเพราะใช้โปรแกรมอ่านหน้าจอ"
2. 🎭 เปลี่ยน Role

เปลี่ยนบทบาท AI:

  • จาก "ผู้เชี่ยวชาญ" → "ครู"
  • จาก "นักเขียน" → "นักข่าว"
  • จาก "ทั่วไป" → "เฉพาะทาง"
"ตอบในฐานะครูที่สอนคนไม่มีพื้นฐาน IT"
3. 📐 ระบุ Format

บอกรูปแบบชัดเจน:

  • "เป็นตาราง 3 คอลัมน์"
  • "เป็น bullet points 5 ข้อ"
  • "ยาวไม่เกิน 200 คำ"
"สรุปเป็นตาราง: ประเด็น | รายละเอียด | ข้อเสนอ"
4. 📝 ให้ตัวอย่าง

แสดงตัวอย่างที่ต้องการ:

  • ให้ตัวอย่าง 1-2 ข้อ
  • แสดงรูปแบบที่ถูกต้อง
  • แสดงสิ่งที่ไม่ต้องการ
"ตัวอย่างที่ต้องการ: 'สิทธิข้อ 1: การศึกษา — คนพิการมีสิทธิ์เรียนฟรี'"
5. 🔄 ถามต่อ (Follow-up)

ไม่ต้องเริ่มใหม่ ถามต่อ:

  • "ปรับให้สั้นกว่านี้"
  • "เพิ่มรายละเอียดข้อ 3"
  • "เปลี่ยนภาษาให้ง่ายขึ้น"
"ดีแล้ว แต่ช่วยปรับภาษาให้เข้าใจง่ายขึ้น เหมือนพูดกับคนทั่วไป"

💻 Lab 6.3 — เขียน Prompt สร้างรูปภาพประชาสัมพันธ์
1 โจทย์: สร้างรูปภาพประชาสัมพันธ์กิจกรรม "วันคนพิการสากล จังหวัดสุรินทร์ ประจำปี 2569" รายละเอียด:
  • กิจกรรม: การแข่งขันกีฬาคนพิการ + นิทรรศการ AI ช่วยเหลือคนพิการ
  • วันที่: 3 ธันวาคม 2569
  • สถานที่: สนามกีฬาจังหวัดสุรินทร์
  • ต้องการรูปสไตล์: สดใส เป็นมิตร แสดงความหลากหลาย
2 เขียน Prompt สำหรับ AI สร้างรูปภาพ (เช่น ChatGPT DALL-E, Gemini Imagen):

✍️ เขียน Prompt ภาพของคุณ: _______________
(ระบุ: เนื้อหา สไตล์ สี ขนาด องค์ประกอบ)

3 ทดลองสร้างภาพ → ถ้าไม่ตรงใจ ปรับ Prompt แล้วลองใหม่ (อย่างน้อย 3 ครั้ง)
💡 Tips สำหรับบทนี้
  • Prompt ยิ่งเฉพาะเจาะจง ยิ่งได้ผลดี — อย่ากลัวเขียนยาว ยิ่งบอกรายละเอียดมาก ยิ่งได้ผลตรงใจ
  • ใช้ Few-shot เมื่อต้องการรูปแบบเฉพาะ — ให้ตัวอย่าง 2-3 ข้อ AI จะทำตามแน่นอน
  • ถามต่อได้เสมอ — ไม่ต้องเริ่มใหม่ทุกครั้ง AI จำบทสนทนาได้
  • ลองหลายแบบ — Prompt ที่ดีมาจากการทดลอง ไม่มีคำตอบตายตัว
  • เก็บ Prompt ที่ใช้ได้ดี — สร้างคลัง Prompt ส่วนตัวไว้ใช้ซ้ำ
📝 สรุปท้ายบท
  • Prompt = วิธีสั่งงาน AI — ยิ่งสั่งชัด ยิ่งได้ผลดี
  • หลัก 4 อย่าง: Role (บทบาท) + Context (บริบท) + Task (งาน) + Format (รูปแบบ)
  • Zero-shot = สั่งตรงๆ | Few-shot = ให้ตัวอย่าง | Chain-of-Thought = คิดทีละขั้น
  • งานราชการใช้ AI ได้: ร่างหนังสือ สรุปประชุม เขียนโครงการ แปลเอกสาร สร้างกำหนดการ
  • ผลไม่ตรงใจ? เพิ่มบริบท / เปลี่ยน Role / ระบุ Format / ให้ตัวอย่าง / ถามต่อ
