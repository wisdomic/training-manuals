บทที่ 3

ประเภทและความแตกต่างของ AI

รู้จักประเภท AI ทั้งหมด เปรียบเทียบเครื่องมือ และเข้าใจ LLM แบบไม่ใช้คณิตศาสตร์

🎯 วัตถุประสงค์การเรียนรู้
  • แยกแยะ AI แต่ละประเภทได้ (Narrow AI, General AI, Super AI)
  • เปรียบเทียบเครื่องมือ AI หลักได้ (ChatGPT, Claude, Gemini, Copilot)
  • เข้าใจว่า LLM คืออะไร ทำงานอย่างไร (แบบไม่ใช้คณิตศาสตร์)

3.1 Narrow AI vs General AI vs Super AI

AI แบ่งได้ 3 ระดับตามความฉลาด — ลองคิดว่าเป็น "ระดับความสามารถ":

🎯 Narrow AI
ทำได้ดีเรื่องเดียว

คำอธิบาย: AI ที่เก่งเฉพาะเรื่อง — ทำอย่างอื่นไม่ได้

ตัวอย่าง:

  • ChatGPT (เก่งเรื่องภาษา)
  • Siri (ผู้ช่วยเสียง)
  • Face ID (จำหน้าคน)
  • Google Maps (หาทาง)
สถานะ: ✅ มีแล้ว — ใช้ทุกวัน!
🧠 General AI (AGI)
ฉลาดเท่ามนุษย์ทุกด้าน

คำอธิบาย: AI ที่คิดเอง ตัดสินใจเอง เรียนรู้เรื่องใหม่ได้ทุกเรื่อง เหมือนมนุษย์

ตัวอย่าง:

  • ยังไม่มีตัวอย่างจริง
  • ในหนัง: Jarvis (Iron Man)
  • ในหนัง: Samantha (Her)
สถานะ: ⏳ กำลังพัฒนา — ยังไม่มีจริง
⚡ Super AI
เก่งกว่ามนุษย์ทุกด้าน

คำอธิบาย: AI ที่ฉลาดกว่ามนุษย์ทุกคนรวมกัน — แก้ปัญหาที่มนุษย์แก้ไม่ได้

ตัวอย่าง:

  • ยังเป็นทฤษฎี
  • ในหนัง: Skynet (Terminator)
  • ในหนัง: Ultron (Avengers)
สถานะ: 🔮 ทฤษฎี — ไม่รู้จะมีจริงไหม
📌 สรุปง่ายๆ
ประเภท คำอธิบาย ตัวอย่าง สถานะ
Narrow AI เก่งเรื่องเดียว ChatGPT, Siri, Face ID ✅ มีแล้ว
General AI ฉลาดเท่าคน Jarvis (ในหนัง) ⏳ กำลังพัฒนา
Super AI เก่งกว่าคน Skynet (ในหนัง) 🔮 ทฤษฎี

3.2 AI แบ่งตามประเภทเนื้อหา

นอกจากแบ่งตามความฉลาด เรายังแบ่ง AI ตาม "ทำอะไรได้" — มี 5 หมวดหลัก:

🖊️ Text AI — สร้างข้อความ

ทำอะไรได้: เขียน แก้ไข สรุป แปล ตอบคำถาม

เครื่องมือยอดนิยม:

  • ChatGPT (OpenAI)
  • Claude (Anthropic)
  • Gemini (Google)
🎨 Image AI — สร้างรูปภาพ

ทำอะไรได้: วาดรูป แก้ไขรูป ออกแบบ สร้างโลโก้

เครื่องมือยอดนิยม:

  • DALL-E 3 (OpenAI)
  • Midjourney
  • Adobe Firefly
🎵 Audio AI — สร้างเสียง/เพลง

ทำอะไรได้: แต่งเพลง สร้างเสียงพูด ถอดเสียงเป็นข้อความ

เครื่องมือยอดนิยม:

  • ElevenLabs (เสียงพูด)
  • Suno (แต่งเพลง)
  • Whisper (ถอดเสียง)
🎬 Video AI — สร้างวิดีโอ

ทำอะไรได้: สร้างวิดีโอ ตัดต่อ ใส่เอฟเฟกต์ สร้าง Avatar

เครื่องมือยอดนิยม:

  • Google Flow (Veo)
  • CapCut AI
  • Runway
💻 Code AI — เขียนโปรแกรม

ทำอะไรได้: เขียนโค้ด แก้บัก สร้างเว็บ สร้างแอป

เครื่องมือยอดนิยม:

  • GitHub Copilot
  • Cursor
  • Kiro (Amazon)
ประเภท ทำอะไรได้ เครื่องมือยอดนิยม ฟรี/เสียเงิน
🖊️ Text AI เขียน สรุป แปล ตอบคำถาม ChatGPT, Claude, Gemini ฟรี (มี Pro เสริม)
🎨 Image AI วาดรูป สร้างภาพจากคำอธิบาย DALL-E 3, Midjourney, Firefly ฟรีจำกัด / Pro
🎵 Audio AI แต่งเพลง สร้างเสียง ถอดเสียง ElevenLabs, Suno, Whisper ฟรีจำกัด / Pro
🎬 Video AI สร้าง/ตัดต่อวิดีโอ Google Flow, CapCut AI, Runway ฟรีจำกัด / Pro
💻 Code AI เขียนโปรแกรม สร้างเว็บ Copilot, Cursor, Kiro ฟรี / Pro

3.3 เปรียบเทียบ ChatGPT vs Claude vs Gemini vs Copilot

ทั้ง 4 ตัวเป็น AI ที่คุยได้ (Text AI) — แต่มีจุดเด่นต่างกัน:

หัวข้อ ChatGPT Claude Gemini Copilot
บริษัทผู้สร้าง OpenAI (สหรัฐ) Anthropic (สหรัฐ) Google (สหรัฐ) Microsoft (สหรัฐ)
จุดแข็ง เก่งรอบด้าน สร้างรูปได้ มี Plugin เยอะ เขียนยาวได้ดี วิเคราะห์เอกสารเก่ง ปลอดภัย เชื่อมกับ Google (Gmail, Drive, YouTube) ค้นเว็บเก่ง อ้างอิงแหล่งข้อมูล ฟรี
จุดอ่อน เวอร์ชันฟรีจำกัดมากขึ้น ไม่ค้นเว็บ ไม่สร้างรูป (ฟรี) ภาษาไทยยังไม่ดีเท่า คำตอบอาจสั้นไป
เหมาะกับงาน งานทั่วไป สร้างรูป เขียนโค้ด วิเคราะห์เอกสารยาว งานเขียน งานวิชาการ ค้นข้อมูลใน Google งาน Google Workspace ค้นเว็บ สรุปข่าว เขียนอีเมล

💡 แนะนำสำหรับผู้เริ่มต้น:
  • ใช้ ChatGPT — ถ้าต้องการทำได้ทุกอย่าง (เขียน + สร้างรูป + ค้นเว็บ)
  • ใช้ Claude — ถ้าต้องการวิเคราะห์เอกสารยาวๆ หรืองานเขียนคุณภาพสูง
  • ใช้ Gemini — ถ้าใช้ Gmail/Google Drive อยู่แล้ว
  • ใช้ Copilot — ถ้าต้องการค้นข้อมูลพร้อมอ้างอิง (ฟรี!)

3.4 LLM คืออะไร ทำงานอย่างไร

🧠 LLM = Large Language Model (โมเดลภาษาขนาดใหญ่)

LLM คือ "สมอง" ที่อยู่เบื้องหลัง ChatGPT, Claude, Gemini ทุกตัว

💬 อธิบายแบบไม่ใช้คณิตศาสตร์:
"LLM เหมือนคนที่อ่านหนังสือมาล้านเล่ม แล้วจำ pattern ว่าคำไหนมักตามหลังคำไหน — พอถามอะไร ก็ต่อคำให้เป็นประโยคที่สมเหตุสมผล"

📦 Token คืออะไร?

Token = หน่วยย่อยที่ AI อ่าน

AI ไม่ได้อ่าน "คำ" เหมือนคน — แต่อ่าน "Token" ซึ่งเล็กกว่าคำ

  • ภาษาอังกฤษ: 1 Token ≈ 0.75 คำ
  • ภาษาไทย: 1 ตัวอักษร ≈ 1-2 Token (เพราะภาษาไทยซับซ้อน)
ตัวอย่าง: "สวัสดีครับ" → AI อ่านเป็น Token หลายตัว ไม่ใช่คำเดียว
📐 Context Window คืออะไร?

Context Window = ความจำระหว่างสนทนา

เหมือน "กระดาษจดบันทึก" ของ AI — จำได้มากแค่ไหนระหว่างคุยกัน

  • ChatGPT: ~128K Token (≈ หนังสือ 2-3 เล่ม)
  • Claude: ~200K Token (≈ หนังสือ 4-5 เล่ม)
  • Gemini: ~1M Token (≈ หนังสือ 20+ เล่ม)
ง่ายๆ: ยิ่ง Context Window ใหญ่ = AI จำบทสนทนาได้ยาวขึ้น
🌡️ Temperature คืออะไร?

Temperature = ระดับความ "สร้างสรรค์" ของ AI

ลองนึกภาพเป็นปุ่มปรับ:

🧊

Temperature = 0

แม่นยำ ตรงประเด็น
เหมาะกับ: สรุปเอกสาร, ตอบคำถามที่ต้องถูกต้อง

🌤️

Temperature = 0.5

สมดุล
เหมาะกับ: งานทั่วไป สรุป + แต่งเล็กน้อย

🔥

Temperature = 1

สร้างสรรค์ หลากหลาย
เหมาะกับ: แต่งเรื่อง, ไอเดียใหม่, บทกลอน

💡 สรุป: Temperature ต่ำ = ตอบตรงๆ ซ้ำได้ | Temperature สูง = ตอบหลากหลาย สร้างสรรค์ แต่อาจ "ฟุ้ง"

💡 Tips สำหรับบทนี้
  • AI ที่ใช้อยู่ทุกวัน = Narrow AI — ทำได้ดีเรื่องเดียว ยังไม่ฉลาดเท่าคน
  • ไม่มี AI ตัวไหนดีที่สุดตลอด — แต่ละตัวเก่งคนละเรื่อง ลองใช้หลายตัว!
  • LLM ไม่ได้ "เข้าใจ" จริง — แค่ต่อคำได้เก่ง อาจตอบผิดได้ (Hallucination)
  • Context Window ใหญ่ = จำได้มาก — ถ้าต้องวิเคราะห์เอกสารยาว ให้เลือก AI ที่ Context Window ใหญ่
  • Temperature ต่ำ = แม่นยำ / Temperature สูง = สร้างสรรค์
📝 สรุปท้ายบท
  • Narrow AI = เก่งเรื่องเดียว (มีแล้ว ใช้ทุกวัน) | General AI = ฉลาดเท่าคน (ยังไม่มี) | Super AI = เก่งกว่าคน (ทฤษฎี)
  • AI มี 5 หมวด: Text, Image, Audio, Video, Code — แต่ละหมวดมีเครื่องมือให้เลือกมากมาย
  • ChatGPT เก่งรอบด้าน | Claude เก่งวิเคราะห์ | Gemini เชื่อม Google | Copilot ค้นเว็บฟรี
  • LLM คือสมองของ AI — อ่านข้อมูลมหาศาล จำ pattern แล้วต่อคำให้เป็นประโยค
  • Token = หน่วยย่อย | Context Window = ความจำ | Temperature = ระดับสร้างสรรค์
