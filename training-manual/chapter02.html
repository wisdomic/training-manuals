2.1 ติดตั้ง Node.js

Node.js คือ โปรแกรมที่ทำให้คอมพิวเตอร์รัน JavaScript ได้ — จำเป็นสำหรับ Clasp และ npm

1 ดาวน์โหลด Node.js

เปิดเว็บ https://nodejs.org

คลิกปุ่ม "LTS" (Long Term Support) สีเขียว — เลือกเวอร์ชันที่แนะนำ

2 ติดตั้ง

เปิดไฟล์ที่ดาวน์โหลด (.msi) → คลิก Next ไปเรื่อยๆ → Install → Finish

✅ ใช้ค่า default ทั้งหมด ไม่ต้องเปลี่ยนอะไร

3 ทดสอบ

เปิด PowerShell (คลิกขวาที่ Start → Windows PowerShell)

node -v
npm -v

ถ้าแสดงเลขเวอร์ชัน เช่น v20.x.x = สำเร็จ ✅

📸 ภาพหน้าจอ: ผลลัพธ์ node -v และ npm -v ใน PowerShell

2.2 ติดตั้ง Git + สร้าง GitHub Account

Git คืออะไร?

Git คือ ระบบควบคุมเวอร์ชัน (Version Control) ช่วยเก็บประวัติการแก้ไขโค้ด ทำงานเป็นทีม และสำรองข้อมูลบน Cloud (GitHub)

ขั้นตอนที่ 1: สร้าง GitHub Account
1 สมัคร GitHub

เปิด https://github.com → คลิก "Sign up" → กรอกข้อมูล → ยืนยัน Email

ขั้นตอนที่ 2: ติดตั้ง Git
1 ดาวน์โหลด Git

เปิด https://git-scm.com → คลิก "Download for Windows"

2 ติดตั้ง

เปิดไฟล์ .exe → คลิก Next ไปเรื่อยๆ (ใช้ค่า default) → Install → Finish

3 ตั้งค่า Git User

เปิด PowerShell แล้วรัน:

git --version
git config --global user.name "ชื่อของคุณ"
git config --global user.email "email@gmail.com"
💡 ทำไมต้องตั้ง user.name? — เพื่อให้ Git รู้ว่าใครเป็นคนแก้ไขโค้ด เวลาทำงานเป็นทีมจะได้รู้ว่าใครทำอะไร

2.3 ติดตั้ง Kiro IDE

1 ดาวน์โหลด Kiro

เปิด https://kiro.dev → คลิก "Download" → เลือก Windows

2 ติดตั้ง

เปิดไฟล์ที่ดาวน์โหลด → ติดตั้งตามขั้นตอน → เปิด Kiro

3 Login

เมื่อเปิด Kiro ครั้งแรก:

  • คลิก "Sign in" (ด้านซ้ายล่าง)
  • เลือก Login ด้วย Google, GitHub, หรือ AWS Builder ID
  • แนะนำใช้ Google Account เดียวกับที่ใช้ Google Sheets
4 เปิดโฟลเดอร์โปรเจกต์

File → Open Folder → เลือกโฟลเดอร์ที่จะทำงาน (เช่น C:\MyProject)

📌 Interface ของ Kiro
ซ้ายExplorer — ดูไฟล์ในโปรเจกต์
กลางEditor — แก้ไขโค้ด
ขวาKiro Chat — สนทนากับ AI
ล่างTerminal — รันคำสั่ง

2.4 ติดตั้ง Google Clasp (CLI)

Clasp (Command Line Apps Script Projects) คือเครื่องมือที่ช่วยอัปโหลดโค้ดจากคอมพิวเตอร์ขึ้น Google Apps Script โดยไม่ต้องเปิด Script Editor

1 ติดตั้ง Clasp

เปิด Terminal ใน Kiro (หรือ PowerShell) แล้วรัน:

npm install -g @google/clasp

รอจนติดตั้งเสร็จ (อาจใช้เวลา 1-2 นาที)

2 ทดสอบ 
clasp --version

ถ้าแสดงเวอร์ชัน เช่น 2.4.x = สำเร็จ ✅

3 เปิด Apps Script API

เปิดเว็บ https://script.google.com/home/usersettings

เปิดสวิตช์ "Google Apps Script API" ให้เป็น ON

📸 ภาพหน้าจอ: หน้า Apps Script Settings แสดง API = ON

2.5 clasp login + เชื่อมต่อ Google Account

1 Login Clasp 
clasp login

เบราว์เซอร์จะเปิดขึ้น → เลือกบัญชี Google → กด "Allow" ทุกสิทธิ์

2 ยืนยันสำเร็จ

Terminal จะแสดง Authorization successful. และเบราว์เซอร์จะแสดง "Logged in!"

3 ทดสอบ clasp 
clasp list

จะแสดงรายการ Apps Script projects ที่มีในบัญชี (ถ้ามี)

❗ ปัญหาที่พบบ่อย:
  • "This app isn't verified" — คลิก "Advanced" → "Go to ... (unsafe)" → Allow
  • "Apps Script API not enabled" — กลับไปทำข้อ 2.4 ขั้นตอนที่ 3
  • "Access denied" — ตรวจสอบว่า login ถูกบัญชี

2.6 ทดสอบเครื่องมือทั้งหมด (Checklist)

รันคำสั่งทั้งหมดนี้ใน Terminal เพื่อตรวจสอบว่าทุกอย่างพร้อม:

node -v          # ต้องแสดง v20.x.x หรือสูงกว่า
npm -v           # ต้องแสดง 10.x.x หรือสูงกว่า
git --version    # ต้องแสดง git version 2.x.x
clasp --version  # ต้องแสดง 2.4.x หรือสูงกว่า
✅ Checklist ความพร้อม
รายการสถานะหมายเหตุ
Node.js ติดตั้งแล้ว☐node -v แสดงเวอร์ชัน
npm ใช้งานได้☐npm -v แสดงเวอร์ชัน
Git ติดตั้งแล้ว☐git --version แสดงเวอร์ชัน
Git user ตั้งค่าแล้ว☐git config --list แสดงชื่อ+email
GitHub account มีแล้ว☐login ได้ที่ github.com
Kiro IDE ติดตั้งแล้ว☐เปิด Kiro ได้ + login แล้ว
Clasp ติดตั้งแล้ว☐clasp --version แสดงเวอร์ชัน
Clasp login แล้ว☐clasp list ไม่ error
Apps Script API เปิดแล้ว☐สวิตช์ ON ในหน้า settings
