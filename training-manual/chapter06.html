พื้นฐาน HTML + CSS + Bootstrap

ระยะเวลา: 2 ชั่วโมง | วันที่ 2

🎯 วัตถุประสงค์การเรียนรู้
  • เข้าใจโครงสร้าง HTML5 พื้นฐาน
  • ใช้ CSS จัดรูปแบบหน้าเว็บได้
  • ใช้ Bootstrap 5 สร้าง UI สวยงามได้อย่างรวดเร็ว

6.1 โครงสร้าง HTML5

HTML5 คือภาษาสำหรับสร้างโครงสร้างหน้าเว็บ ทุกหน้าเว็บต้องมีโครงสร้างพื้นฐานนี้:

<!DOCTYPE html>
<html lang="th">
<head>
    <!-- meta charset: กำหนดภาษาไทย -->
    <meta charset="UTF-8">
    
    <!-- viewport: ให้แสดงผลมือถือถูกต้อง -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- title: ชื่อแท็บในเบราว์เซอร์ -->
    <title>ระบบลงทะเบียน</title>
    
    <!-- CSS: ใส่สไตล์ -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- เนื้อหาที่แสดงผลอยู่ใน body -->
    <h1>สวัสดีชาวโลก</h1>
    <p>นี่คือย่อหน้าแรก</p>
    
    <!-- JavaScript: ใส่ท้าย body เสมอ -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
แท็ก HTML ที่ใช้บ่อย
แท็กหน้าที่ตัวอย่าง
<h1>-<h6>หัวข้อ (ใหญ่→เล็ก)<h1>ชื่อระบบ</h1>
<p>ย่อหน้า<p>คำอธิบาย</p>
<div>กล่องจัดกลุ่ม<div class="container">...</div>
<a>ลิงก์<a href="url">คลิก</a>
<img>รูปภาพ<img src="pic.jpg" alt="รูป">
<table>ตาราง<table><tr><td>...</td></tr></table>
<form>ฟอร์มกรอกข้อมูล<form><input><button></form>
<button>ปุ่ม<button class="btn">บันทึก</button>

6.2 CSS เบื้องต้น

CSS (Cascading Style Sheets) ใช้จัดรูปแบบ ตกแต่ง สี ขนาด ตำแหน่งของ HTML:

3 วิธีใส่ CSS
วิธีตัวอย่างเหมาะกับ
Inline <p style="color: red;">ข้อความ</p> แก้ชั่วคราว
Internal <style> p { color: red; } </style> GAS (ใส่ใน .html)
External <link href="style.css" rel="stylesheet"> เว็บทั่วไป
Selectors — เลือก Element ที่จะจัดสไตล์
<style>
/* เลือกทุก p */
p { color: #333; }

/* เลือกด้วย class (ใช้จุด) */
.highlight { background-color: yellow; }

/* เลือกด้วย id (ใช้ #) */
#main-title { font-size: 24px; font-weight: bold; }

/* เลือกลูกของ element */
.card .card-title { color: blue; }
</style>

<h1 id="main-title">ชื่อระบบ</h1>
<p class="highlight">ข้อความเน้น</p>
Properties ที่ใช้บ่อย
.my-box {
  /* สี */
  color: #333;                  /* สีตัวอักษร */
  background-color: #f8f9fa;   /* สีพื้นหลัง */
  
  /* ขนาด */
  width: 100%;
  padding: 16px;               /* ระยะห่างด้านใน */
  margin: 8px 0;               /* ระยะห่างด้านนอก */
  
  /* ขอบ */
  border: 1px solid #ddd;
  border-radius: 8px;          /* มุมโค้ง */
  
  /* ข้อความ */
  font-size: 16px;
  text-align: center;
}

6.3 Bootstrap 5 — Grid System, Components

Bootstrap 5 คือ CSS Framework ที่ช่วยสร้างหน้าเว็บสวยงามและ Responsive ได้ทันที:

Grid System — จัดวาง Layout
<!-- Container: กล่องหลักจัดกลางหน้า -->
<div class="container">
  
  <!-- Row: แถว -->
  <div class="row">
    
    <!-- Col: คอลัมน์ (รวม 12 ส่วน) -->
    <div class="col-md-4">ซ้าย (4/12)</div>
    <div class="col-md-8">ขวา (8/12)</div>
    
  </div>
  
  <!-- 3 คอลัมน์เท่ากัน -->
  <div class="row">
    <div class="col-md-4">1</div>
    <div class="col-md-4">2</div>
    <div class="col-md-4">3</div>
  </div>
  
</div>
Components ที่ใช้บ่อยในระบบราชการ
Cards — กล่องข้อมูล
<div class="card">
  <div class="card-header bg-primary text-white">หัวข้อ</div>
  <div class="card-body">
    <h5 class="card-title">ชื่อ</h5>
    <p class="card-text">รายละเอียด</p>
    <button class="btn btn-primary">ดำเนินการ</button>
  </div>
</div>
Forms — ฟอร์ม
<form>
  <div class="mb-3">
    <label class="form-label">ชื่อ-สกุล</label>
    <input type="text" class="form-control" id="fullName" required>
  </div>
  <div class="mb-3">
    <label class="form-label">หลักสูตร</label>
    <select class="form-select" id="course">
      <option value="">-- เลือกหลักสูตร --</option>
    </select>
  </div>
  <button type="submit" class="btn btn-primary">ลงทะเบียน</button>
</form>
Tables — ตาราง
<table class="table table-bordered table-hover">
  <thead class="table-dark">
    <tr><th>#</th><th>ชื่อ</th><th>สถานะ</th><th>จัดการ</th></tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>สมชาย ใจดี</td>
      <td><span class="badge bg-success">ยืนยัน</span></td>
      <td><button class="btn btn-sm btn-warning">แก้ไข</button></td>
    </tr>
  </tbody>
</table>
Alerts + Modals
<!-- Alert -->
<div class="alert alert-success" role="alert">
  ✅ บันทึกข้อมูลสำเร็จ!
</div>

<!-- Modal (ป๊อปอัพ) -->
<div class="modal fade" id="confirmModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">ยืนยันการลบ</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">คุณต้องการลบข้อมูลนี้?</div>
      <div class="modal-footer">
        <button class="btn btn-secondary" data-bs-dismiss="modal">ยกเลิก</button>
        <button class="btn btn-danger">ลบ</button>
      </div>
    </div>
  </div>
</div>
Navbar — แถบนำทาง
<nav class="navbar navbar-dark bg-primary sticky-top">
  <div class="container">
    <a class="navbar-brand" href="#">🏛️ ระบบลงทะเบียน</a>
    <span class="navbar-text text-white">สนพ.สุรินทร์</span>
  </div>
</nav>
สรุป Class ที่ใช้บ่อยที่สุด
Classหน้าที่
containerกล่องจัดกลาง มีระยะห่างข้าง
row / col-md-*จัดคอลัมน์ (Grid)
btn btn-primaryปุ่มสีน้ำเงิน
form-control / form-selectInput / Dropdown สวยๆ
table table-borderedตารางมีเส้นขอบ
card / card-bodyกล่องข้อมูล
alert alert-successกล่องแจ้งเตือนสีเขียว
badge bg-warningป้ายสถานะสีเหลือง
mb-3 / mt-4 / p-3ระยะห่าง (margin/padding)
text-center / fw-boldจัดกลาง / ตัวหนา

6.4 JavaScript เบื้องต้น (สำหรับ GAS Frontend)

JavaScript ที่จำเป็นต้องรู้สำหรับทำ Frontend ใน GAS:

ตัวแปร + ฟังก์ชัน
// ประกาศตัวแปร
var name = 'สมชาย';        // ข้อความ
var age = 30;              // ตัวเลข
var isActive = true;       // จริง/เท็จ
var courses = [];          // Array (รายการ)

// ฟังก์ชัน
function showAlert(message) {
  alert(message);
}

// เรียกใช้
showAlert('สวัสดี ' + name);
DOM Manipulation — เปลี่ยนหน้าเว็บ
// ดึง element ด้วย id
var titleEl = document.getElementById('pageTitle');
titleEl.textContent = 'ชื่อใหม่';           // เปลี่ยนข้อความ
titleEl.style.color = 'red';               // เปลี่ยนสี

// ดึงหลาย elements ด้วย class
var items = document.querySelectorAll('.item');

// เปลี่ยน HTML ข้างใน
document.getElementById('output').innerHTML = '<b>สำเร็จ!</b>';

// แสดง/ซ่อน
document.getElementById('loading').style.display = 'block';  // แสดง
document.getElementById('loading').style.display = 'none';   // ซ่อน
Event Listeners — ตอบสนองผู้ใช้
// คลิกปุ่ม
document.getElementById('btnSave').addEventListener('click', function() {
  var name = document.getElementById('inputName').value;
  if (!name) {
    alert('กรุณากรอกชื่อ');
    return;
  }
  // ส่งข้อมูลไป Backend
  google.script.run
    .withSuccessHandler(function() { alert('บันทึกสำเร็จ!'); })
    .saveData(name);
});

// กด Enter ในช่อง input
document.getElementById('searchBox').addEventListener('keyup', function(e) {
  if (e.key === 'Enter') {
    searchData(this.value);
  }
});
Fetch — เรียก API (ใน GAS ใช้ google.script.run แทน)
// ในเว็บทั่วไปใช้ fetch
// แต่ใน GAS ใช้ google.script.run แทน (ง่ายกว่า)

// ตัวอย่าง pattern ที่ใช้จริงใน GAS frontend:
function loadData() {
  showLoading(true);
  google.script.run
    .withSuccessHandler(function(data) {
      showLoading(false);
      renderTable(data);
    })
    .withFailureHandler(function(err) {
      showLoading(false);
      showError(err.message);
    })
    .getData();
}

function showLoading(show) {
  document.getElementById('spinner').style.display = show ? 'block' : 'none';
}

6.5 Responsive Design

Responsive Design = หน้าเว็บปรับตัวตามขนาดหน้าจอ (มือถือ, แท็บเล็ต, คอมพิวเตอร์)

Breakpoints ของ Bootstrap 5
Prefixขนาดหน้าจออุปกรณ์
(none)< 576pxมือถือเล็ก
sm≥ 576pxมือถือใหญ่
md≥ 768pxแท็บเล็ต
lg≥ 992pxแล็ปท็อป
xl≥ 1200pxคอมพิวเตอร์
xxl≥ 1400pxจอใหญ่
ซ่อน/แสดง ตามขนาดจอ
<!-- ซ่อนบนมือถือ แสดงบนแท็บเล็ตขึ้นไป -->
<div class="d-none d-md-block">
  เนื้อหานี้ไม่แสดงบนมือถือ
</div>

<!-- แสดงเฉพาะมือถือ ซ่อนบนแท็บเล็ตขึ้นไป -->
<div class="d-block d-md-none">
  เนื้อหานี้แสดงเฉพาะมือถือ
</div>

<!-- รูปภาพ Responsive -->
<img src="photo.jpg" class="img-fluid" alt="รูปภาพ">

<!-- ตาราง Responsive (เลื่อนซ้ายขวาบนมือถือ) -->
<div class="table-responsive">
  <table class="table">...</table>
</div>
ตัวอย่าง: Grid ที่ปรับตามหน้าจอ
<div class="row">
  <!-- มือถือ: เต็มจอ (12) / แท็บเล็ต: ครึ่งจอ (6) / คอม: 1/3 จอ (4) -->
  <div class="col-12 col-md-6 col-lg-4 mb-3">
    <div class="card">
      <div class="card-body">Card 1</div>
    </div>
  </div>
  <div class="col-12 col-md-6 col-lg-4 mb-3">
    <div class="card">
      <div class="card-body">Card 2</div>
    </div>
  </div>
  <div class="col-12 col-md-6 col-lg-4 mb-3">
    <div class="card">
      <div class="card-body">Card 3</div>
    </div>
  </div>
</div>
💡 เคล็ดลับ: บอก Kiro ว่า "ให้ทำ Responsive ด้วย Bootstrap, มือถือแสดง 1 คอลัมน์ คอมแสดง 3 คอลัมน์" — Kiro จะใส่ class ให้อัตโนมัติ!
