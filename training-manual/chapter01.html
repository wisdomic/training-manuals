แนะนำ AI กับการพัฒนาระบบ

1.1 AI คืออะไร — ภาพรวมปัญญาประดิษฐ์สำหรับผู้เริ่มต้น

ความหมายของ AI

ปัญญาประดิษฐ์ (Artificial Intelligence: AI) คือ เทคโนโลยีที่ทำให้คอมพิวเตอร์สามารถเรียนรู้ คิด วิเคราะห์ และตัดสินใจได้คล้ายมนุษย์ โดยไม่ต้องเขียนโปรแกรมสั่งทุกขั้นตอน

🏛️ เปรียบเทียบให้เข้าใจง่าย

ถ้า คอมพิวเตอร์ทั่วไป เปรียบเหมือน "ลูกน้องที่ทำตามคำสั่งทุกอย่าง"
AI เปรียบเหมือน "ผู้ช่วยที่เรียนรู้ได้ คิดเองได้ และเสนอทางเลือกให้"

ประเภทของ AI
ประเภทตัวอย่างการใช้งาน
Generative AIChatGPT, Claude, Geminiสร้างข้อความ, โค้ด, รูปภาพ
Machine Learningแนะนำสินค้า, ตรวจสแปมเรียนรู้จากข้อมูล ทำนายผล
Computer Visionจดจำใบหน้า, อ่าน OCRวิเคราะห์ภาพ
NLPแปลภาษา, สรุปเอกสารเข้าใจภาษามนุษย์
AI ที่เกี่ยวข้องกับการเขียนโปรแกรม

ในหลักสูตรนี้เราจะเน้น Generative AI สำหรับเขียนโค้ด ซึ่งสามารถ:

  • เขียนโค้ดจากคำอธิบายภาษาไทย/อังกฤษ
  • แก้ Bug อัตโนมัติ
  • อธิบายโค้ดที่ซับซ้อน
  • สร้างระบบทั้งระบบจาก Requirement
  • Deploy ขึ้นเซิร์ฟเวอร์ให้
💡 ข้อสรุป: คุณไม่จำเป็นต้องเป็นโปรแกรมเมอร์ — แค่อธิบายสิ่งที่ต้องการ AI จะเขียนโค้ดให้คุณ!

1.2 AI Coding Tools — เปรียบเทียบเครื่องมือ AI สำหรับเขียนโค้ด

ปัจจุบันมีเครื่องมือ AI สำหรับช่วยเขียนโปรแกรมมากมาย แต่ละตัวมีจุดเด่นต่างกัน:

เครื่องมือ บริษัท จุดเด่น ข้อจำกัด ราคา
ChatGPT OpenAI สนทนาได้ดี, หลายภาษา ไม่เชื่อมต่อกับ IDE โดยตรง ฟรี / $20/เดือน
Claude Anthropic เขียนโค้ดยาวได้ดี, ปลอดภัย ไม่มี IDE ของตัวเอง ฟรี / $20/เดือน
GitHub Copilot Microsoft เติมโค้ดอัตโนมัติใน VS Code ช่วยเป็นบรรทัด ไม่ใช่ทั้งระบบ $10/เดือน
Cursor Cursor Inc. IDE + AI, Composer mode ค่าใช้จ่ายสูง, เน้น vibe coding $20/เดือน
Kiro AI ⭐ AWS (Amazon) Spec-Driven, Deploy ได้, ฟรี! ใหม่ (2025), ยังพัฒนาอยู่ ฟรี (Preview)
🏆 ทำไมเราเลือก Kiro AI?
  1. ฟรี — ไม่มีค่าใช้จ่ายในช่วง Preview
  2. สร้างทั้งระบบ — ไม่ใช่แค่เติมโค้ดทีละบรรทัด แต่สร้างทั้ง Backend + Frontend
  3. Deploy ได้ — อัปโหลดขึ้น Google Apps Script ให้เลย
  4. Spec-Driven — เขียน requirement ภาษาไทย แล้ว AI สร้างระบบให้
  5. AWS backing — มี Amazon สนับสนุน เชื่อถือได้
  6. ทำงานเป็นทีมได้ — Git + GitHub integration

1.3 Kiro AI คืออะไร — ความเป็นมาและความสามารถ

ความเป็นมา

Kiro เป็น Agentic IDE (สภาพแวดล้อมพัฒนาซอฟต์แวร์ที่ขับเคลื่อนด้วย AI Agent) พัฒนาโดย Amazon Web Services (AWS) เปิดตัวกลางปี 2025 โดยสร้างบน Amazon Bedrock ใช้โมเดล Claude Sonnet สำหรับงานวิเคราะห์ และ Amazon Nova สำหรับงานสร้างโค้ด

แนวคิดหลัก: Spec-Driven Development

ต่างจาก AI Coding ทั่วไปที่ต้อง "พิมพ์ Prompt ไปเรื่อยๆ" Kiro ใช้แนวคิด Spec-Driven Development คือ:

1️⃣
Requirements

บอก AI ว่าต้องการอะไร

2️⃣
Design

AI ออกแบบระบบให้

3️⃣
Tasks

แบ่งงานเป็นขั้นตอน

4️⃣
Implementation

AI เขียนโค้ดให้ทีละ Task

ความสามารถหลักของ Kiro
ความสามารถรายละเอียด
🗣️ Vibe Modeสนทนากับ AI สร้างโค้ดได้ทันที (เหมือน ChatGPT แต่เชื่อมกับไฟล์)
📋 Spec Modeสร้างระบบแบบมีแบบแปลน (Requirements → Design → Tasks)
📁 File Managementสร้าง/แก้ไข/ลบไฟล์ได้โดยตรง
💻 Terminalรันคำสั่ง command line ได้ (git, clasp, npm)
🔗 MCPเชื่อมต่อ API ภายนอก (Database, Cloud Services)
📐 Steeringกำหนดกฎ/มาตรฐานให้ AI ปฏิบัติตาม
⚡ Hooksอัตโนมัติ — ทำงานเมื่อเกิดเหตุการณ์ (บันทึกไฟล์, push code)
🌐 Web Searchค้นหาข้อมูลจากอินเทอร์เน็ตได้
🤖 Sub-agentsมอบหมายงานย่อยให้ AI ตัวอื่นช่วย

1.4 ทำไมต้อง Kiro — จุดเด่นและกรณีศึกษา

จุดเด่นที่แตกต่าง
❌ ปัญหาของ AI Coding ทั่วไป
  • Prompt ไปเรื่อยๆ ไม่มีโครงสร้าง
  • โค้ดไม่สอดคล้องกันทั้งระบบ
  • ลืม context เมื่อแชทยาว
  • ต้อง copy-paste โค้ดไปวางเอง
  • Deploy ต้องทำเอง
✅ Kiro แก้ปัญหาเหล่านี้
  • Spec-Driven มีแบบแปลนชัดเจน
  • Steering กำหนดมาตรฐานโค้ดทั้งโปรเจกต์
  • อ่านไฟล์ทั้งโปรเจกต์ ไม่ลืม context
  • แก้ไขไฟล์โดยตรง ไม่ต้อง copy-paste
  • Deploy ด้วย clasp ได้จากใน IDE
กรณีศึกษา: สนพ.สุรินทร์

อ.ปู (ปัญญา ผาธรรม) เจ้าหน้าที่ฝ่ายแผนงานและสารสนเทศ สำนักงานพัฒนาฝีมือแรงงานสุรินทร์ ใช้ Kiro AI พัฒนาระบบสำเร็จแล้วกว่า 40+ ระบบ ได้แก่:

  • ระบบจำหน่ายซอฟต์แวร์ (WISDOM ICT) — หน้าร้าน + Admin + สมาชิก
  • ระบบประเมินผลฝึกอบรม (Training Evaluation)
  • ระบบบริหารภาระงาน (WisdomTask)
  • ระบบลงทะเบียนฝึกอบรม, AI Chatbot, ระบบหนังสือราชการ ฯลฯ

ทั้งหมดนี้สร้างด้วย Kiro AI + Google Apps Script โดยไม่ต้องเขียนโค้ดเอง!

1.5 ภาพรวมหลักสูตร — สิ่งที่จะได้เรียนรู้ใน 5 วัน

วันหัวข้อหลักผลลัพธ์
